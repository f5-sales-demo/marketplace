O plugin **f5xc-platform** fornece automação completa para a plataforma
F5 Distributed Cloud. Abrange operações no console web via
Chrome DevTools MCP, operações de API REST com consciência de especificação via cURL,
autenticação com múltiplos provedores e análise de configuração em
98 tipos de recursos em 38 domínios.

<Badge text="v3.1.1" variant="note" /> <Badge text="Produtividade" variant="tip" />

## Instalação

```
/plugin install f5xc-platform@f5-sales-demo-marketplace
```

## Habilidades

### api-operations

Operações CRUD de API com consciência de especificação, cobrindo 98 tipos de recursos em
38 domínios. Fornece endpoints precisos, templates de payload,
ordenação de dependências e composições de fluxo de trabalho em múltiplas etapas para
balanceadores de carga HTTP, pools de origem, políticas WAF, zonas DNS,
certificados, sites em nuvem e mais.

### config-analysis

Capacidade de análise de configuração e consultoria. Analisa configurações
JSON de clientes para responder perguntas sobre postura de segurança,
habilitação de recursos, configurações de modo, exclusões e melhores práticas.
Suporta perguntas e respostas em múltiplas rodadas com contexto comprimido entre despachos.

### api-auth

Autenticação de API com suporte a autenticação via cabeçalho de token de API e
autenticação por certificado P12 com procedimentos de validação de token.

### console-auth

Autenticação com múltiplos provedores para o console web F5 XC.
Detecta automaticamente o tipo de login: login nativo F5 XC, SSO do Azure com
sessão em cache, ou SSO do Azure com MFA de push verificado pelo DUO.

### console-navigator

Navegação determinística para seções do console F5 XC. Traduz
nomes de seções em caminhos de URL, verifica a integridade da sessão e confirma
o carregamento da página.

### platform-index / api-index / console-index

Roteadores de intenção que classificam as solicitações do usuário e delegam para a
habilidade e o agente apropriados. Ativam automaticamente e não
são invocáveis pelo usuário.

## Comandos

### /check-api-token

```
/f5xc-platform:check-api-token
```

Valida o seu token de API F5 XC delegando ao agente
api-operator.

### /login-console

```
/f5xc-platform:login-console
```

Autentica no console do F5 Distributed Cloud por meio do
agente console-operator.

## Agentes

### api-operator

Agente autônomo de API REST que executa sequências de cURL e jq para
CRUD de recursos, validação de token e operações de configuração.
Executa em um subagente para manter os payloads JSON verbosos da API fora do
contexto da sessão principal.

### console-operator

Agente autônomo de automação de navegador que executa sequências de ferramentas do Chrome DevTools MCP
para autenticação, navegação e interações com formulários.
Executa em um subagente para manter os snapshots do navegador com tokens pesados fora da sessão principal.

### config-analyzer

Agente de análise de configuração somente leitura. Analisa configurações
JSON de clientes em relação a arquivos de referência de perfil de recursos para responder
perguntas sobre postura de segurança, habilitação de recursos, transições
de modo e melhores práticas.

<Aside type="caution">
  O agente config-analyzer é **efêmero** &mdash; instanciado do zero
  para cada pergunta, sem memória entre os despachos. Perguntas de acompanhamento
  devem incluir conclusões anteriores comprimidas no prompt de despacho.
</Aside>
