O plugin **f5xc-meddpicc** fornece um framework abrangente de qualificação
de vendas e execução de negócios MEDDPICC para vendas B2B empresariais
complexas. Ele transforma o MEDDPICC de um exercício de checklist em um
sistema vivo de execução de negócios com qualificação baseada em evidências,
revisões estruturadas de negócios e coaching acionável.

<Badge text="v1.0.4" variant="note" /> <Badge text="Produtividade" variant="tip" />

## Instalação

```
/plugin install f5xc-meddpicc@f5-sales-demo-marketplace
```

## O que é MEDDPICC?

MEDDPICC é um framework de qualificação e execução de negócios projetado para
vendas B2B complexas, nas quais múltiplos stakeholders, riscos técnicos e
etapas de aquisição podem comprometer o momentum.

| Letra | Elemento | Finalidade |
| ----- | -------- | ---------- |
| **M** | Métricas (Metrics) | Resultados de negócio quantificados que o cliente espera |
| **E** | Comprador Econômico (Economic Buyer) | Indivíduo com autoridade final sobre o orçamento |
| **D** | Critérios de Decisão (Decision Criteria) | Requisitos usados para avaliar as opções |
| **D** | Processo de Decisão (Decision Process) | Caminho passo a passo da avaliação ao acordo |
| **P** | Processo de Contratação (Paper Process) | Mecânica de aquisição (jurídico, segurança, assinatura) |
| **I** | Identificar a Dor (Identify Pain) | Dor de negócio relevante que cria urgência |
| **C** | Campeão (Champion) | Defensor interno com influência |
| **C** | Concorrência (Competition) | Todas as alternativas (fornecedores, desenvolvimento interno, não fazer nada) |

## Skills

### meddpicc-coach

Coaching abrangente de metodologia MEDDPICC que se ativa automaticamente
quando você discute qualificação de negócios, saúde do pipeline, precisão
de forecast ou qualquer elemento do MEDDPICC. Fornece orientação baseada em
evidências, recomendações de ritmo operacional e links para material de
referência detalhado.

<Aside type="tip" title="Ativação automática">
  Esta skill se ativa automaticamente quando relevante — nenhum comando slash
  é necessário.
</Aside>

### deal-qualification

Produz um scorecard MEDDPICC pontuado (0&ndash;24) com análise de lacunas,
avaliação de riscos e recomendação de forecast. Cada elemento é pontuado de
0&ndash;3 com base em evidências verificadas.

### deal-review

Facilita inspeções estruturadas semanais de negócios com rastreamento de
delta de evidências, identificação de riscos e itens de ação com responsáveis
nomeados e datas de vencimento.

### mutual-action-plan

Constrói um Plano de Ação Mútuo voltado ao cliente vinculado aos elementos
do MEDDPICC. Inclui marcos de avaliação, roteiro de aquisição, prontidão
para implementação e mapeamento de stakeholders.

### champion-test

Aplica uma avaliação de 5 critérios (Poder, Ganho Pessoal, Acesso,
Inteligência, Ação) com pontuação de 0&ndash;15 para determinar se seu
contato interno é um verdadeiro campeão, campeão em desenvolvimento, coach
ou contato amigável.

## Comandos

| Comando | Finalidade |
| ------- | ---------- |
| `/f5xc-meddpicc:qualify-deal [account]` | Pontuar um negócio com o scorecard MEDDPICC |
| `/f5xc-meddpicc:deal-review [account]` | Facilitar a inspeção semanal de negócios |
| `/f5xc-meddpicc:champion-test [contact]` | Avaliar se o contato é um verdadeiro campeão |
| `/f5xc-meddpicc:build-map [account]` | Construir um Plano de Ação Mútuo |

## Agentes

### deal-analyst

Agente de pesquisa somente leitura que analisa a saúde do negócio revisando
documentação local, notas de reuniões e planos de conta. Produz relatórios
estruturados de avaliação MEDDPICC com citações de evidências, análise de
lacunas e recomendações de ações priorizadas.

## Biblioteca de Referência

O plugin inclui material de referência abrangente que é carregado sob
demanda:

- **Framework MEDDPICC** &mdash; definições completas e requisitos de
  evidências para todos os 8 elementos
- **Perguntas de Discovery** &mdash; perguntas testadas em campo para cada
  elemento ao longo do ciclo de negócios
- **Papéis e Responsabilidades** &mdash; matriz RACI de responsabilidade por
  função na equipe (AE, SE, CSM, PS, Produto, Canal, Deal Desk, Jurídico)
- **Gates de Estágio do Negócio** &mdash; o que deve ser verdadeiro antes de
  avançar (da Qualificação Inicial ao Fechamento Ganho)
- **Anti-Padrões** &mdash; 10 modos de falha comuns com remediação baseada
  em evidências
- **Rubrica de Pontuação** &mdash; critérios objetivos de pontuação 0&ndash;3
  por elemento
- **Templates de Revisão** &mdash; conjuntos de perguntas de inspeção semanal
  com indicadores de alerta
- **Templates de MAP** &mdash; estrutura de plano de ação voltada ao cliente
  com roteiros de avaliação, aquisição e implementação
