Este guia apresenta o processo de adição do marketplace f5-sales-demo
ao Claude Code e a instalação do seu primeiro plugin.

## Pré-requisitos

- Claude Code **v1.0.33** ou posterior (suporte ao marketplace de plugins)
- Uma conta no GitHub com acesso à organização
  [f5-sales-demo](https://github.com/f5-sales-demo)

## Instalação

<Steps>

1. **Adicionar o marketplace**

   Em qualquer sessão do Claude Code, execute:

   ```
   /plugin marketplace add f5-sales-demo/marketplace
   ```

   Isso registra o marketplace como uma fonte de plugins. Você só
   precisa fazer isso uma vez.

2. **Navegar pelos plugins disponíveis**

   Liste o que o marketplace oferece:

   ```
   /plugin marketplace list f5-sales-demo-marketplace
   ```

3. **Instalar um plugin**

   Instale o plugin de que você precisa:

   ```
   /plugin install f5xc-docs-tools@f5-sales-demo-marketplace
   /plugin install f5xc-sales-engineer@f5-sales-demo-marketplace
   ```

4. **Usar o plugin**

   Navegue até qualquer repositório de conteúdo do f5-sales-demo.

   Para as **ferramentas de documentação**, execute o comando de revisão MDX:

   ```
   /f5xc-docs-tools:review-mdx
   ```

   Para o **engenheiro de vendas**, utilize frases de ativação como
   "prepare the demo", "run the demo" ou "walk through
   the demo" — o plugin é ativado automaticamente.

</Steps>

## Atualizando Plugins

Para atualizar um plugin instalado para a versão mais recente:

```
/plugin update f5xc-docs-tools@f5-sales-demo-marketplace
/plugin update f5xc-sales-engineer@f5-sales-demo-marketplace
```

## Desinstalação

Remover um plugin:

```
/plugin uninstall f5xc-docs-tools@f5-sales-demo-marketplace
/plugin uninstall f5xc-sales-engineer@f5-sales-demo-marketplace
```

Remover o marketplace completamente:

```
/plugin marketplace remove f5-sales-demo-marketplace
```

## Configuração para Equipes

Para pré-configurar o marketplace para toda a sua equipe, adicione-o
ao arquivo `.claude/settings.json` do seu projeto:

```json
{
  "extraKnownMarketplaces": [
    "f5-sales-demo/marketplace"
  ],
  "enabledPlugins": [
    "f5xc-docs-tools@f5-sales-demo-marketplace",
    "f5xc-sales-engineer@f5-sales-demo-marketplace"
  ]
}
```

<Aside type="tip" title="Configurações compartilhadas">
  Fazer o commit do arquivo `.claude/settings.json` no seu repositório significa que
  todos os membros da equipe terão o marketplace e os plugins
  automaticamente — sem necessidade de configuração manual.
</Aside>

Isso garante que todos os colaboradores tenham as mesmas ferramentas disponíveis
desde a primeira sessão do Claude Code no repositório.
