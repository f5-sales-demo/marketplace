Um **marketplace de plugins do Claude Code** é uma coleção curada de
plugins que estendem o Claude Code com novas habilidades, comandos e
agentes. Os marketplaces permitem que equipes compartilhem ferramentas
para que todos os desenvolvedores tenham as mesmas capacidades sem
configuração manual.

O **Marketplace f5-sales-demo** fornece plugins desenvolvidos para
a plataforma de documentação [f5-sales-demo](https://github.com/f5-sales-demo)
— um ecossistema de mais de 20 repositórios que
publicam sites Astro/Starlight no GitHub Pages usando um pipeline
de build compartilhado.

## Como Funciona

1. **Adicione o marketplace** ao Claude Code — isso o registra
   como uma fonte de plugins
2. **Instale plugins** do marketplace — cada plugin adiciona
   habilidades e comandos à sua sessão do Claude Code
3. **Use as ferramentas** — os plugins são ativados automaticamente com base no
   contexto, ou você invoca seus comandos diretamente

## Plugins Disponíveis

| Plugin | Categoria | Descrição |
|--------|-----------|-----------|
| [f5xc-docs-tools](/marketplace/plugins/f5xc-docs-tools/) | Produtividade | Ferramentas de validação e revisão de conteúdo MDX para repositórios de documentação |
| [f5xc-sales-engineer](/marketplace/plugins/f5xc-sales-engineer/) | Produtividade | Framework de persona para Engenheiro de Vendas — execução de demos, apresentação, Q&A, gerenciamento de ambiente |

## Início Rápido

Adicione o marketplace e instale o plugin de ferramentas de documentação:

```
/plugin marketplace add f5-sales-demo/marketplace
/plugin install f5xc-docs-tools@f5-sales-demo-marketplace
```

Consulte [Primeiros Passos](/marketplace/getting-started/) para um
guia completo.

## Arquitetura

Um marketplace é um repositório Git que contém um ou mais
plugins. Cada plugin empacota habilidades, comandos e agentes
em uma única unidade instalável.

```
Marketplace (marketplace.json)
└── Plugin (plugin.json)
    ├── Skills (SKILL.md)
    │   └── References (data files)
    ├── Commands (.md)
    └── Agents (.md)
```

- **Marketplace** — o registro. Lista os plugins disponíveis
  com metadados para descoberta e instalação.
- **Plugin** — a unidade instalável. Contém uma ou mais
  habilidades, comandos ou agentes.
- **Skill** — conhecimento contextual que o Claude ativa
  automaticamente com base na tarefa em questão.
- **Command** — um comando de barra que o usuário invoca diretamente
  (ex.: `/review-mdx`).
- **Agent** — um sub-agente especializado que lida com tarefas complexas
  e de múltiplas etapas de forma autônoma.
