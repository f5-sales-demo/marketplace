Un **mercado de plugins de Claude Code** es una colección curada de
plugins que amplían Claude Code con nuevas habilidades, comandos y
agentes. Los mercados permiten que los equipos compartan herramientas para que cada
desarrollador obtenga las mismas capacidades sin configuración manual.

El **Mercado f5-sales-demo** proporciona plugins creados para
la plataforma de documentación [f5-sales-demo](https://github.com/f5-sales-demo)
— un ecosistema de más de 20 repositorios que
publican sitios Astro/Starlight en GitHub Pages mediante una canalización
de compilación compartida.

## Cómo funciona

1. **Agregar el mercado** a Claude Code — esto lo registra
   como una fuente de plugins
2. **Instalar plugins** del mercado — cada plugin agrega
   habilidades y comandos a su sesión de Claude Code
3. **Usar las herramientas** — los plugins se activan automáticamente según el
   contexto, o puede invocar sus comandos directamente

## Inicio rápido

Agregue el mercado e instale el plugin de herramientas de documentación:

```
/plugin marketplace add f5-sales-demo/marketplace
/plugin install f5xc-docs-tools@f5-sales-demo-marketplace
```

Consulte [Primeros pasos](/marketplace/es/getting-started/) para un
recorrido completo.

## Arquitectura

Un mercado es un repositorio de Git que contiene uno o más
plugins. Cada plugin empaqueta habilidades, comandos y agentes
en una unidad instalable única.

```
Marketplace (marketplace.json)
└── Plugin (plugin.json)
    ├── Skills (SKILL.md)
    │   └── References (data files)
    ├── Commands (.md)
    └── Agents (.md)
```

- **Marketplace** — el registro. Lista los plugins disponibles
  con metadatos para descubrimiento e instalación.
- **Plugin** — la unidad instalable. Contiene una o más
  habilidades, comandos o agentes.
- **Skill** — conocimiento contextual que Claude activa
  automáticamente según la tarea en cuestión.
- **Command** — un comando slash que el usuario invoca directamente
  (p. ej., `/review-mdx`).
- **Agent** — un subagente especializado que gestiona tareas complejas
  de múltiples pasos de forma autónoma.
