El plugin **f5xc-platform** proporciona automatización completa para la
Plataforma F5 Distributed Cloud. Cubre operaciones en la consola web mediante
Chrome DevTools MCP, operaciones de API REST con reconocimiento de especificaciones mediante cURL,
autenticación multi-proveedor y análisis de configuración en
98 tipos de recursos en 38 dominios.

<Badge text="v3.1.1" variant="note" /> <Badge text="Productivity" variant="tip" />

## Instalación

```
/plugin install f5xc-platform@f5-sales-demo-marketplace
```

## Habilidades

### api-operations

Operaciones CRUD de API con reconocimiento de especificaciones que cubren 98 tipos de recursos en
38 dominios. Proporciona endpoints precisos, plantillas de payload,
ordenamiento de dependencias y composiciones de flujo de trabajo en varios pasos para
balanceadores de carga HTTP, grupos de origen, políticas WAF, zonas DNS,
certificados, sitios en la nube y más.

### config-analysis

Capacidad de análisis de configuración y asesoramiento. Analiza configuraciones
JSON de clientes para responder preguntas sobre postura de seguridad,
habilitación de funcionalidades, configuraciones de modo, exclusiones y mejores prácticas.
Admite preguntas y respuestas en múltiples turnos con contexto comprimido entre despachos.

### api-auth

Autenticación de API compatible con autenticación mediante encabezado de token de API y
autenticación con certificado P12, con procedimientos de validación de tokens.

### console-auth

Autenticación multi-proveedor para la consola web de F5 XC.
Detecta automáticamente el tipo de inicio de sesión: inicio de sesión nativo de F5 XC, SSO de Azure con
sesión en caché, o SSO de Azure con MFA de push verificado por DUO.

### console-navigator

Navegación determinista hacia secciones de la F5 XC Consola. Traduce
nombres de secciones a rutas URL, verifica el estado de la sesión y confirma
la carga de la página.

### platform-index / api-index / console-index

Enrutadores de intención que clasifican las solicitudes de los usuarios y las delegan al
skill y agente apropiados. Se activan automáticamente y no son
invocables por el usuario.

## Comandos

### /check-api-token

```
/f5xc-platform:check-api-token
```

Valida su token de API de F5 XC delegando en el agente
api-operator.

### /login-console

```
/f5xc-platform:login-console
```

Autentica en la consola de F5 Distributed Cloud mediante el
agente console-operator.

## Agentes

### api-operator

Agente de API REST autónomo que ejecuta secuencias de cURL y jq para
operaciones CRUD de recursos, validación de tokens y operaciones de configuración.
Se ejecuta en un subagente para mantener los payloads JSON detallados de la API fuera del
contexto de la sesión principal.

### console-operator

Agente autónomo de automatización de navegador que ejecuta secuencias de herramientas de Chrome DevTools MCP
para autenticación, navegación e interacciones con formularios.
Se ejecuta en un subagente para mantener las capturas del navegador con tokens fuera de
la sesión principal.

### config-analyzer

Agente de análisis de configuración de solo lectura. Analiza configuraciones JSON
de clientes en comparación con archivos de referencia de perfiles de recursos para responder
preguntas sobre postura de seguridad, habilitación de funcionalidades, transiciones de modo
y mejores prácticas.

<Aside type="caution">
  El agente config-analyzer es **efímero** &mdash; se inicializa de forma nueva
  para cada pregunta sin memoria entre despachos. Las preguntas de seguimiento
  deben incluir los hallazgos previos comprimidos en el prompt de despacho.
</Aside>
