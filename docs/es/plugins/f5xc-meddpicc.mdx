El plugin **f5xc-meddpicc** proporciona un marco integral de calificación de ventas y ejecución de acuerdos MEDDPICC para ventas complejas empresariales B2B. Transforma MEDDPICC de un ejercicio de verificación en un sistema de ejecución de acuerdos dinámico con calificación basada en evidencia, revisiones estructuradas de acuerdos y orientación práctica.

<Badge text="v1.0.4" variant="note" /> <Badge text="Productivity" variant="tip" />

## Instalación

```
/plugin install f5xc-meddpicc@f5-sales-demo-marketplace
```

## ¿Qué es MEDDPICC?

MEDDPICC es un marco de calificación y ejecución de acuerdos diseñado para ventas B2B complejas donde múltiples partes interesadas, riesgos técnicos y etapas de adquisición pueden descarrilar el impulso.

| Letra | Elemento | Propósito |
| ------ | ------- | ------- |
| **M** | Métricas | Resultados empresariales cuantificados que el cliente espera |
| **E** | Comprador económico | Individuo con autoridad final sobre el presupuesto |
| **D** | Criterios de decisión | Requisitos utilizados para evaluar opciones |
| **D** | Proceso de decisión | Ruta paso a paso desde la evaluación hasta el acuerdo |
| **P** | Proceso documental | Mecánica de adquisición (legal, seguridad, firma) |
| **I** | Identificación del problema | Problema empresarial urgente que genera necesidad |
| **C** | Campeón | Defensor interno con influencia |
| **C** | Competencia | Todas las alternativas (proveedores, desarrollo propio, no hacer nada) |

## Habilidades

### meddpicc-coach

Orientación integral sobre la metodología MEDDPICC que se activa automáticamente cuando se discute la calificación de acuerdos, la salud del pipeline, la precisión de las previsiones o cualquier elemento MEDDPICC. Proporciona guía basada en evidencia, recomendaciones de ritmo operativo y enlaces a material de referencia detallado.

<Aside type="tip" title="Activación automática">
  Esta habilidad se activa automáticamente cuando es relevante — no se
  necesita ningún comando slash.
</Aside>

### deal-qualification

Produce un cuadro de mando MEDDPICC puntuado (0&ndash;24) con análisis de brechas, evaluación de riesgos y recomendación de previsión. Cada elemento se puntúa de 0&ndash;3 en función de la evidencia verificada.

### deal-review

Facilita inspecciones estructuradas semanales de acuerdos con seguimiento de variaciones de evidencia, identificación de riesgos y elementos de acción con responsables asignados y fechas límite.

### mutual-action-plan

Construye un Plan de Acción Mutuo orientado al cliente vinculado a los elementos MEDDPICC. Incluye hitos de evaluación, hoja de ruta de adquisición, preparación para la implementación y mapeo de partes interesadas.

### champion-test

Aplica una evaluación de 5 criterios (Poder, Victoria personal, Acceso, Inteligencia, Acción) con puntuación de 0&ndash;15 para determinar si su contacto interno es un verdadero campeón, un campeón emergente, un asesor o un contacto favorable.

## Comandos

| Comando | Propósito |
| ------- | ------- |
| `/f5xc-meddpicc:qualify-deal [account]` | Puntuar un acuerdo con el cuadro de mando MEDDPICC |
| `/f5xc-meddpicc:deal-review [account]` | Facilitar la inspección semanal de acuerdos |
| `/f5xc-meddpicc:champion-test [contact]` | Evaluar si un contacto es un verdadero campeón |
| `/f5xc-meddpicc:build-map [account]` | Construir un Plan de Acción Mutuo |

## Agentes

### deal-analyst

Agente de investigación de solo lectura que analiza la salud de los acuerdos revisando documentación local, notas de reuniones y planes de cuentas. Produce informes estructurados de evaluación MEDDPICC con citas de evidencia, análisis de brechas y recomendaciones de acción priorizadas.

## Biblioteca de referencia

El plugin incluye material de referencia integral que se carga bajo demanda:

- **Marco MEDDPICC** &mdash; definiciones completas y requisitos de evidencia para los 8 elementos
- **Preguntas de descubrimiento** &mdash; preguntas probadas en campo para cada elemento a lo largo del ciclo de ventas
- **Roles y responsabilidades** &mdash; matriz de propiedad RACI por rol de equipo (AE, SE, CSM, PS, Producto, Canal, Deal Desk, Legal)
- **Etapas de avance del acuerdo** &mdash; lo que debe cumplirse antes de avanzar (desde la Calificación inicial hasta el Cierre ganado)
- **Anti-patrones** &mdash; 10 modos de fallo comunes con soluciones basadas en evidencia
- **Rúbrica de puntuación** &mdash; criterios de puntuación objetivos de 0&ndash;3 por elemento
- **Plantillas de revisión** &mdash; conjuntos de preguntas de inspección semanal con indicadores de alerta
- **Plantillas MAP** &mdash; estructura del plan de acción orientada al cliente con hojas de ruta de evaluación, adquisición e implementación
