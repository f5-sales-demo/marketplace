El plugin **f5xc-brand** aplica la identidad de marca corporativa F5
en todos los tipos de contenido. Cubre la paleta F5 de 45 colores,
reglas tipográficas (titulares Neusa Next Pro Wide, cuerpo Proxima Nova),
uso exclusivo de logotipos en SVG, selección de paquetes de iconos, accesibilidad
WCAG AA y terminología de productos.

<Badge text="v1.0.2" variant="note" /> <Badge text="Productivity" variant="tip" />

## Instalación

```
/plugin install f5xc-brand@f5-sales-demo-marketplace
```

## Habilidades

### brand-guardian

Aplicación proactiva de la identidad de marca para toda la creación de contenido.
Se activa automáticamente al crear documentación, diagramas,
presentaciones, miniaturas de vídeo, gráficos para redes sociales, diagramas
Mermaid o dibujos en Excalidraw. Garantiza el uso correcto de colores F5,
tipografía, logotipos, iconos, tono de voz y accesibilidad.

### brand-reviewer

Revisa el contenido existente para verificar el cumplimiento de la marca F5. Audita colores,
tipografía, formato de logotipo, selección de iconos, texto alternativo, ratios de contraste
y terminología de productos. Genera informes estructurados con
VIOLACIONES, ADVERTENCIAS y SUGERENCIAS.

### visual-content

Reglas específicas por formato para contenido visual de marca. Proporciona
orientación para diagramas Mermaid, dibujos en Excalidraw, capturas de pantalla,
presentaciones de PowerPoint, miniaturas de YouTube, guiones de vídeo y
gráficos para redes sociales, con selección de paquetes de iconos, diseños de diapositivas,
fuentes de reserva y dimensiones específicas por plataforma.

## Comandos

### /review-brand

```
/f5xc-brand:review-brand [path-or-glob]
```

Revisa el contenido para verificar el cumplimiento de la marca F5. Acepta una ruta
opcional o un patrón glob para delimitar el alcance de la revisión. Sin argumentos, revisa
todo el contenido del directorio actual.
