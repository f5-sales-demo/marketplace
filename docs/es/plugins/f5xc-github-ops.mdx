El plugin **f5xc-github-ops** proporciona Automatización exclusiva de operaciones de GitHub
para repositorios de f5-sales-demo. Gestiona el ciclo de vida completo
de Git y GitHub, desde la creación de incidencias hasta el monitoreo
posterior a la fusión, aplicando el desarrollo basado en incidencias y
las verificaciones de lint previas al commit.

<Badge text="v2.2.2" variant="note" /> <Badge text="Productividad" variant="tip" />

## Instalación

```
/plugin install f5xc-github-ops@f5-sales-demo-marketplace
```

## Habilidades

### workflow-lifecycle

Flujo de trabajo de gobernanza del repositorio que gestiona la creación
de incidencias, convenciones de nomenclatura de ramas, creación de PR,
sondeo de CI con retroalimentación de errores en las incidencias,
monitoreo posterior a la fusión, verificación y criterios de finalización
de tareas. Se activa cuando se dice "commit", "push", "create a PR",
"merge" o se realiza cualquier operación de Git.

### upstream-contribution

Flujo de trabajo de contribución upstream para el desarrollo basado en
forks. Gestiona la detección de forks, investigación upstream
(incidencias, PR, CONTRIBUTING.md, patrones de CI), creación de
incidencias y PR entre repositorios, gestión de incidencias de
seguimiento local, monitoreo de CI upstream y resolución guiada.

## Agentes

### github-ops

Agente exclusivo para todas las operaciones mecánicas de Git y GitHub.
Gestiona las verificaciones de lint previas al commit, creación de
incidencias, creación de ramas, preparación, commit, push, creación de
PR, sondeo de CI con retroalimentación de errores, reintento en fallos
de infraestructura, monitoreo posterior a la fusión, limpieza de ramas
y gestión de configuraciones del repositorio a través de la API de
GitHub.

Devuelve informes de estado estructurados: `COMPLETE`,
`PRE_COMMIT_FAILED`, `CI_FAILED`, `BLOCKED` o `FAILED`.

<Aside type="caution">
  Este agente es el gestor **exclusivo** de las operaciones de Git.
  Nunca ejecute `git commit`, `git push` ni `gh pr create` directamente
  en repositorios que utilicen este plugin.
</Aside>
