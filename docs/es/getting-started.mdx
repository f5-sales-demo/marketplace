Esta guía le explica cómo agregar el marketplace f5-sales-demo
a Claude Code e instalar su primer plugin.

## Requisitos previos

- Claude Code **v1.0.33** o posterior (compatibilidad con marketplace de plugins)
- Una cuenta de GitHub con acceso a la organización
  [f5-sales-demo](https://github.com/f5-sales-demo)

## Instalación

<Steps>

1. **Agregar el marketplace**

   En cualquier sesión de Claude Code, ejecute:

   ```
   /plugin marketplace add f5-sales-demo/marketplace
   ```

   Esto registra el marketplace como una fuente de plugins. Solo
   necesita hacerlo una vez.

2. **Explorar los plugins disponibles**

   Liste lo que ofrece el marketplace:

   ```
   /plugin marketplace list f5-sales-demo-marketplace
   ```

3. **Instalar un plugin**

   Instale el plugin que necesita:

   ```
   /plugin install f5xc-docs-tools@f5-sales-demo-marketplace
   /plugin install f5xc-sales-engineer@f5-sales-demo-marketplace
   ```

4. **Usar el plugin**

   Navegue a cualquier repositorio de contenido de f5-sales-demo.

   Para las **herramientas de docs**, ejecute el comando de revisión MDX:

   ```
   /f5xc-docs-tools:review-mdx
   ```

   Para **sales engineer**, utilice frases de activación como
   "prepare the demo", "run the demo" o "walk through
   the demo" — el plugin se activa automáticamente.

</Steps>

## Actualización de plugins

Para actualizar un plugin instalado a la versión más reciente:

```
/plugin update f5xc-docs-tools@f5-sales-demo-marketplace
/plugin update f5xc-sales-engineer@f5-sales-demo-marketplace
```

## Desinstalación

Elimine un plugin:

```
/plugin uninstall f5xc-docs-tools@f5-sales-demo-marketplace
/plugin uninstall f5xc-sales-engineer@f5-sales-demo-marketplace
```

Elimine el marketplace por completo:

```
/plugin marketplace remove f5-sales-demo-marketplace
```

## Configuración para equipos

Para preconfigurar el marketplace para todo su equipo, agréguelo
al archivo `.claude/settings.json` de su proyecto:

```json
{
  "extraKnownMarketplaces": [
    "f5-sales-demo/marketplace"
  ],
  "enabledPlugins": [
    "f5xc-docs-tools@f5-sales-demo-marketplace",
    "f5xc-sales-engineer@f5-sales-demo-marketplace"
  ]
}
```

<Aside type="tip" title="Configuración compartida">
  Confirmar `.claude/settings.json` en su repositorio significa que
  cada miembro del equipo obtiene el marketplace y los plugins
  automáticamente — sin necesidad de configuración manual.
</Aside>

Esto garantiza que cada colaborador tenga las mismas herramientas disponibles
desde su primera sesión de Claude Code en el repositorio.
