Esta guía le explica cómo añadir el marketplace f5xc-salesdemos
a Claude Code e instalar su primer plugin.

## Requisitos previos

- Claude Code **v1.0.33** o posterior (compatibilidad con marketplace de plugins)
- Una cuenta de GitHub con acceso a la organización
  [f5xc-salesdemos](https://github.com/f5xc-salesdemos)

## Instalación

<Steps>

1. **Añadir el marketplace**

   En cualquier sesión de Claude Code, ejecute:

   ```
   /plugin marketplace add f5xc-salesdemos/marketplace
   ```

   Esto registra el marketplace como fuente de plugins. Solo
   necesita hacerlo una vez.

2. **Explorar los plugins disponibles**

   Liste lo que ofrece el marketplace:

   ```
   /plugin marketplace list f5xc-salesdemos-marketplace
   ```

3. **Instalar un plugin**

   Instale el plugin que necesite:

   ```
   /plugin install f5xc-docs-tools@f5xc-salesdemos-marketplace
   /plugin install f5xc-sales-engineer@f5xc-salesdemos-marketplace
   ```

4. **Usar el plugin**

   Navegue a cualquier repositorio de contenido de f5xc-salesdemos.

   Para **docs tools**, ejecute el comando de revisión MDX:

   ```
   /f5xc-docs-tools:review-mdx
   ```

   Para **sales engineer**, utilice frases activadoras como
   "prepare the demo", "run the demo" o "walk through
   the demo" — el plugin se activa automáticamente.

</Steps>

## Actualización de plugins

Para actualizar un plugin instalado a la última versión:

```
/plugin update f5xc-docs-tools@f5xc-salesdemos-marketplace
/plugin update f5xc-sales-engineer@f5xc-salesdemos-marketplace
```

## Desinstalación

Eliminar un plugin:

```
/plugin uninstall f5xc-docs-tools@f5xc-salesdemos-marketplace
/plugin uninstall f5xc-sales-engineer@f5xc-salesdemos-marketplace
```

Eliminar el marketplace por completo:

```
/plugin marketplace remove f5xc-salesdemos-marketplace
```

## Configuración de equipo

Para preconfigurar el marketplace para todo su equipo, añádalo
al archivo `.claude/settings.json` de su proyecto:

```json
{
  "extraKnownMarketplaces": [
    "f5xc-salesdemos/marketplace"
  ],
  "enabledPlugins": [
    "f5xc-docs-tools@f5xc-salesdemos-marketplace",
    "f5xc-sales-engineer@f5xc-salesdemos-marketplace"
  ]
}
```

<Aside type="tip" title="Configuración compartida">
  Incluir `.claude/settings.json` en el repositorio mediante commit significa
  que cada miembro del equipo obtiene el marketplace y los plugins
  automáticamente — sin necesidad de configuración manual.
</Aside>

Esto garantiza que cada colaborador disponga de las mismas herramientas
desde su primera sesión de Claude Code en el repositorio.
