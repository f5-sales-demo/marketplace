एक **Claude Code प्लगइन मार्केटप्लेस** प्लगइन का एक क्यूरेटेड संग्रह है
जो Claude Code को नए कौशल, कमांड और एजेंट के साथ विस्तारित करता है।
मार्केटप्लेस टीमों को टूलिंग साझा करने देता है ताकि हर डेवलपर को
मैन्युअल सेटअप के बिना समान क्षमताएं मिलें।

**f5xc-salesdemos मार्केटप्लेस** [f5xc-salesdemos](https://github.com/f5xc-salesdemos)
डॉक्यूमेंटेशन प्लेटफ़ॉर्म के लिए बनाए गए प्लगइन प्रदान करता है — यह 20+ रिपॉज़िटरीज़
का एक इकोसिस्टम है जो एक साझा बिल्ड पाइपलाइन का उपयोग करके Astro/Starlight
साइट्स को GitHub Pages पर प्रकाशित करता है।

## यह कैसे काम करता है

1. **मार्केटप्लेस जोड़ें** Claude Code में — यह इसे एक प्लगइन स्रोत
   के रूप में पंजीकृत करता है
2. **प्लगइन इंस्टॉल करें** मार्केटप्लेस से — प्रत्येक प्लगइन आपके
   Claude Code सत्र में कौशल और कमांड जोड़ता है
3. **टूल्स का उपयोग करें** — प्लगइन संदर्भ के आधार पर स्वचालित रूप से
   सक्रिय होते हैं, या आप उनके कमांड सीधे इनवोक कर सकते हैं

## उपलब्ध प्लगइन

## त्वरित प्रारंभ

मार्केटप्लेस जोड़ें और docs tools प्लगइन इंस्टॉल करें:

```
/plugin marketplace add f5xc-salesdemos/marketplace
/plugin install f5xc-docs-tools@f5xc-salesdemos-marketplace
```

पूर्ण वॉकथ्रू के लिए [शुरुआत करना](/marketplace/getting-started/) देखें।

## आर्किटेक्चर

एक मार्केटप्लेस एक Git रिपॉज़िटरी है जिसमें एक या अधिक प्लगइन होते हैं।
प्रत्येक प्लगइन कौशल, कमांड और एजेंट को एक एकल इंस्टॉल करने योग्य
इकाई में पैकेज करता है।

```
Marketplace (marketplace.json)
└── Plugin (plugin.json)
    ├── Skills (SKILL.md)
    │   └── References (data files)
    ├── Commands (.md)
    └── Agents (.md)
```

- **मार्केटप्लेस** — रजिस्ट्री। खोज और इंस्टॉलेशन के लिए
  मेटाडेटा के साथ उपलब्ध प्लगइन सूचीबद्ध करता है।
- **प्लगइन** — इंस्टॉल करने योग्य इकाई। इसमें एक या अधिक
  कौशल, कमांड, या एजेंट होते हैं।
- **कौशल** — प्रासंगिक ज्ञान जिसे Claude कार्य के आधार पर
  स्वचालित रूप से सक्रिय करता है।
- **कमांड** — एक स्लैश कमांड जिसे उपयोगकर्ता सीधे इनवोक करता है
  (उदा., `/review-mdx`)।
- **एजेंट** — एक विशेष उप-एजेंट जो जटिल, बहु-चरणीय कार्यों को
  स्वायत्त रूप से संभालता है।
