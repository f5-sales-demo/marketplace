एक **Claude Code प्लगइन मार्केटप्लेस** प्लगइन्स का एक क्यूरेटेड संग्रह है
जो Claude Code को नए कौशल, कमांड और एजेंट्स के साथ विस्तारित करता है।
मार्केटप्लेस टीमों को टूलिंग साझा करने देता है ताकि हर डेवलपर को
बिना मैन्युअल सेटअप के समान क्षमताएं मिलें।

**f5-sales-demo मार्केटप्लेस** उन प्लगइन्स प्रदान करता है जो
[f5-sales-demo](https://github.com/f5-sales-demo)
दस्तावेज़ीकरण प्लेटफ़ॉर्म के लिए बनाए गए हैं — 20+ रिपॉजिटरी का एक इकोसिस्टम जो
साझा बिल्ड पाइपलाइन का उपयोग करके GitHub Pages पर Astro/Starlight साइट्स प्रकाशित करता है।

## यह कैसे काम करता है

1. **मार्केटप्लेस जोड़ें** Claude Code में — यह इसे एक प्लगइन स्रोत के रूप में पंजीकृत करता है
2. **प्लगइन्स इंस्टॉल करें** मार्केटप्लेस से — प्रत्येक प्लगइन आपके Claude Code सत्र में कौशल और कमांड जोड़ता है
3. **उपकरण उपयोग करें** — प्लगइन्स संदर्भ के आधार पर स्वचालित रूप से सक्रिय होते हैं, या आप उनके कमांड सीधे आमंत्रित करते हैं

## उपलब्ध प्लगइन्स

| प्लगइन | श्रेणी | विवरण |
|--------|----------|-------------|
| [f5xc-docs-tools](/marketplace/plugins/f5xc-docs-tools/) | उत्पादकता | दस्तावेज़ीकरण रिपॉजिटरी के लिए MDX कंटेंट सत्यापन और समीक्षा उपकरण |
| [f5xc-sales-engineer](/marketplace/plugins/f5xc-sales-engineer/) | उत्पादकता | Sales Engineer पर्सोना फ्रेमवर्क — डेमो निष्पादन, प्रस्तुति, Q&A, पर्यावरण प्रबंधन |

## त्वरित प्रारंभ

मार्केटप्लेस जोड़ें और docs tools प्लगइन इंस्टॉल करें:

```
/plugin marketplace add f5-sales-demo/marketplace
/plugin install f5xc-docs-tools@f5-sales-demo-marketplace
```

पूर्ण वॉकथ्रू के लिए [प्रारंभ करना](/marketplace/getting-started/) देखें।

## आर्किटेक्चर

एक मार्केटप्लेस एक Git रिपॉजिटरी है जिसमें एक या अधिक प्लगइन्स होते हैं।
प्रत्येक प्लगइन कौशल, कमांड और एजेंट्स को एक एकल इंस्टॉल करने योग्य इकाई में पैकेज करता है।

```
Marketplace (marketplace.json)
└── Plugin (plugin.json)
    ├── Skills (SKILL.md)
    │   └── References (data files)
    ├── Commands (.md)
    └── Agents (.md)
```

- **मार्केटप्लेस** — रजिस्ट्री। खोज और इंस्टॉलेशन के लिए मेटाडेटा के साथ उपलब्ध प्लगइन्स सूचीबद्ध करता है।
- **प्लगइन** — इंस्टॉल करने योग्य इकाई। इसमें एक या अधिक कौशल, कमांड, या एजेंट्स होते हैं।
- **कौशल** — प्रासंगिक ज्ञान जिसे Claude कार्य के आधार पर स्वचालित रूप से सक्रिय करता है।
- **कमांड** — एक स्लैश कमांड जिसे उपयोगकर्ता सीधे आमंत्रित करता है (जैसे, `/review-mdx`)।
- **एजेंट** — एक विशेष सब-एजेंट जो जटिल, बहु-चरणीय कार्यों को स्वायत्त रूप से संभालता है।
