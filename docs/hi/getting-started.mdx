यह मार्गदर्शिका आपको Claude Code में f5-sales-demo
मार्केटप्लेस जोड़ने और अपना पहला प्लगइन इंस्टॉल करने की प्रक्रिया से परिचित कराती है।

## पूर्वापेक्षाएँ

- Claude Code **v1.0.33** या बाद का संस्करण (प्लगइन मार्केटप्लेस समर्थन सहित)
- [f5-sales-demo](https://github.com/f5-sales-demo)
  संगठन तक पहुँच वाला एक GitHub खाता

## इंस्टॉलेशन

<Steps>

1. **मार्केटप्लेस जोड़ें**

   किसी भी Claude Code सत्र में, यह चलाएँ:

   ```
   /plugin marketplace add f5-sales-demo/marketplace
   ```

   इससे मार्केटप्लेस एक प्लगइन स्रोत के रूप में पंजीकृत हो जाता है। आपको
   यह केवल एक बार करना होगा।

2. **उपलब्ध प्लगइन देखें**

   मार्केटप्लेस में उपलब्ध विकल्पों की सूची देखें:

   ```
   /plugin marketplace list f5-sales-demo-marketplace
   ```

3. **एक प्लगइन इंस्टॉल करें**

   आवश्यक प्लगइन इंस्टॉल करें:

   ```
   /plugin install f5xc-docs-tools@f5-sales-demo-marketplace
   /plugin install f5xc-sales-engineer@f5-sales-demo-marketplace
   ```

4. **प्लगइन का उपयोग करें**

   किसी भी f5-sales-demo कंटेंट रिपॉज़िटरी पर जाएँ।

   **docs tools** के लिए, MDX समीक्षा कमांड चलाएँ:

   ```
   /f5xc-docs-tools:review-mdx
   ```

   **sales engineer** के लिए, "prepare the demo", "run the demo", या "walk through
   the demo" जैसे ट्रिगर वाक्यांशों का उपयोग करें — प्लगइन स्वचालित रूप से सक्रिय हो जाता है।

</Steps>

## प्लगइन अपडेट करना

किसी इंस्टॉल किए गए प्लगइन को नवीनतम संस्करण में अपडेट करने के लिए:

```
/plugin update f5xc-docs-tools@f5-sales-demo-marketplace
/plugin update f5xc-sales-engineer@f5-sales-demo-marketplace
```

## अनइंस्टॉल करना

कोई प्लगइन हटाएँ:

```
/plugin uninstall f5xc-docs-tools@f5-sales-demo-marketplace
/plugin uninstall f5xc-sales-engineer@f5-sales-demo-marketplace
```

मार्केटप्लेस को पूरी तरह हटाएँ:

```
/plugin marketplace remove f5-sales-demo-marketplace
```

## टीम कॉन्फ़िगरेशन

अपनी पूरी टीम के लिए मार्केटप्लेस को पूर्व-कॉन्फ़िगर करने हेतु, इसे
अपने प्रोजेक्ट की `.claude/settings.json` में जोड़ें:

```json
{
  "extraKnownMarketplaces": [
    "f5-sales-demo/marketplace"
  ],
  "enabledPlugins": [
    "f5xc-docs-tools@f5-sales-demo-marketplace",
    "f5xc-sales-engineer@f5-sales-demo-marketplace"
  ]
}
```

<Aside type="tip" title="साझा सेटिंग्स">
  `.claude/settings.json` को अपनी रिपॉज़िटरी में कमिट करने का अर्थ है कि
  प्रत्येक टीम सदस्य को मार्केटप्लेस और प्लगइन स्वचालित रूप से मिल जाते हैं —
  किसी मैन्युअल सेटअप की आवश्यकता नहीं।
</Aside>

इससे यह सुनिश्चित होता है कि प्रत्येक योगदानकर्ता के पास रिपॉज़िटरी में अपने
पहले Claude Code सत्र से ही समान उपकरण उपलब्ध हों।
