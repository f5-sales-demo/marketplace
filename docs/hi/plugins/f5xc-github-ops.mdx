**f5xc-github-ops** प्लगइन f5xc-salesdemos रिपॉज़िटरीज़ के लिए विशेष GitHub ऑपरेशंस
ऑटोमेशन प्रदान करता है। यह इश्यू क्रिएशन से लेकर पोस्ट-मर्ज
मॉनिटरिंग तक संपूर्ण Git और GitHub लाइफ़साइकल को संभालता है,
इश्यू-ड्रिवन डेवलपमेंट और प्री-कमिट लिंट गेट्स को लागू करता है।

<Badge text="v2.2.2" variant="note" /> <Badge text="Productivity" variant="tip" />

## इंस्टॉलेशन

```
/plugin install f5xc-github-ops@f5xc-salesdemos-marketplace
```

## स्किल्स

### workflow-lifecycle

रिपॉज़िटरी गवर्नेंस वर्कफ़्लो जो इश्यू क्रिएशन, ब्रांच
नेमिंग कन्वेंशन, PR क्रिएशन, एरर फ़ीडबैक के साथ CI पोलिंग
इश्यूज़ पर, पोस्ट-मर्ज मॉनिटरिंग, वेरिफ़िकेशन, और टास्क
कम्प्लीशन क्राइटेरिया को संभालता है। जब आप "commit", "push",
"create a PR", "merge" कहते हैं, या कोई भी Git ऑपरेशन करते हैं
तब यह सक्रिय होता है।

### upstream-contribution

फ़ोर्क-आधारित डेवलपमेंट के लिए अपस्ट्रीम कॉन्ट्रिब्यूशन वर्कफ़्लो। फ़ोर्क
डिटेक्शन, अपस्ट्रीम रिसर्च (इश्यूज़, PRs, CONTRIBUTING.md,
CI पैटर्न), क्रॉस-रिपो इश्यू और PR क्रिएशन, लोकल ट्रैकिंग
इश्यू मैनेजमेंट, अपस्ट्रीम CI मॉनिटरिंग, और गाइडेड रिज़ॉल्यूशन
को संभालता है।

## एजेंट्स

### github-ops

सभी मैकेनिकल Git और GitHub ऑपरेशंस के लिए विशेष एजेंट।
प्री-कमिट लिंट गेट्स, इश्यू क्रिएशन, ब्रांच क्रिएशन,
स्टेजिंग, कमिटिंग, पुशिंग, PR क्रिएशन, एरर फ़ीडबैक के साथ
CI पोलिंग, इन्फ्रास्ट्रक्चर फ़ेल्योर रीट्राय, पोस्ट-मर्ज
मॉनिटरिंग, ब्रांच क्लीनअप, और GitHub API के माध्यम से
रिपॉज़िटरी सेटिंग्स मैनेजमेंट को संभालता है।

स्ट्रक्चर्ड स्टेटस रिपोर्ट्स लौटाता है: `COMPLETE`,
`PRE_COMMIT_FAILED`, `CI_FAILED`, `BLOCKED`, या `FAILED`।

<Aside type="caution">
  यह एजेंट Git ऑपरेशंस का **विशेष** हैंडलर है।
  इस प्लगइन का उपयोग करने वाली रिपॉज़िटरीज़ में कभी भी सीधे
  `git commit`, `git push`, या `gh pr create` न चलाएं।
</Aside>
