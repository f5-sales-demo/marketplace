**osint-framework** प्लगइन OSINT उपकरण कैटलॉग और
जांच कौशल प्रदान करता है — [osintframework.com](https://osintframework.com) से मैप किए गए
34 श्रेणियों में 1,064 निःशुल्क इंटेलिजेंस-संग्रह उपकरण।
इसमें श्रेणी-आधारित कौशल, निष्पादन योग्य जांच पाइपलाइन,
CLI उपकरण निष्पादन, OPSEC-जागरूक वर्कफ़्लो, और क्रॉस-इन्वेस्टिगेशन
सहसंबंध के लिए एक स्थायी एंटिटी ग्राफ़ शामिल हैं।

<Badge text="v1.0.1" variant="note" /> <Badge text="Security" variant="tip" />

## इंस्टॉलेशन

```
/plugin install osint-framework@f5xc-salesdemos-marketplace
```

## कमांड

### /osint-investigate

किसी भी लक्ष्य के विरुद्ध पूर्ण OSINT जांच चलाएं। स्वचालित रूप से
लक्ष्य प्रकार (ईमेल, डोमेन, IP, उपयोगकर्ता नाम, कंपनी,
व्यक्ति का नाम) का पता लगाता है और सही श्रेणी कौशल पर रूट करता है।

```
/osint-investigate j.smith@example.com
/osint-investigate example.com
/osint-investigate 198.51.100.1
/osint-investigate example-user
```

### /osint-search

किसी क्षमता क्वेरी से मेल खाने वाले उपकरणों के लिए उपकरण कैटलॉग खोजें।

```
/osint-search subdomain enumeration
/osint-search email breach check
/osint-search satellite imagery
```

### /osint-catalog

श्रेणी के अनुसार पूर्ण उपकरण कैटलॉग ब्राउज़ करें। बिना तर्क के
उपकरण गणना के साथ सभी 34 श्रेणियां दिखाता है।

```
/osint-catalog
/osint-catalog domain
/osint-catalog threat-intel
```

## कौशल

### osint-index

शीर्ष-स्तरीय आशय राउटर। जब आप "find information about", "investigate",
"look up", "background check" कहते हैं या किसी ओपन-सोर्स इंटेलिजेंस कार्य
का वर्णन करते हैं तो स्वतः-सक्रिय होता है। लक्ष्य प्रकार के आधार पर
सही श्रेणी कौशल पर रूट करता है।

| लक्ष्य | रूट करता है |
|--------|----------|
| उपयोगकर्ता नाम या हैंडल | `username-recon` |
| ईमेल पता | `email-recon` |
| डोमेन नाम | `domain-recon` |
| IP पता | `ip-address-recon` |
| व्यक्ति का नाम + कंपनी | `people-search` + `business-records` |
| कंपनी / संगठन | `business-records` + `domain-recon` |
| मैलवेयर हैश | `malicious-file-analysis` + `threat-intelligence` |

### osint-catalog

उपकरण कैटलॉग ब्राउज़र। पूर्ण उपकरण रजिस्ट्री, श्रेणी इंडेक्स,
जांच पाइपलाइन, रेट-लिमिट संदर्भ और सहसंबंध इंजन दस्तावेज़ीकरण तक
पहुँच प्रदान करता है।

## एजेंट

### osint-investigator

स्वायत्त मल्टी-टूल जांच ऑर्केस्ट्रेटर। पूर्ण जांच वर्कफ़्लो चलाता है —
उपकरण उपलब्धता जांच, लक्ष्य प्रकार पहचान, अनुक्रमिक उपकरण निष्पादन
(पहले निष्क्रिय), परिणाम एकत्रीकरण, और विश्वास स्तरों तथा OPSEC
वर्गीकरणों के साथ संरचित रिपोर्ट निर्माण।

एंटिटी ग्राफ़ के साथ एकीकृत होता है: प्रत्येक उपकरण परिणाम
क्रॉस-इन्वेस्टिगेशन सहसंबंध के लिए `/tmp/osint-graph/` में
एंटिटी और संबंध बनाता है।

<Aside type="caution">
  सक्रिय रिकॉन उपकरण (nmap, nuclei, sherlock) लक्ष्य
  इंफ्रास्ट्रक्चर से सीधा संपर्क करते हैं। OPSEC चेतावनी हुक
  सक्रिय उपकरण चलाने से पहले सचेत करता है। हमेशा सुनिश्चित करें
  कि आपके पास अनुमति है।
</Aside>

### osint-researcher

रीड-ओनली कैटलॉग लुकअप एजेंट। किसी दिए गए कार्य के लिए सर्वोत्तम
उपकरणों की सिफारिश करने हेतु संदर्भ फ़ाइलें खोजता है। उपकरण कभी
निष्पादित नहीं करता — केवल अनुशंसा करता है और समझाता है। उपकरण
खोज के लिए श्रेणी कौशल द्वारा आंतरिक रूप से उपयोग किया जाता है।

### osint-executor

सिंगल-टूल CLI निष्पादन एजेंट। एक बार में एक उपकरण विशिष्ट
मापदंडों के साथ चलाता है, आउटपुट पार्स करता है, और संरचित परिणाम
लौटाता है। 20+ उपकरणों के लिए सटीक फ़्लैग, आउटपुट फ़ॉर्मेट और
jq पार्सिंग पैटर्न के साथ प्रति-उपकरण निष्पादन रेसिपी शामिल हैं।

## रेट सीमा

सभी API कॉल HTTP 429/503 पर अंतर्निहित एक्सपोनेंशियल बैकऑफ़ के साथ
`osint_curl` हेल्पर का उपयोग करती हैं। प्रमुख सीमाएं:

| API | निःशुल्क सीमा | एनव वेरिएबल |
|-----|-----------|---------|
| ipinfo.io | 1,000/दिन | — |
| crt.sh | 60/मिनट | — |
| NVD | 5/30s (बिना की) | `NVD_API_KEY` |
| OpenCorporates | की आवश्यक | `OPENCORPORATES_API_KEY` |
| Shodan | 1/सेकंड | `SHODAN_API_KEY` |
| GitHub API | 60/घंटा (बिना auth) | `GH_TOKEN` |

जिन APIs को की आवश्यकता है, यदि एनव वेरिएबल सेट नहीं है तो वे
स्वचालित रूप से एक स्पष्ट संदेश के साथ छोड़ दी जाती हैं।

## सहसंबंध इंजन

प्लगइन में `/tmp/osint-graph/` पर एक स्थायी JSON एंटिटी ग्राफ़
शामिल है जो जांचों में निष्कर्ष संचित करता है।

```bash
source plugins/osint-framework/scripts/osint-graph.sh
osint_graph_init

# Add entities
P=$(osint_entity_add "person" "J. Smith" --tool github-api)
C=$(osint_entity_add "company" "F5" --tool github-api)

# Link them
osint_rel_add "$P" "$C" "works_at" --tool github-api

# Query: all entities within 2 hops of the person
osint_graph_query "$P" 2

# Generate report
osint_graph_report
```

**विश्वास स्कोरिंग** मल्टी-सोर्स फ्यूज़न का उपयोग करती है:
प्रत्येक पुष्टि करने वाले स्रोत के लिए `confidence = 1 - product(1 - ci)`।
0.80 पर दो स्रोत → 0.96 संयुक्त विश्वास।

## कानूनी सूचना

सभी उपकरण केवल सार्वजनिक रूप से उपलब्ध जानकारी का उपयोग करते हैं।
उपयोगकर्ता लागू कानूनों, विनियमों और प्लेटफ़ॉर्म सेवा की शर्तों के
अनुपालन को सुनिश्चित करने के लिए जिम्मेदार हैं। OSINT उपकरणों का
उपयोग कभी भी अनधिकृत पहुँच, उत्पीड़न या किसी भी अवैध गतिविधि के
लिए न करें।
