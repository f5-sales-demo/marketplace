**f5xc-platform** प्लगइन F5 Distributed Cloud प्लेटफ़ॉर्म के लिए संपूर्ण स्वचालन प्रदान करता है। यह Chrome DevTools MCP के माध्यम से वेब कंसोल संचालन, cURL के माध्यम से spec-aware REST API संचालन, मल्टी-प्रोवाइडर प्रमाणीकरण, और 38 डोमेन में 98 संसाधन प्रकारों में कॉन्फ़िगरेशन विश्लेषण को कवर करता है।

<Badge text="v3.1.1" variant="note" /> <Badge text="Productivity" variant="tip" />

## स्थापना

```
/plugin install f5xc-platform@f5-sales-demo-marketplace
```

## कौशल

### api-operations

38 डोमेन में 98 संसाधन प्रकारों को कवर करने वाले spec-aware API CRUD संचालन। HTTP लोड बैलेंसर, ऑरिजिन पूल, WAF नीतियों, DNS ज़ोन, प्रमाणपत्र, क्लाउड साइट, और अधिक के लिए सटीक एंडपॉइंट, पेलोड टेम्पलेट, निर्भरता क्रम, और मल्टी-स्टेप वर्कफ़्लो संयोजन प्रदान करता है।

### config-analysis

कॉन्फ़िगरेशन विश्लेषण और सलाहकार क्षमता। सुरक्षा स्थिति, फ़ीचर सक्रियण, मोड सेटिंग, अपवर्जन, और सर्वोत्तम प्रथाओं के बारे में प्रश्नों का उत्तर देने के लिए ग्राहक JSON कॉन्फ़िगरेशन का विश्लेषण करता है। डिस्पैच के बीच संकुचित संदर्भ के साथ मल्टी-टर्न Q&A का समर्थन करता है।

### api-auth

API टोकन हेडर प्रमाणीकरण और टोकन सत्यापन प्रक्रियाओं के साथ P12 प्रमाणपत्र प्रमाणीकरण का समर्थन करने वाला API प्रमाणीकरण।

### console-auth

F5 XC वेब कंसोल के लिए मल्टी-प्रोवाइडर प्रमाणीकरण। लॉगिन प्रकार का स्वतः पता लगाता है: नेटिव F5 XC लॉगिन, कैश्ड सेशन के साथ Azure SSO, या DUO वेरिफ़ाइड पुश MFA के साथ Azure SSO।

### console-navigator

F5 XC कंसोल अनुभागों पर निर्धारक नेविगेशन। अनुभाग नामों को URL पथों में अनुवाद करता है, सेशन स्वास्थ्य सत्यापित करता है, और पेज लोड की पुष्टि करता है।

### platform-index / api-index / console-index

इंटेंट राउटर जो उपयोगकर्ता अनुरोधों को वर्गीकृत करते हैं और उचित कौशल और एजेंट को सौंपते हैं। ये स्वतः सक्रिय होते हैं और उपयोगकर्ता द्वारा आह्वान योग्य नहीं हैं।

## कमांड

### /check-api-token

```
/f5xc-platform:check-api-token
```

api-operator एजेंट को सौंपकर आपके F5 XC API टोकन को सत्यापित करता है।

### /login-console

```
/f5xc-platform:login-console
```

console-operator एजेंट के माध्यम से F5 Distributed Cloud कंसोल पर प्रमाणीकृत करता है।

## एजेंट

### api-operator

संसाधन CRUD, टोकन सत्यापन, और कॉन्फ़िगरेशन संचालन के लिए cURL और jq अनुक्रम निष्पादित करने वाला स्वायत्त REST API एजेंट। मुख्य सेशन संदर्भ से विस्तृत API JSON पेलोड को दूर रखने के लिए सबएजेंट में चलता है।

### console-operator

प्रमाणीकरण, नेविगेशन, और फ़ॉर्म इंटरैक्शन के लिए Chrome DevTools MCP टूल अनुक्रम निष्पादित करने वाला स्वायत्त ब्राउज़र स्वचालन एजेंट। मुख्य सेशन से टोकन-हेवी ब्राउज़र स्नैपशॉट को दूर रखने के लिए सबएजेंट में चलता है।

### config-analyzer

रीड-ओनली कॉन्फ़िगरेशन विश्लेषण एजेंट। सुरक्षा स्थिति, फ़ीचर सक्रियण, मोड संक्रमण, और सर्वोत्तम प्रथाओं के बारे में प्रश्नों का उत्तर देने के लिए संसाधन प्रोफ़ाइल संदर्भ फ़ाइलों के विरुद्ध ग्राहक JSON कॉन्फ़िगरेशन का विश्लेषण करता है।

<Aside type="caution">
  config-analyzer एजेंट **अल्पकालिक** है &mdash; प्रत्येक प्रश्न के लिए नए सिरे से उत्पन्न किया जाता है और डिस्पैच के बीच कोई स्मृति नहीं होती। अनुवर्ती प्रश्नों में डिस्पैच प्रॉम्प्ट में संकुचित पिछले निष्कर्ष शामिल होने चाहिए।
</Aside>
