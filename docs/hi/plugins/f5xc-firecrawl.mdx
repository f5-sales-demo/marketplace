**f5xc-firecrawl** प्लगइन ओपन-सोर्स [firecrawl](https://github.com/mendableai/firecrawl)
इंजन के माध्यम से स्थानीय सेल्फ-होस्टेड वेब स्क्रैपिंग प्रदान करता है। कोई API कुंजियाँ नहीं, कोई सब्सक्रिप्शन नहीं, कोई क्लाउड निर्भरता नहीं। सभी
ऑपरेशन devcontainer के अंदर `localhost:3002` पर स्थानीय firecrawl इंस्टेंस के विरुद्ध चलते हैं।

<Badge text="v1.1.0" variant="note" /> <Badge text="Productivity" variant="tip" />

## इंस्टॉलेशन

```
/plugin install f5xc-firecrawl@f5xc-salesdemos-marketplace
```

## कमांड्स

### /scrape

एक एकल URL को स्क्रैप करें और कंटेंट को मार्कडाउन के रूप में एक्सट्रैक्ट करें।

```
/scrape https://docs.example.com/getting-started
/scrape https://example.com --format markdown,links --wait 2000
```

### /batch-scrape

एक साथ कई URL स्क्रैप करें।

```
/batch-scrape https://example.com https://example.org https://example.net
```

### /crawl

एक प्रारंभिक URL से कई पेजों को क्रॉल करें।

```
/crawl https://docs.example.com --limit 20 --depth 2
/crawl https://docs.example.com --include /api/* --exclude /blog/*
```

### /map

किसी वेबसाइट पर सभी URL खोजें।

```
/map https://docs.example.com
/map https://docs.example.com --search api --subdomains
```

### /search

वेब पर खोजें और वैकल्पिक रूप से परिणामों को स्क्रैप करें।

```
/search "firecrawl web scraping" --limit 10
/search "AI tools 2026" --scrape --time month
```

### /extract

वेब पेजों से LLM-संचालित संरचित डेटा एक्सट्रैक्शन।

```
/extract https://example.com "Extract the main heading and any links"
/extract https://example.com/pricing --schema '{"plans": [{"name": "string", "price": "string"}]}'
```

<Aside type="note">
  extract कमांड के लिए `OPENAI_BASE_URL` और `OPENAI_API_KEY` के माध्यम से कॉन्फ़िगर किया गया एक OpenAI-संगत LLM प्रॉक्सी आवश्यक है।
</Aside>

### /llmstxt

किसी साइट के लिए llms.txt फ़ाइल जनरेट करें।

```
/llmstxt https://docs.example.com
```

## स्किल्स

### web-scraper

जब आप किसी URL को स्क्रैप करने, वेबसाइट क्रॉल करने, साइट
URL मैप करने, वेब सर्च करने, संरचित डेटा एक्सट्रैक्ट करने, llms.txt जनरेट करने,
कई URL बैच स्क्रैप करने, या वेब पेज को मार्कडाउन में कन्वर्ट करने के लिए कहते हैं तो ऑटो-एक्टिवेट होता है।
तुरंत firecrawl-operator एजेंट को डेलीगेट करता है।

## एजेंट्स

### firecrawl-operator

स्वायत्त वेब स्क्रैपिंग एजेंट जो स्थानीय firecrawl API के
विरुद्ध `curl` + `jq` अनुक्रम निष्पादित करता है। सभी
v1 एंडपॉइंट्स को कवर करने वाले 11 प्रोटोकॉल का समर्थन करता है। रीड-ओनली एजेंट (कोई Write, Edit, या Agent टूल्स नहीं)।

| प्रोटोकॉल | एंडपॉइंट | प्रकार |
|----------|----------|------|
| HEALTH | `GET /` | Sync |
| SCRAPE | `POST /v1/scrape` | Sync |
| BATCH_SCRAPE | `POST /v1/batch/scrape` | Async |
| CRAWL | `POST /v1/crawl` | Async |
| CRAWL_CANCEL | `DELETE /v1/crawl/:id` | Sync |
| CRAWL_ACTIVE | `GET /v1/crawl/active` | Sync |
| CRAWL_ERRORS | `GET /v1/crawl/:id/errors` | Sync |
| MAP | `POST /v1/map` | Sync |
| SEARCH | `POST /v1/search` | Sync |
| EXTRACT | `POST /v1/extract` | Async |
| LLMSTXT | `POST /v1/llmstxt` | Async |

## इन्फ्रास्ट्रक्चर

इस प्लगइन के लिए devcontainer में firecrawl स्टैक का चलना आवश्यक है:

| कंपोनेंट | पोर्ट | उद्देश्य |
|-----------|------|---------|
| Firecrawl API | 3002 | सभी scrape/crawl/map/search/extract एंडपॉइंट्स |
| Playwright | 3000 | JavaScript रेंडरिंग इंजन |
| Redis | 6379 | जॉब क्यू बैकएंड |
| PostgreSQL | socket | Crawl/batch जॉब पर्सिस्टेंस |
| LiteLLM proxy | OPENAI_BASE_URL | extract के लिए LLM बैकएंड (वैकल्पिक) |

जब `ENABLE_FIRECRAWL=true` (डिफ़ॉल्ट) होता है तो स्टैक स्वचालित रूप से शुरू होता है। एक SessionStart हुक जाँचता है कि API पहुँच योग्य है और
सेवा बंद होने पर चेतावनी देता है।

## क्लाउड Firecrawl से अंतर

यह प्लगइन सेल्फ-होस्टेड ओपन-सोर्स संस्करण का उपयोग करता है:

- स्क्रैपिंग के लिए कोई प्रमाणीकरण या API कुंजियाँ आवश्यक नहीं
- कोई क्रेडिट सीमा या रेट लिमिटिंग नहीं
- v1 API एंडपॉइंट्स का उपयोग करता है (v2 नहीं)
- ब्राउज़र सेशन और डीप रिसर्च उपलब्ध नहीं हैं
- Extract होस्टेड मॉडल की बजाय आपके स्वयं के LLM प्रॉक्सी का उपयोग करता है
- पूरी तरह से स्थानीय कंटेनर नेटवर्क के भीतर चलता है
