**f5xc-sales-engineer** प्लगइन F5 XC डेमो रिपॉजिटरी के लिए एक पुनः उपयोगयोग्य Sales Engineer
पर्सोना फ्रेमवर्क प्रदान करता है। यह पूर्ण डेमो मीटिंग जीवनचक्र को संभालता है — प्री-मीटिंग सत्यापन से लेकर लाइव
निष्पादन, Q&A और teardown तक।

<Badge text="v1.0.6" variant="note" /> <Badge text="Productivity" variant="tip" />

## इंस्टॉलेशन

```
/plugin install f5xc-sales-engineer@f5-sales-demo-marketplace
```

## कौशल

### demo-executor

चार-चरण मीटिंग जीवनचक्र के साथ API-संचालित डेमो निष्पादन:

1. **Prepare** — `demo-housekeeping` एजेंट के माध्यम से प्री-मीटिंग वातावरण सत्यापन
2. **Execute** — ग्राहक-सामना नैरेशन के साथ निर्धारक चरण निष्पादन
3. **Q&A** — लाइव वातावरण के साथ इम्प्रोवाइज़ेशन मोड
4. **Teardown** — `demo-housekeeping` एजेंट के माध्यम से पोस्ट-मीटिंग क्लीनअप

`DEMO_PRODUCT_EXPERTISE.md` से उत्पाद-विशिष्ट सामग्री और `docs/demo/` से डेमो
कमांड पढ़ता है।

### presenter

पूर्व-कॉन्फ़िगर डेमो वातावरण का उपयोग करते हुए as-built walkthrough प्रस्तुति। प्रत्येक चरण में show → narrate → connect → pause पैटर्न का अनुसरण करता है। `DEMO_WALKTHROUGH_CONFIG.md` से walkthrough क्रम पढ़ता है।

### subject-matter-expert

सख्त उद्धरण आवश्यकताओं के साथ संदर्भ-समर्थित Q&A। कभी अनुमान नहीं लगाता — प्रत्येक उत्तर में एक संदर्भ या प्रमाण शामिल होता है। आवश्यकता पड़ने पर `demo-researcher` एजेंट को शोध सौंपता है।

### sales-engineer

रोल इंडेक्स जो उपयोगकर्ता के इरादे को सही पर्सोना कौशल पर रूट करता है। तब सक्रिय होता है जब अनुरोध डेमो-संबंधित हो लेकिन किसी विशिष्ट ट्रिगर वाक्यांश से मेल न खाए।

## एजेंट

### demo-housekeeping

प्री-मीटिंग सत्यापन और पोस्ट-मीटिंग teardown के लिए स्वायत्त एजेंट। `DEMO_READINESS_MATRIX.md` में परिभाषित Readiness Verification Matrix (T0–T5) चलाता है और `docs/demo/phase-4-teardown.mdx` से teardown कमांड निष्पादित करता है।

### demo-researcher

उद्धरण सहित सत्यापित उत्तर खोजने वाला रीड-ओनली रिसर्च लाइब्रेरियन। पहले स्थानीय `docs/` खोजता है, फिर `DEMO_SOURCE_INDEX.md` से अनुक्रमित बाहरी स्रोत, WebSearch को फ़ॉलबैक के रूप में उपयोग करता है।

| फ़ाइल | उद्देश्य | किसके लिए आवश्यक |
| ---- | ------- | ------------ |

| `DEMO_WALKTHROUGH_CONFIG.md` | डेमो ऐप URL, walkthrough क्रम, डिटेक्शन टाइमिंग | presenter |

| `DEMO_READINESS_MATRIX.md` | आवश्यक/वैकल्पिक वेरिएबल, readiness जाँच (T0–T5), API एंडपॉइंट | demo-housekeeping |
| `docs/demo/` | cURL कमांड और एविडेंस गेट्स के साथ चरण फ़ाइलें | demo-executor |
