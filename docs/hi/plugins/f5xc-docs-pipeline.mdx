**f5xc-docs-pipeline** प्लगइन f5-sales-demo रिपॉजिटरीज़ के लिए डॉक्युमेंटेशन पाइपलाइन
आर्किटेक्चर संबंधी ज्ञान प्रदान करता है। यह कंटेंट ऑथरिंग में मार्गदर्शन करता है,
मल्टी-रेपो बिल्ड में कॉन्फिग स्वामित्व को स्पष्ट करता है, और लोकल Docker-आधारित
प्रीव्यू उपलब्ध कराता है।

<Badge text="v1.0.2" variant="note" /> <Badge text="Productivity" variant="tip" />

## इंस्टॉलेशन

```
/plugin install f5xc-docs-pipeline@f5-sales-demo-marketplace
```

## स्किल्स

### content-author

फ़ाइल संरचना, MDX सिंटैक्स नियम, फ्रंटमैटर आवश्यकताएँ, इमेज संदर्भ, और लोकल Docker
प्रीव्यू को कवर करने वाला कंटेंट ऑथरिंग गाइड। `docs/` कंटेंट बनाते या संपादित करते
समय, MDX फ़ाइलों के साथ कार्य करते समय, या लोकल प्रीव्यू सेटअप करते समय सक्रिय होता है।

### pipeline-navigator

चार अपस्ट्रीम रिपॉजिटरीज़ में परिवर्तन कहाँ करने हैं यह समझाने वाला डॉक्युमेंटेशन
पाइपलाइन आर्किटेक्चर: docs-theme, docs-builder, docs-control, और docs-icons।
कॉन्फिग स्वामित्व (`astro.config.mjs`, `content.config.ts`), रिलीज़ डिस्पैच चेन,
और मैनेज्ड फ़ाइल ट्रैकिंग को स्पष्ट करता है।

<Aside type="tip">
  pipeline-navigator का उपयोग तब करें जब आपको यह समझना हो कि कोई विशिष्ट
  कॉन्फिगरेशन फ़ाइल या घटक किस रिपॉजिटरी के स्वामित्व में है।
</Aside>

## कमांड्स

### /preview-docs

```
/f5xc-docs-pipeline:preview-docs
```

डॉक्युमेंटेशन का प्रीव्यू करने के लिए एक लोकल Docker डेव सर्वर शुरू करता है।
लाइव प्रीव्यू के लिए आपकी लोकल `docs/` डायरेक्टरी माउंट करके शेयर्ड Docker
बिल्ड इमेज चलाता है।
