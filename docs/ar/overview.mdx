**سوق مكوّنات Claude Code الإضافية** هو مجموعة منسّقة من المكوّنات التي تمدّ Claude Code بمهارات وأوامر ووكلاء جدد. تتيح الأسواق للفرق مشاركة الأدوات بحيث يحصل كل مطوّر على نفس الإمكانيات دون الحاجة إلى إعداد يدوي.

يوفّر **f5-sales-demo Marketplace** مكوّنات مبنيّة لمنصة توثيق [f5-sales-demo](https://github.com/f5-sales-demo) — منظومة تضمّ أكثر من 20 مستودعاً تنشر مواقع Astro/Starlight على GitHub Pages باستخدام خط أنابيب بناء مشترك.

## كيفية العمل

1. **أضف السوق** إلى Claude Code — يُسجّل هذا الإجراء السوق بوصفه مصدراً للمكوّنات الإضافية
2. **ثبّت المكوّنات** من السوق — يُضيف كل مكوّن مهارات وأوامر إلى جلسة Claude Code الخاصة بك
3. **استخدم الأدوات** — تُفعَّل المكوّنات تلقائياً بناءً على السياق، أو يمكنك استدعاء أوامرها مباشرةً

## المكوّنات المتاحة

| المكوّن | الفئة | الوصف |
|--------|----------|-------------|
| [f5xc-docs-tools](/marketplace/plugins/f5xc-docs-tools/) | الإنتاجية | أدوات التحقق من صحة محتوى MDX ومراجعته في مستودعات التوثيق |
| [f5xc-sales-engineer](/marketplace/plugins/f5xc-sales-engineer/) | الإنتاجية | إطار عمل شخصية مهندس المبيعات — تنفيذ العروض التوضيحية والتقديم والأسئلة والأجوبة وإدارة البيئة |

## البدء السريع

أضف السوق وثبّت مكوّن أدوات التوثيق:

```
/plugin marketplace add f5-sales-demo/marketplace
/plugin install f5xc-docs-tools@f5-sales-demo-marketplace
```

راجع [البدء](/marketplace/getting-started/) للاطلاع على شرح تفصيلي كامل.

## البنية المعمارية

السوق هو مستودع Git يحتوي على مكوّن إضافي واحد أو أكثر. يُحزم كل مكوّن المهارات والأوامر والوكلاء في وحدة واحدة قابلة للتثبيت.

```
Marketplace (marketplace.json)
└── Plugin (plugin.json)
    ├── Skills (SKILL.md)
    │   └── References (data files)
    ├── Commands (.md)
    └── Agents (.md)
```

- **السوق** — السجل. يُدرج المكوّنات المتاحة مع البيانات الوصفية اللازمة للاكتشاف والتثبيت.
- **المكوّن** — الوحدة القابلة للتثبيت. تحتوي على مهارة واحدة أو أكثر، أو أوامر، أو وكلاء.
- **المهارة** — معرفة سياقية يُفعّلها Claude تلقائياً بناءً على المهمة المطروحة.
- **الأمر** — أمر يبدأ بشرطة مائلة يستدعيه المستخدم مباشرةً (مثل `/review-mdx`).
- **الوكيل** — وكيل فرعي متخصص يتولى التعامل مع المهام المعقدة متعددة الخطوات باستقلالية.
