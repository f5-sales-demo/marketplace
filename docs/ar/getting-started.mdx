يرشدك هذا الدليل خلال عملية إضافة سوق f5-sales-demo
إلى Claude Code وتثبيت أول إضافة لك.

## المتطلبات الأساسية

- Claude Code الإصدار **v1.0.33** أو أحدث (دعم سوق الإضافات)
- حساب GitHub مع صلاحية الوصول إلى منظمة
  [f5-sales-demo](https://github.com/f5-sales-demo)

## التثبيت

<Steps>

1. **إضافة السوق**

   في أي جلسة Claude Code، نفّذ الأمر التالي:

   ```
   /plugin marketplace add f5-sales-demo/marketplace
   ```

   يُسجّل هذا الأمر السوق كمصدر للإضافات. تحتاج إلى
   تنفيذه مرة واحدة فقط.

2. **استعراض الإضافات المتاحة**

   اعرض ما يوفره السوق:

   ```
   /plugin marketplace list f5-sales-demo-marketplace
   ```

3. **تثبيت إضافة**

   ثبّت الإضافة التي تحتاجها:

   ```
   /plugin install f5xc-docs-tools@f5-sales-demo-marketplace
   /plugin install f5xc-sales-engineer@f5-sales-demo-marketplace
   ```

4. **استخدام الإضافة**

   انتقل إلى أي مستودع محتوى خاص بـ f5-sales-demo.

   لـ **أدوات التوثيق**، نفّذ أمر مراجعة MDX:

   ```
   /f5xc-docs-tools:review-mdx
   ```

   لـ **مهندس المبيعات**، استخدم عبارات التشغيل مثل
   "prepare the demo"، أو "run the demo"، أو "walk through
   the demo" — يتفعّل الإضافة تلقائيًا.

</Steps>

## تحديث الإضافات

لتحديث إضافة مثبّتة إلى أحدث إصدار:

```
/plugin update f5xc-docs-tools@f5-sales-demo-marketplace
/plugin update f5xc-sales-engineer@f5-sales-demo-marketplace
```

## إلغاء التثبيت

إزالة إضافة:

```
/plugin uninstall f5xc-docs-tools@f5-sales-demo-marketplace
/plugin uninstall f5xc-sales-engineer@f5-sales-demo-marketplace
```

إزالة السوق بالكامل:

```
/plugin marketplace remove f5-sales-demo-marketplace
```

## إعداد الفريق

لضبط إعدادات السوق مسبقًا لجميع أعضاء فريقك، أضفه
إلى ملف `.claude/settings.json` في مشروعك:

```json
{
  "extraKnownMarketplaces": [
    "f5-sales-demo/marketplace"
  ],
  "enabledPlugins": [
    "f5xc-docs-tools@f5-sales-demo-marketplace",
    "f5xc-sales-engineer@f5-sales-demo-marketplace"
  ]
}
```

<Aside type="tip" title="إعدادات مشتركة">
  إيداع ملف `.claude/settings.json` في مستودعك يعني أن
  كل عضو في الفريق سيحصل على السوق والإضافات تلقائيًا —
  دون الحاجة إلى أي إعداد يدوي.
</Aside>

يضمن ذلك أن كل مساهم لديه نفس الأدوات المتاحة
منذ أول جلسة Claude Code في المستودع.
