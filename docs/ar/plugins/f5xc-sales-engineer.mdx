يوفر المكوّن الإضافي **f5xc-sales-engineer** إطار عمل قابلاً لإعادة الاستخدام
لشخصية مهندس المبيعات في مستودعات العروض التوضيحية لـ F5 XC. يتولى إدارة
دورة حياة اجتماع العرض التوضيحي بالكامل — بدءًا من التحقق قبل الاجتماع
وصولاً إلى التنفيذ المباشر والأسئلة والأجوبة والإزالة.

<Badge text="v1.0.6" variant="note" /> <Badge text="Productivity" variant="tip" />

## التثبيت

```
/plugin install f5xc-sales-engineer@f5-sales-demo-marketplace
```

## المهارات

### demo-executor

تنفيذ العرض التوضيحي المدفوع بـ API مع دورة حياة اجتماع من أربع مراحل:

1. **الإعداد** — التحقق من البيئة قبل الاجتماع عبر
   وكيل `demo-housekeeping`
2. **التنفيذ** — تنفيذ المراحل بصورة حتمية مع سرد موجّه للعميل
3. **الأسئلة والأجوبة** — وضع الارتجال مع البيئة المباشرة
4. **الإزالة** — تنظيف ما بعد الاجتماع عبر وكيل `demo-housekeeping`

يقرأ المحتوى الخاص بالمنتج من `DEMO_PRODUCT_EXPERTISE.md` وأوامر
العرض التوضيحي من `docs/demo/`.

### presenter

عرض تقديمي للمشروع كما هو مبني باستخدام بيئة عرض توضيحي مهيأة مسبقًا.
يتبع نمط: عرض ← سرد ← ربط ← توقف في كل خطوة. يقرأ ترتيب العرض
من `DEMO_WALKTHROUGH_CONFIG.md`.

### subject-matter-expert

الأسئلة والأجوبة المستندة إلى المراجع مع متطلبات استشهاد صارمة. لا يخمّن
أبدًا — كل إجابة تتضمن مرجعًا أو دليلاً. يفوّض البحث إلى وكيل
`demo-researcher` عند الحاجة.

### sales-engineer

فهرس الأدوار الذي يوجّه نية المستخدم إلى مهارة الشخصية الصحيحة.
يُفعَّل عندما يكون الطلب متعلقًا بالعرض التوضيحي لكنه لا يطابق
عبارة تشغيل محددة.

## الوكلاء

### demo-housekeeping

وكيل مستقل للتحقق قبل الاجتماع وإزالة ما بعد الاجتماع. يُشغّل مصفوفة
التحقق من الجاهزية (T0–T5) المحددة في `DEMO_READINESS_MATRIX.md`
وينفّذ أوامر الإزالة من `docs/demo/phase-4-teardown.mdx`.

### demo-researcher

مكتبي بحثي للقراءة فقط يجد إجابات موثّقة مع استشهادات. يبحث في
`docs/` المحلية أولاً، ثم في المصادر الخارجية المفهرسة من
`DEMO_SOURCE_INDEX.md`، مع WebSearch كخيار احتياطي.

## ملفات الاتفاقيات

<Aside type="tip" title="المحتوى الخاص بالمنتج">

</Aside>

أنشئ هذه الملفات في جذر مستودعك لتهيئة المكوّن الإضافي:

| الملف | الغرض | مطلوب لـ |
| ---- | ------- | ------------ |
| `DEMO_PRODUCT_EXPERTISE.md` | قدرات المنتج وإشارات الكشف وتغطية التهديدات ومحاذاة الامتثال ومرجع API | جميع المهارات |
| `DEMO_WALKTHROUGH_CONFIG.md` | عنوان URL لتطبيق العرض التوضيحي وترتيب العرض وتوقيت الكشف | presenter |

| `DEMO_READINESS_MATRIX.md` | المتغيرات المطلوبة والاختيارية وفحوصات الجاهزية (T0–T5) ونقاط نهاية API | demo-housekeeping |
| `docs/demo/` | ملفات المراحل مع أوامر cURL وبوابات الأدلة | demo-executor |
