يوفر مكوّن **f5xc-meddpicc** إطار عمل شاملاً لتأهيل المبيعات وتنفيذ الصفقات وفق منهجية MEDDPICC في مبيعات B2B المؤسسية المعقدة. يُحوّل هذا المكوّن منهجية MEDDPICC من مجرد قائمة تحقق إلى نظام حي لتنفيذ الصفقات، يعتمد على التأهيل المبني على الأدلة، ومراجعات الصفقات المنظّمة، والتوجيه القابل للتنفيذ.

```
/plugin install f5xc-meddpicc@f5-sales-demo-marketplace
```

## ما هي منهجية MEDDPICC؟

MEDDPICC هو إطار عمل لتأهيل المبيعات وتنفيذ الصفقات، مصمم لمبيعات B2B المعقدة التي يمكن أن تُعرقل زخمها تعدد أصحاب المصلحة، والمخاطر التقنية، وخطوات المشتريات.

| الحرف | العنصر | الغرض |
| ------ | ------- | ------- |
| **M** | المقاييس (Metrics) | النتائج التجارية الكمية التي يتوقعها العميل |
| **E** | المشتري الاقتصادي (Economic Buyer) | الشخص الذي يملك صلاحية الموافقة النهائية على الميزانية |
| **D** | معايير القرار (Decision Criteria) | المتطلبات المستخدمة لتقييم الخيارات |
| **D** | عملية القرار (Decision Process) | المسار التفصيلي من التقييم إلى الاتفاق |
| **P** | عملية الأوراق (Paper Process) | آليات المشتريات (القانونية، والأمان، والتوقيع) |
| **I** | تحديد الألم (Identify Pain) | ألم تجاري مقنع يخلق إلحاحية |
| **C** | المؤيد (Champion) | مناصر داخلي قوي |
| **C** | المنافسة (Competition) | جميع البدائل (الموردون، البناء الداخلي، عدم التحرك) |

## المهارات

### meddpicc-coach

توجيه شامل لمنهجية MEDDPICC يُفعَّل تلقائياً عند مناقشة تأهيل الصفقات، وصحة خط الأنابيب، ودقة التوقعات، أو أي عنصر من عناصر MEDDPICC. يوفر إرشادات مبنية على الأدلة، وتوصيات لإيقاع العمل، وروابط لمواد مرجعية تفصيلية.

<Aside type="tip" title="يُفعَّل تلقائياً">
  تُفعَّل هذه المهارة تلقائياً عند الاقتضاء — دون الحاجة إلى أمر slash.
</Aside>

### deal-qualification

تُنتج بطاقة تقييم MEDDPICC مُسجَّلة (0&ndash;24) مع تحليل الفجوات، وتقييم المخاطر، وتوصية التوقعات. يُسجَّل كل عنصر من 0 إلى 3 بناءً على الأدلة الموثّقة.

### deal-review

يُيسّر عمليات تفتيش أسبوعية منظّمة للصفقات مع تتبع الفجوات في الأدلة، وتحديد المخاطر، وبنود العمل المُرفقة بأسماء المسؤولين وتواريخ الاستحقاق.

### mutual-action-plan

يبني خطة عمل مشتركة موجّهة للعميل مرتبطة بعناصر MEDDPICC. تشمل مراحل التقييم، وخارطة طريق المشتريات، وجاهزية التنفيذ، ورسم خرائط أصحاب المصلحة.

### champion-test

يُطبّق تقييماً وفق 5 معايير (القوة، الفوز الشخصي، الوصول، المعلومات الاستخباراتية، الإجراء) بتسجيل من 0 إلى 15 لتحديد ما إذا كان جهة اتصالك الداخلية مؤيداً حقيقياً، أو مؤيداً ناشئاً، أو مرشداً، أو جهة اتصال ودية.

## الوكلاء

### deal-analyst

وكيل بحث للقراءة فقط يحلل صحة الصفقة من خلال مراجعة الوثائق المحلية، وملاحظات الاجتماعات، وخطط الحسابات. يُنتج تقارير تقييم MEDDPICC منظّمة مع استشهادات بالأدلة، وتحليل الفجوات، وتوصيات إجراءات مرتّبة حسب الأولوية.

## المكتبة المرجعية

يتضمن المكوّن مواد مرجعية شاملة تُحمَّل عند الطلب:

- **إطار عمل MEDDPICC** &mdash; تعريفات كاملة ومتطلبات الأدلة لجميع العناصر الثمانية
- **أسئلة الاستكشاف** &mdash; أسئلة مجرَّبة ميدانياً لكل عنصر عبر دورة الصفقة
- **الأدوار والمسؤوليات** &mdash; مصفوفة ملكية RACI حسب دور الفريق (AE، SE، CSM، PS، المنتج، القناة، مكتب الصفقات، القانوني)
- **بوابات مراحل الصفقة** &mdash; ما يجب أن يكون صحيحاً قبل التقدم (من التأهيل المبكر حتى الإغلاق الناجح)
- **الأنماط المضادة** &mdash; 10 أنماط فشل شائعة مع معالجة مبنية على الأدلة
- **مقياس التسجيل** &mdash; معايير التسجيل الموضوعية من 0 إلى 3 لكل عنصر
- **قوالب المراجعة** &mdash; مجموعات أسئلة التفتيش الأسبوعية مع مؤشرات الإنذار الأحمر
- **قوالب MAP** &mdash; هيكل خطة العمل الموجّهة للعميل مع خارطة طريق التقييم والمشتريات والتنفيذ
