يوفر مكوّن **f5xc-docs-pipeline** معرفة بنية خط أنابيب التوثيق لمستودعات f5-sales-demo. يرشد عملية تأليف المحتوى، ويوضح ملكية الإعدادات عبر عمليات البناء متعددة المستودعات، ويوفر معاينة محلية مستندة إلى Docker.

<Badge text="v1.0.2" variant="note" /> <Badge text="Productivity" variant="tip" />

## التثبيت

```
/plugin install f5xc-docs-pipeline@f5-sales-demo-marketplace
```

## المهارات

### content-author

دليل تأليف المحتوى الذي يغطي هيكل الملفات وقواعد صياغة MDX ومتطلبات frontmatter ومراجع الصور والمعاينة المحلية عبر Docker. يُفعَّل عند إنشاء محتوى `docs/` أو تحريره، أو العمل مع ملفات MDX، أو إعداد المعاينة المحلية.

### pipeline-navigator

بنية خط أنابيب التوثيق التي توضح أين يمكن إجراء التغييرات عبر المستودعات الأربعة الأولية: docs-theme وdocs-builder وdocs-control وdocs-icons. يوضح ملكية الإعدادات (`astro.config.mjs`، `content.config.ts`)، وسلسلة إرسال الإصدارات، وتتبع الملفات المُدارة.

<Aside type="tip">
  استخدم pipeline-navigator عندما تحتاج إلى فهم أي مستودع يمتلك ملف إعداد أو مكوّناً معيناً.
</Aside>

## الأوامر

### /preview-docs

```
/f5xc-docs-pipeline:preview-docs
```

يشغّل خادم تطوير Docker محلياً لمعاينة التوثيق. يُشغّل صورة بناء Docker المشتركة مع تحميل دليل `docs/` المحلي الخاص بك للمعاينة الفورية.
