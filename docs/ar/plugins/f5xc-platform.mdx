يوفر مكوّن **f5xc-platform** أتمتةً كاملةً لمنصة F5
Distributed Cloud. يشمل عمليات وحدة التحكم على الويب عبر
Chrome DevTools MCP، وعمليات REST API المدركة للمواصفات عبر cURL،
والمصادقة متعددة الموفرين، وتحليل الإعدادات عبر
98 نوعًا من الموارد في 38 نطاقًا.

<Badge text="v3.1.1" variant="note" /> <Badge text="Productivity" variant="tip" />

## التثبيت

```
/plugin install f5xc-platform@f5-sales-demo-marketplace
```

## المهارات

### api-operations

عمليات CRUD لـ API المدركة للمواصفات، تغطي 98 نوعًا من الموارد عبر
38 نطاقًا. توفر نقاط نهاية دقيقة، وقوالب حمولات،
وترتيبًا للتبعيات، وتركيبات سير عمل متعددة الخطوات لموازني تحميل HTTP،
ومجمعات المصدر، وسياسات WAF، ومناطق DNS،
والشهادات، ومواقع السحابة، وغير ذلك.

### config-analysis

قدرة تحليل الإعدادات والاستشارة. تحلل إعدادات JSON للعملاء
للإجابة عن أسئلة تتعلق بوضعية الأمان،
وتفعيل الميزات، وإعدادات الوضع، والاستثناءات، وأفضل الممارسات.
تدعم حوارات متعددة الأدوار مع سياق مضغوط بين عمليات الإرسال.

### api-auth

مصادقة API تدعم مصادقة رأس رمز API
ومصادقة شهادة P12 مع إجراءات التحقق من الرمز.

### console-auth

مصادقة متعددة الموفرين لوحدة تحكم F5 XC على الويب.
تكتشف تلقائيًا نوع تسجيل الدخول: تسجيل الدخول الأصلي لـ F5 XC، أو Azure SSO مع
جلسة مخزنة مؤقتًا، أو Azure SSO مع MFA بالدفع المُتحقق منه عبر DUO.

### console-navigator

التنقل الحتمي إلى أقسام وحدة تحكم F5 XC. يترجم
أسماء الأقسام إلى مسارات URL، ويتحقق من سلامة الجلسة، ويؤكد
تحميل الصفحة.

### platform-index / api-index / console-index

موجّهات الغرض التي تصنّف طلبات المستخدمين وتفوّضها إلى
المهارة والعامل المناسبَين. تُفعَّل تلقائيًا ولا يمكن استدعاؤها من قِبَل المستخدم.

## الأوامر

### /check-api-token

```
/f5xc-platform:check-api-token
```

يتحقق من صحة رمز F5 XC API الخاص بك عن طريق التفويض إلى عامل
api-operator.

### /login-console

```
/f5xc-platform:login-console
```

يُجري المصادقة إلى وحدة تحكم F5 Distributed Cloud عبر
عامل console-operator.

## العوامل

### api-operator

عامل REST API مستقل ينفّذ تسلسلات cURL وjq لعمليات CRUD
على الموارد، والتحقق من الرموز، وعمليات الإعداد.
يعمل في عامل فرعي للإبقاء على حمولات JSON التفصيلية لـ API خارج
سياق الجلسة الرئيسية.

### console-operator

عامل أتمتة متصفح مستقل ينفّذ تسلسلات أدوات Chrome DevTools MCP
للمصادقة، والتنقل، وتفاعلات النماذج.
يعمل في عامل فرعي للإبقاء على لقطات المتصفح الثقيلة بالرموز خارج
الجلسة الرئيسية.

### config-analyzer

عامل تحليل إعدادات للقراءة فقط. يحلل إعدادات JSON للعملاء
مقابل ملفات مرجعية لملفات تعريف الموارد للإجابة عن
أسئلة تتعلق بوضعية الأمان، وتفعيل الميزات، وانتقالات الوضع، وأفضل الممارسات.

<Aside type="caution">
  عامل config-analyzer **مؤقت** &mdash; يُنشأ من جديد
  لكل سؤال دون ذاكرة بين عمليات الإرسال. يجب أن تتضمن أسئلة المتابعة
  النتائج السابقة المضغوطة في موجّه الإرسال.
</Aside>
