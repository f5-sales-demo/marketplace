يوفر ملحق **osint-framework** كتالوج أدوات OSINT ومهارات
التحقيق — 1,064 أداة مجانية لجمع المعلومات الاستخباراتية
عبر 34 فئة مُعيَّنة من [osintframework.com](https://osintframework.com).
يتضمن مهارات قائمة على الفئات، وخطوط أنابيب تحقيق قابلة للتنفيذ،
وتنفيذ أدوات سطر الأوامر، وسير عمل واعٍ بالأمن التشغيلي (OPSEC)،
ورسم بياني دائم للكيانات لربط التحقيقات المتقاطعة.

<Badge text="v1.0.1" variant="note" /> <Badge text="Security" variant="tip" />

## التثبيت

```
/plugin install osint-framework@f5xc-salesdemos-marketplace
```

## الأوامر

### /osint-investigate

تشغيل تحقيق OSINT كامل ضد أي هدف. يكتشف تلقائياً
نوع الهدف (بريد إلكتروني، نطاق، عنوان IP، اسم مستخدم، شركة،
اسم شخص) ويوجّه إلى مهارات الفئة الصحيحة.

```
/osint-investigate j.smith@example.com
/osint-investigate example.com
/osint-investigate 198.51.100.1
/osint-investigate example-user
```

### /osint-search

البحث في كتالوج الأدوات عن الأدوات المطابقة لاستعلام القدرة.

```
/osint-search subdomain enumeration
/osint-search email breach check
/osint-search satellite imagery
```

### /osint-catalog

تصفح كتالوج الأدوات الكامل حسب الفئة. بدون معاملات يعرض
جميع الفئات الـ 34 مع عدد الأدوات.

```
/osint-catalog
/osint-catalog domain
/osint-catalog threat-intel
```

## المهارات

### osint-index

موجّه النوايا الرئيسي. يتم تفعيله تلقائياً عندما تقول "ابحث عن
معلومات حول"، "تحقق"، "ابحث عن"، "فحص خلفية"،
أو تصف أي مهمة استخبارات مفتوحة المصدر. يوجّه إلى
مهارة الفئة الصحيحة بناءً على نوع الهدف.

| الهدف | يوجَّه إلى |
|--------|----------|
| اسم مستخدم أو معرّف | `username-recon` |
| عنوان بريد إلكتروني | `email-recon` |
| اسم نطاق | `domain-recon` |
| عنوان IP | `ip-address-recon` |
| اسم شخص + شركة | `people-search` + `business-records` |
| شركة / مؤسسة | `business-records` + `domain-recon` |
| تجزئة برمجيات خبيثة | `malicious-file-analysis` + `threat-intelligence` |

### osint-catalog

متصفح كتالوج الأدوات. يوفر الوصول إلى سجل الأدوات الكامل،
وفهرس الفئات، وخطوط أنابيب التحقيق، ومرجع حدود المعدل،
ووثائق محرك الربط.

## الوكلاء

### osint-investigator

منسّق تحقيق مستقل متعدد الأدوات. يشغّل سير عمل
التحقيق الكامل — فحص توفر الأدوات، اكتشاف نوع الهدف،
التنفيذ التسلسلي للأدوات (السلبي أولاً)، تجميع النتائج،
وتوليد تقارير منظمة مع مستويات الثقة وتصنيفات OPSEC.

يتكامل مع رسم الكيانات البياني: كل نتيجة أداة تُنشئ كيانات
وعلاقات في `/tmp/osint-graph/` لربط التحقيقات المتقاطعة.

<Aside type="caution">
  أدوات الاستطلاع النشط (nmap, nuclei, sherlock) تتصل مباشرة
  بالبنية التحتية للهدف. يُنبّه خطّاف تحذير OPSEC قبل
  تشغيل الأدوات النشطة. تأكد دائماً من حصولك على التصريح.
</Aside>

### osint-researcher

وكيل بحث في الكتالوج للقراءة فقط. يبحث في الملفات المرجعية
للتوصية بأفضل الأدوات لمهمة معينة. لا يُنفّذ الأدوات أبداً —
يوصي ويشرح فقط. يُستخدم داخلياً من قبل مهارات الفئات
لاكتشاف الأدوات.

### osint-executor

وكيل تنفيذ أداة CLI واحدة. يشغّل أداة واحدة في كل مرة مع
معاملات محددة، ويحلل المخرجات، ويُعيد نتائج منظمة.
يتضمن وصفات تنفيذ لكل أداة لأكثر من 20 أداة مع العلامات الدقيقة،
وتنسيقات المخرجات، وأنماط تحليل jq.

## تحديد المعدل

جميع استدعاءات API تستخدم المساعد `osint_curl` مع تراجع أسي
مدمج عند HTTP 429/503. الحدود الرئيسية:

| واجهة API | الحد المجاني | متغير البيئة |
|-----|-----------|---------|
| ipinfo.io | 1,000/يوم | — |
| crt.sh | 60/دقيقة | — |
| NVD | 5/30 ثانية (بدون مفتاح) | `NVD_API_KEY` |
| OpenCorporates | يتطلب مفتاحاً | `OPENCORPORATES_API_KEY` |
| Shodan | 1/ثانية | `SHODAN_API_KEY` |
| GitHub API | 60/ساعة (بدون مصادقة) | `GH_TOKEN` |

يتم تخطي واجهات API التي تتطلب مفاتيح تلقائياً مع رسالة واضحة
إذا لم يتم تعيين متغير البيئة.

## محرك الربط

يتضمن الملحق رسماً بيانياً دائماً للكيانات بتنسيق JSON في
`/tmp/osint-graph/` يجمع النتائج عبر التحقيقات.

```bash
source plugins/osint-framework/scripts/osint-graph.sh
osint_graph_init

# Add entities
P=$(osint_entity_add "person" "J. Smith" --tool github-api)
C=$(osint_entity_add "company" "F5" --tool github-api)

# Link them
osint_rel_add "$P" "$C" "works_at" --tool github-api

# Query: all entities within 2 hops of the person
osint_graph_query "$P" 2

# Generate report
osint_graph_report
```

**تقييم الثقة** يستخدم دمج متعدد المصادر:
`confidence = 1 - product(1 - ci)` لكل مصدر مؤكِّد.
مصدران بثقة 0.80 ← ثقة مجمعة 0.96.

## إشعار قانوني

جميع الأدوات تستخدم فقط المعلومات المتاحة للعموم. المستخدمون
مسؤولون عن ضمان الامتثال للقوانين واللوائح المعمول بها
وشروط خدمة المنصات. لا تستخدم أبداً أدوات OSINT
للوصول غير المصرح به، أو المضايقة، أو أي نشاط غير قانوني.
