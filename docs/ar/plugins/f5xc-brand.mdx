يفرض ملحق **f5xc-brand** هوية العلامة التجارية المؤسسية لشركة F5
عبر جميع أنواع المحتوى. يغطي لوحة ألوان F5 المكونة من 45 لونًا،
وقواعد الخطوط (عناوين Neusa Next Pro Wide، ومتن
Proxima Nova)، واستخدام الشعارات بصيغة SVG فقط، واختيار حزم الأيقونات،
وإمكانية الوصول وفق معيار WCAG AA، ومصطلحات المنتجات.

<Badge text="v1.0.2" variant="note" /> <Badge text="Productivity" variant="tip" />

## التثبيت

```
/plugin install f5xc-brand@f5xc-salesdemos-marketplace
```

## المهارات

### brand-guardian

تطبيق استباقي لهوية العلامة التجارية عند إنشاء أي محتوى.
يتم تفعيلها تلقائيًا عند إنشاء الوثائق أو المخططات أو
العروض التقديمية أو الصور المصغرة للفيديو أو رسومات وسائل التواصل الاجتماعي أو مخططات Mermaid
أو رسومات Excalidraw. تضمن الاستخدام الصحيح لألوان F5
وخطوطها وشعاراتها وأيقوناتها ونبرة الصوت وإمكانية الوصول.

### brand-reviewer

مراجعة المحتوى الحالي للتحقق من التزامه بمعايير العلامة التجارية لـ F5. تدقيق الألوان
والخطوط وصيغة الشعار واختيار الأيقونات والنص البديل ونسب التباين
ومصطلحات المنتجات. تنتج تقارير منظمة تتضمن
المخالفات والتحذيرات والاقتراحات.

### visual-content

قواعد خاصة بالتنسيقات للمحتوى المرئي ذي العلامة التجارية. توفر
إرشادات لمخططات Mermaid ورسومات Excalidraw ولقطات الشاشة
وعروض PowerPoint التقديمية والصور المصغرة لـ YouTube ونصوص الفيديو
ورسومات وسائل التواصل الاجتماعي مع اختيار حزم الأيقونات وتخطيطات الشرائح
والخطوط البديلة والأبعاد الخاصة بكل منصة.

## الأوامر

### /review-brand

```
/f5xc-brand:review-brand [path-or-glob]
```

مراجعة المحتوى للتحقق من التزامه بمعايير العلامة التجارية لـ F5. يقبل مسارًا اختياريًا
أو نمط glob لتحديد نطاق المراجعة. بدون وسائط، تتم مراجعة
جميع المحتوى في الدليل الحالي.
