このガイドでは、f5-sales-demo マーケットプレイスを Claude Code に追加し、最初のプラグインをインストールする手順を説明します。

## 前提条件

- Claude Code **v1.0.33** 以降（プラグインマーケットプレイスのサポート）
- [f5-sales-demo](https://github.com/f5-sales-demo) 組織へのアクセス権を持つ GitHub アカウント

## インストール

<Steps>

1. **マーケットプレイスを追加する**

   任意の Claude Code セッションで、次のコマンドを実行します:

   ```
   /plugin marketplace add f5-sales-demo/marketplace
   ```

   これにより、マーケットプレイスがプラグインソースとして登録されます。この操作は一度だけ行えば十分です。

2. **利用可能なプラグインを確認する**

   マーケットプレイスで提供されているプラグインの一覧を表示します:

   ```
   /plugin marketplace list f5-sales-demo-marketplace
   ```

3. **プラグインをインストールする**

   必要なプラグインをインストールします:

   ```
   /plugin install f5xc-docs-tools@f5-sales-demo-marketplace
   /plugin install f5xc-sales-engineer@f5-sales-demo-marketplace
   ```

4. **プラグインを使用する**

   任意の f5-sales-demo コンテンツリポジトリに移動します。

   **ドキュメントツール**の場合は、MDX レビューコマンドを実行します:

   ```
   /f5xc-docs-tools:review-mdx
   ```

   **セールスエンジニア**の場合は、「prepare the demo」、「run the demo」、「walk through the demo」などのトリガーフレーズを使用します — プラグインは自動的に起動します。

</Steps>

## プラグインの更新

インストール済みプラグインを最新バージョンに更新するには:

```
/plugin update f5xc-docs-tools@f5-sales-demo-marketplace
/plugin update f5xc-sales-engineer@f5-sales-demo-marketplace
```

## アンインストール

プラグインを削除するには:

```
/plugin uninstall f5xc-docs-tools@f5-sales-demo-marketplace
/plugin uninstall f5xc-sales-engineer@f5-sales-demo-marketplace
```

マーケットプレイスを完全に削除するには:

```
/plugin marketplace remove f5-sales-demo-marketplace
```

## チームの設定

チーム全体に対してマーケットプレイスをあらかじめ設定するには、プロジェクトの `.claude/settings.json` に追加します:

```json
{
  "extraKnownMarketplaces": [
    "f5-sales-demo/marketplace"
  ],
  "enabledPlugins": [
    "f5xc-docs-tools@f5-sales-demo-marketplace",
    "f5xc-sales-engineer@f5-sales-demo-marketplace"
  ]
}
```

<Aside type="tip" title="共有設定">
  `.claude/settings.json` をリポジトリにコミットすることで、すべてのチームメンバーがマーケットプレイスとプラグインを自動的に利用できるようになります — 手動のセットアップは不要です。
</Aside>

これにより、すべてのコントリビューターがリポジトリで最初の Claude Code セッションから同じツールを利用できるようになります。
