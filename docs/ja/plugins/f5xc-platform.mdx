**f5xc-platform** プラグインは、F5 Distributed Cloud プラットフォームの完全な自動化を提供します。Chrome DevTools MCP によるWebコンソール操作、cURL による仕様準拠のREST API操作、マルチプロバイダー認証、および38ドメインにわたる98のリソースタイプの設定分析をカバーします。

<Badge text="v3.1.1" variant="note" /> <Badge text="Productivity" variant="tip" />

## インストール

```
/plugin install f5xc-platform@f5-sales-demo-marketplace
```

## スキル

### api-operations

38ドメインにわたる98のリソースタイプをカバーする、仕様準拠のAPI CRUD操作。HTTPロードバランサー、オリジンプール、WAFポリシー、DNSゾーン、証明書、クラウドサイトなどに対して、正確なエンドポイント、ペイロードテンプレート、依存関係の順序付け、およびマルチステップワークフローの構成を提供します。

### config-analysis

設定分析およびアドバイザリー機能。お客様のJSON設定を分析し、セキュリティポスチャ、機能の有効化、モード設定、除外設定、およびベストプラクティスに関する質問に回答します。ディスパッチ間で圧縮されたコンテキストを使用したマルチターンQ&Aをサポートします。

### api-auth

APIトークンヘッダー認証およびP12証明書認証（トークン検証手順を含む）をサポートするAPI認証。

### console-auth

F5 XC Webコンソール向けのマルチプロバイダー認証。ネイティブF5 XCログイン、キャッシュセッションを使用したAzure SSO、またはDUO検証済みプッシュMFAを使用したAzure SSOのログインタイプを自動検出します。

### console-navigator

F5 XC コンソールセクションへの確定的なナビゲーション。セクション名をURLパスに変換し、セッションの正常性を検証し、ページのロードを確認します。

### platform-index / api-index / console-index

ユーザーリクエストを分類し、適切なスキルおよびエージェントに委任するインテントルーター。自動的にアクティベートされ、ユーザーが直接呼び出すことはできません。

## コマンド

### /check-api-token

```
/f5xc-platform:check-api-token
```

api-operatorエージェントに委任することで、F5 XC APIトークンを検証します。

### /login-console

```
/f5xc-platform:login-console
```

console-operatorエージェントを介してF5 Distributed Cloudコンソールに認証します。

## エージェント

### api-operator

リソースのCRUD、トークン検証、および設定操作のためにcURLおよびjqシーケンスを実行する自律型REST APIエージェント。冗長なAPI JSONペイロードをメインセッションコンテキストから除外するため、サブエージェントとして実行されます。

### console-operator

認証、ナビゲーション、およびフーム操作のためにChrome DevTools MCPツールシーケンスを実行する自律型ブラウザ自動化エージェント。トークンが多数含まれるブラウザスナップショットをメインセッションから除外するため、サブエージェントとして実行されます。

### config-analyzer

読み取り専用の設定分析エージェント。お客様のJSON設定をリソースプロファイル参照ファイルと照合して分析し、セキュリティポスチャ、機能の有効化、モードの移行、およびベストプラクティスに関する質問に回答します。

<Aside type="caution">
  config-analyzerエージェントは**エフェメラル**です &mdash; 各質問ごとに新たに生成され、ディスパッチ間でメモリを保持しません。フォローアップの質問には、ディスパッチプロンプトに圧縮された以前の調査結果を含める必要があります。
</Aside>
