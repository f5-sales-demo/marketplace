**f5xc-brand** プラグインは、すべてのコンテンツタイプにわたって F5 コーポレートブランドアイデンティティを適用します。45 色の F5 パレット、タイポグラフィルール（Neusa Next Pro Wide ヘッドライン、Proxima Nova ボディ）、SVG 限定のロゴ使用、アイコンパックの選択、WCAG AA アクセシビリティ、および製品用語を対象としています。

<Badge text="v1.0.2" variant="note" /> <Badge text="Productivity" variant="tip" />

## インストール

```
/plugin install f5xc-brand@f5-sales-demo-marketplace
```

## スキル

### brand-guardian

すべてのコンテンツ作成に対するプロアクティブなブランドアイデンティティの適用。
ドキュメント、ダイアグラム、プレゼンテーション、動画サムネイル、ソーシャルメディアグラフィック、Mermaid チャート、または Excalidraw 図の作成時に自動的に有効化されます。F5 のカラー、タイポグラフィ、ロゴ、アイコン、トーンオブボイス、およびアクセシビリティの正しい使用を保証します。

### brand-reviewer

既存コンテンツの F5 ブランドコンプライアンスのレビュー。カラー、タイポグラフィ、ロゴ形式、アイコン選択、代替テキスト、コントラスト比、および製品用語を監査します。VIOLATIONS（違反）、WARNINGS（警告）、SUGGESTIONS（提案）を含む構造化されたレポートを生成します。

### visual-content

ブランデッドビジュアルコンテンツ向けのフォーマット固有のルール。Mermaid ダイアグラム、Excalidraw 図、スクリーンショット、PowerPoint プレゼンテーション、YouTube サムネイル、動画スクリプト、およびソーシャルメディアグラフィックに対し、アイコンパックの選択、スライドレイアウト、フォントフォールバック、およびプラットフォーム固有のサイズを含むガイダンスを提供します。

## コマンド

### /review-brand

```
/f5xc-brand:review-brand [path-or-glob]
```

コンテンツの F5 ブランドコンプライアンスをレビューします。レビューの範囲を指定するための任意のパスまたはグロブパターンを受け付けます。引数なしの場合、現在のディレクトリ内のすべてのコンテンツをレビューします。
