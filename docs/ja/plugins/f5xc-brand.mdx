**f5xc-brand** プラグインは、すべてのコンテンツタイプにわたって F5 のコーポレートブランドアイデンティティを適用します。45色の F5 パレット、タイポグラフィルール（Neusa Next Pro Wide 見出し、Proxima Nova 本文）、SVG のみのロゴ使用、アイコンパック選択、WCAG AA アクセシビリティ、および製品用語を対象としています。

<Badge text="v1.0.2" variant="note" /> <Badge text="Productivity" variant="tip" />

## インストール

```
/plugin install f5xc-brand@f5xc-salesdemos-marketplace
```

## スキル

### brand-guardian

すべてのコンテンツ作成に対するプロアクティブなブランドアイデンティティの適用。ドキュメント、図、プレゼンテーション、動画サムネイル、ソーシャルメディアグラフィック、Mermaid チャート、または Excalidraw 図面を作成する際に自動的にアクティブになります。F5 のカラー、タイポグラフィ、ロゴ、アイコン、トーン・オブ・ボイス、およびアクセシビリティの正しい使用を保証します。

### brand-reviewer

既存のコンテンツの F5 ブランド準拠をレビューします。カラー、タイポグラフィ、ロゴ形式、アイコン選択、代替テキスト、コントラスト比、および製品用語を監査します。VIOLATIONS（違反）、WARNINGS（警告）、SUGGESTIONS（提案）を含む構造化されたレポートを生成します。

### visual-content

ブランド化されたビジュアルコンテンツのフォーマット固有ルール。Mermaid 図、Excalidraw 図面、スクリーンショット、PowerPoint プレゼンテーション、YouTube サムネイル、動画スクリプト、およびソーシャルメディアグラフィックに対して、アイコンパック選択、スライドレイアウト、フォントフォールバック、プラットフォーム固有のサイズに関するガイダンスを提供します。

## コマンド

### /review-brand

```
/f5xc-brand:review-brand [path-or-glob]
```

コンテンツの F5 ブランド準拠をレビューします。レビュー範囲を指定するためのオプションのパスまたは glob パターンを受け付けます。引数なしの場合、現在のディレクトリ内のすべてのコンテンツをレビューします。
