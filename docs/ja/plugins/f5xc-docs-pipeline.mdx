**f5xc-docs-pipeline** プラグインは、f5-sales-demo リポジトリ向けのドキュメントパイプラインアーキテクチャに関する知識を提供します。コンテンツ作成のガイド、マルチリポジトリビルドにおける設定の所有権の明確化、およびローカル Docker ベースのプレビューを提供します。

<Badge text="v1.0.2" variant="note" /> <Badge text="Productivity" variant="tip" />

## インストール

```
/plugin install f5xc-docs-pipeline@f5-sales-demo-marketplace
```

## スキル

### content-author

ファイル構成、MDX 構文ルール、フロントマターの要件、画像参照、およびローカル Docker プレビューを網羅したコンテンツ作成ガイド。`docs/` コンテンツの作成・編集、MDX ファイルの操作、またはローカルプレビューのセットアップ時に有効になります。

### pipeline-navigator

4 つの上流リポジトリ（docs-theme、docs-builder、docs-control、docs-icons）にわたる変更箇所を説明するドキュメントパイプラインアーキテクチャ。設定の所有権（`astro.config.mjs`、`content.config.ts`）、リリースディスパッチチェーン、および管理対象ファイルの追跡を明確にします。

<Aside type="tip">
  特定の設定ファイルやコンポーネントを所有するリポジトリを把握する必要がある場合は、pipeline-navigator を使用してください。
</Aside>

## コマンド

### /preview-docs

```
/f5xc-docs-pipeline:preview-docs
```

ドキュメントをプレビューするためのローカル Docker 開発サーバーを起動します。ライブプレビュー用にローカルの `docs/` ディレクトリをマウントした状態で、共有 Docker ビルドイメージを実行します。
