**f5xc-docs-pipeline** プラグインは、f5xc-salesdemos リポジトリ向けのドキュメントパイプラインアーキテクチャの知識を提供します。コンテンツオーサリングのガイド、マルチリポジトリビルドにおける設定の所有権の明確化、およびローカルでの Docker ベースのプレビューを提供します。

<Badge text="v1.0.2" variant="note" /> <Badge text="Productivity" variant="tip" />

## インストール

```
/plugin install f5xc-docs-pipeline@f5xc-salesdemos-marketplace
```

## スキル

### content-author

ファイル構造、MDX 構文ルール、フロントマターの要件、画像参照、およびローカル Docker プレビューを網羅するコンテンツオーサリングガイドです。`docs/` コンテンツの作成や編集、MDX ファイルの操作、またはローカルプレビューのセットアップ時にアクティブになります。

### pipeline-navigator

4つの上流リポジトリ（docs-theme、docs-builder、docs-control、docs-icons）にまたがる変更箇所を説明するドキュメントパイプラインアーキテクチャです。設定の所有権（`astro.config.mjs`、`content.config.ts`）、リリースディスパッチチェーン、および管理対象ファイルの追跡を明確にします。

<Aside type="tip">
  特定の設定ファイルやコンポーネントをどのリポジトリが所有しているかを理解する必要がある場合は、pipeline-navigator を使用してください。
</Aside>

## コマンド

### /preview-docs

```
/f5xc-docs-pipeline:preview-docs
```

ドキュメントをプレビューするためのローカル Docker 開発サーバーを起動します。ローカルの `docs/` ディレクトリをマウントした共有 Docker ビルドイメージを実行し、ライブプレビューを提供します。
