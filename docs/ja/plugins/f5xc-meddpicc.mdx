**f5xc-meddpicc** プラグインは、複雑なエンタープライズB2B営業向けの包括的なMEDDPICC営業資格審査およびディール実行フレームワークを提供します。MEDDPICCをチェックボックス作業から、エビデンスに基づく資格審査、構造化されたディールレビュー、および実践的なコーチングを備えた生きたディール実行システムへと変革します。

<Badge text="v1.0.4" variant="note" /> <Badge text="Productivity" variant="tip" />

## インストール

```
/plugin install f5xc-meddpicc@f5-sales-demo-marketplace
```

## MEDDPICCとは？

MEDDPICCは、複数のステークホルダー、技術的リスク、および調達ステップがモメンタムを妨げる可能性がある複雑なB2B営業向けに設計された資格審査およびディール実行フレームワークです。

| 文字 | 要素 | 目的 |
| ------ | ------- | ------- |
| **M** | Metrics（指標） | 顧客が期待する定量化されたビジネス成果 |
| **E** | Economic Buyer（経済的意思決定者） | 最終的な予算承認権限を持つ個人 |
| **D** | Decision Criteria（決定基準） | 選択肢を評価するために使用される要件 |
| **D** | Decision Process（決定プロセス） | 評価から合意までの段階的な経路 |
| **P** | Paper Process（書類プロセス） | 調達の仕組み（法務、セキュリティ、署名） |
| **I** | Identify Pain（課題の特定） | 緊迫感を生む切迫したビジネス課題 |
| **C** | Champion（チャンピオン） | 影響力のある社内支持者 |
| **C** | Competition（競合） | すべての代替案（ベンダー、内製開発、現状維持） |

## スキル

### meddpicc-coach

ディールの資格審査、パイプラインの健全性、予測精度、またはMEDDPICCの各要素について話し合う際に自動的に起動する、包括的なMEDDPICCメソドロジーコーチング。エビデンスに基づくガイダンス、運用リズムの推奨事項、および詳細な参考資料へのリンクを提供します。

<Aside type="tip" title="自動起動">
  このスキルは関連する場面で自動的に起動します — スラッシュコマンドは不要です。
</Aside>

### deal-qualification

ギャップ分析、リスク評価、および予測推奨事項を含む、スコア付きMEDDPICCスコアカード（0&ndash;24）を生成します。各要素は、確認されたエビデンスに基づいて0&ndash;3でスコアリングされます。

### deal-review

エビデンスのデルタ追跡、リスク識別、ならびに担当者と期日が明記されたアクションアイテムを含む、構造化された週次ディール検査を促進します。

### mutual-action-plan

MEDDPICCの要素に紐付けた顧客向けの相互アクションプランを構築します。評価マイルストーン、調達ロードマップ、導入準備状況、およびステークホルダーマッピングが含まれます。

### champion-test

5つの評価基準（権限、個人的な成果、アクセス、情報収集、行動）を適用し、0&ndash;15でスコアリングすることで、社内コンタクトが真のチャンピオン、新興チャンピオン、コーチ、または友好的なコンタクトのいずれであるかを判定します。

## コマンド

| コマンド | 目的 |
| ------- | ------- |
| `/f5xc-meddpicc:qualify-deal [account]` | MEDDPICCスコアカードでディールをスコアリングする |
| `/f5xc-meddpicc:deal-review [account]` | 週次ディール検査を促進する |
| `/f5xc-meddpicc:champion-test [contact]` | コンタクトが真のチャンピオンかどうかを評価する |
| `/f5xc-meddpicc:build-map [account]` | 相互アクションプランを構築する |

## エージェント

### deal-analyst

ローカルドキュメント、ミーティングノート、およびアカウントプランを確認することでディールの健全性を分析する読み取り専用のリサーチエージェント。エビデンスの引用、ギャップ分析、および優先順位付きのアクション推奨事項を含む構造化されたMEDDPICC評価レポートを生成します。

## 参考ライブラリ

プラグインには、オンデマンドで読み込まれる包括的な参考資料が含まれています。

- **MEDDPICCフレームワーク** &mdash; 全8要素の完全な定義とエビデンス要件
- **ディスカバリー質問集** &mdash; ディールサイクル全体を通じた各要素向けの実地テスト済み質問
- **役割と責任** &mdash; チームロール別のRACI責任マトリクス（AE、SE、CSM、PS、プロダクト、チャネル、ディールデスク、法務）
- **ディールステージゲート** &mdash; 次のステージへ進む前に満たすべき条件（初期資格審査からクローズド・ウォンまで）
- **アンチパターン** &mdash; エビデンスに基づく是正策を含む10の一般的な失敗モード
- **スコアリングルーブリック** &mdash; 各要素に対する客観的な0&ndash;3スコアリング基準
- **レビューテンプレート** &mdash; レッドフラグ指標を含む週次検査質問セット
- **MAPテンプレート** &mdash; 評価、調達、および導入ロードマップを含む顧客向けアクションプランの構造
