**f5xc-meddpicc** プラグインは、複雑なエンタープライズB2B営業向けの包括的なMEDDPICC営業資格審査およびディール実行フレームワークを提供します。MEDDPICCを単なるチェックボックス作業から、エビデンスに基づく資格審査、構造化されたディールレビュー、実行可能なコーチングを備えた生きたディール実行システムへと変革します。

<Badge text="v1.0.4" variant="note" /> <Badge text="Productivity" variant="tip" />

## インストール

```
/plugin install f5xc-meddpicc@f5xc-salesdemos-marketplace
```

## MEDDPICCとは？

MEDDPICCは、複数のステークホルダー、技術的リスク、調達プロセスが推進力を阻害する可能性がある複雑なB2B営業向けに設計された資格審査およびディール実行フレームワークです。

| 文字 | 要素 | 目的 |
| ------ | ------- | ------- |
| **M** | Metrics（メトリクス） | 顧客が期待する定量化されたビジネス成果 |
| **E** | Economic Buyer（経済的購買者） | 最終的な予算権限を持つ個人 |
| **D** | Decision Criteria（意思決定基準） | オプションを評価するために使用される要件 |
| **D** | Decision Process（意思決定プロセス） | 評価から合意までの段階的なパス |
| **P** | Paper Process（書類プロセス） | 調達の仕組み（法務、セキュリティ、署名） |
| **I** | Identify Pain（ペインの特定） | 緊急性を生み出す切迫したビジネス課題 |
| **C** | Champion（チャンピオン） | 影響力のある社内推進者 |
| **C** | Competition（競合） | すべての代替案（ベンダー、自社構築、現状維持） |

## スキル

### meddpicc-coach

ディールの資格審査、パイプラインの健全性、フォーキャストの精度、またはMEDDPICCの各要素について議論する際に自動的にアクティベートされる包括的なMEDDPICC方法論コーチングです。エビデンスに基づくガイダンス、運用リズムの推奨事項、および詳細なリファレンス資料へのリンクを提供します。

<Aside type="tip" title="自動アクティベート">
  このスキルは関連する場面で自動的にアクティベートされます。スラッシュコマンドは不要です。
</Aside>

### deal-qualification

ギャップ分析、リスク評価、フォーキャスト推奨事項を含むスコア付きMEDDPICCスコアカード（0&ndash;24）を生成します。各要素は検証済みのエビデンスに基づいて0&ndash;3でスコアリングされます。

### deal-review

エビデンスの差分追跡、リスクの特定、担当者名と期日が設定されたアクションアイテムを含む構造化された週次ディール検査を促進します。

### mutual-action-plan

MEDDPICC要素に紐づけた顧客向けの相互アクションプラン（Mutual Action Plan）を構築します。評価マイルストーン、調達ロードマップ、導入準備、ステークホルダーマッピングが含まれます。

### champion-test

5つの基準（Power、Personal Win、Access、Intel、Action）による評価を適用し、0&ndash;15でスコアリングすることで、社内コンタクトが真のチャンピオン、成長中のチャンピオン、コーチ、または友好的なコンタクトのいずれであるかを判定します。

## コマンド

| コマンド | 目的 |
| ------- | ------- |
| `/f5xc-meddpicc:qualify-deal [account]` | MEDDPICCスコアカードでディールをスコアリング |
| `/f5xc-meddpicc:deal-review [account]` | 週次ディール検査を実施 |
| `/f5xc-meddpicc:champion-test [contact]` | コンタクトが真のチャンピオンかどうかを評価 |
| `/f5xc-meddpicc:build-map [account]` | 相互アクションプランを構築 |

## エージェント

### deal-analyst

ローカルのドキュメント、ミーティングノート、アカウントプランをレビューすることでディールの健全性を分析する読み取り専用のリサーチエージェントです。エビデンスの引用、ギャップ分析、優先順位付けされたアクション推奨事項を含む構造化されたMEDDPICC評価レポートを生成します。

## リファレンスライブラリ

このプラグインには、オンデマンドで読み込まれる包括的なリファレンス資料が含まれています：

- **MEDDPICCフレームワーク** &mdash; 全8要素の完全な定義とエビデンス要件
- **ディスカバリー質問集** &mdash; ディールサイクル全体にわたる各要素向けの実証済み質問集
- **役割と責任** &mdash; チームロール別のRACIオーナーシップマトリクス（AE、SE、CSM、PS、Product、Channel、Deal Desk、Legal）
- **ディールステージゲート** &mdash; 次のステージに進む前に満たすべき条件（初期資格審査からClosed Wonまで）
- **アンチパターン** &mdash; エビデンスに基づく改善策を含む10の一般的な失敗パターン
- **スコアリングルーブリック** &mdash; 各要素ごとの客観的な0&ndash;3スコアリング基準
- **レビューテンプレート** &mdash; レッドフラグインジケーター付きの週次検査質問セット
- **MAPテンプレート** &mdash; 評価、調達、導入ロードマップを含む顧客向けアクションプラン構造
