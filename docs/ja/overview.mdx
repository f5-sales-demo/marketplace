**Claude Codeプラグインマーケットプレイス**は、Claude Codeを新しいスキル、コマンド、エージェントで拡張するプラグインのキュレーションされたコレクションです。マーケットプレイスを使用すると、チーム間でツールを共有でき、すべての開発者が手動セットアップなしで同じ機能を利用できます。

**f5xc-salesdemos マーケットプレイス**は、[f5xc-salesdemos](https://github.com/f5xc-salesdemos)ドキュメントプラットフォーム向けに構築されたプラグインを提供します。このプラットフォームは、共有ビルドパイプラインを使用してAstro/StarlightサイトをGitHub Pagesに公開する20以上のリポジトリで構成されるエコシステムです。

## 仕組み

1. **マーケットプレイスを追加** — Claude Codeにプラグインソースとして登録します
2. **プラグインをインストール** — 各プラグインがClaude Codeセッションにスキルとコマンドを追加します
3. **ツールを使用** — プラグインはコンテキストに基づいて自動的に有効化されるか、コマンドを直接呼び出して使用します

## 利用可能なプラグイン

| プラグイン | カテゴリ | 説明 |
|--------|----------|-------------|
| [f5xc-docs-tools](/marketplace/plugins/f5xc-docs-tools/) | 生産性 | ドキュメントリポジトリ向けのMDXコンテンツ検証・レビューツール |
| [f5xc-sales-engineer](/marketplace/plugins/f5xc-sales-engineer/) | 生産性 | セールスエンジニアペルソナフレームワーク — デモ実行、プレゼンテーション、Q&A、環境管理 |

## クイックスタート

マーケットプレイスを追加し、ドキュメントツールプラグインをインストールします：

```
/plugin marketplace add f5xc-salesdemos/marketplace
/plugin install f5xc-docs-tools@f5xc-salesdemos-marketplace
```

完全なウォークスルーについては、[はじめに](/marketplace/getting-started/)を参照してください。

## アーキテクチャ

マーケットプレイスは、1つ以上のプラグインを含むGitリポジトリです。各プラグインは、スキル、コマンド、エージェントを単一のインストール可能なユニットにパッケージ化します。

```
Marketplace (marketplace.json)
└── Plugin (plugin.json)
    ├── Skills (SKILL.md)
    │   └── References (data files)
    ├── Commands (.md)
    └── Agents (.md)
```

- **マーケットプレイス** — レジストリです。検出とインストールのためのメタデータを含む利用可能なプラグインを一覧表示します。
- **プラグイン** — インストール可能なユニットです。1つ以上のスキル、コマンド、またはエージェントを含みます。
- **スキル** — タスクに応じてClaudeが自動的に有効化するコンテキスト知識です。
- **コマンド** — ユーザーが直接呼び出すスラッシュコマンドです（例：`/review-mdx`）。
- **エージェント** — 複雑な複数ステップのタスクを自律的に処理する特化型サブエージェントです。
