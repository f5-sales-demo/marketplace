**Claude Code プラグインマーケットプレイス**とは、新しいスキル、コマンド、エージェントで Claude Code を拡張するプラグインをまとめたコレクションです。マーケットプレイスを活用することで、チーム全体でツールを共有し、すべての開発者が手動セットアップなしに同じ機能を利用できるようになります。

**f5-sales-demo マーケットプレイス**は、[f5-sales-demo](https://github.com/f5-sales-demo) ドキュメントプラットフォーム向けに構築されたプラグインを提供します。これは、共有ビルドパイプラインを使用して Astro/Starlight サイトを GitHub Pages に公開する 20 以上のリポジトリで構成されるエコシステムです。

## 仕組み

1. **マーケットプレイスを追加する** — Claude Code に登録し、プラグインソースとして認識させます
2. **プラグインをインストールする** — マーケットプレイスから各プラグインをインストールすると、スキルとコマンドが Claude Code セッションに追加されます
3. **ツールを使用する** — プラグインはコンテキストに基づいて自動的に有効化されるか、コマンドを直接呼び出して使用します

## 利用可能なプラグイン

| プラグイン | カテゴリ | 説明 |
|--------|----------|-------------|
| [f5xc-docs-tools](/marketplace/ja/plugins/f5xc-docs-tools/) | 生産性 | ドキュメントリポジトリ向け MDX コンテンツ検証およびレビューツール |
| [f5xc-sales-engineer](/marketplace/ja/plugins/f5xc-sales-engineer/) | 生産性 | セールスエンジニアペルソナフレームワーク — デモ実行、プレゼンテーション、Q&A、環境管理 |

## クイックスタート

マーケットプレイスを追加し、ドキュメントツールプラグインをインストールします：

```
/plugin marketplace add f5-sales-demo/marketplace
/plugin install f5xc-docs-tools@f5-sales-demo-marketplace
```

詳細な手順については、[はじめに](/marketplace/ja/getting-started/)を参照してください。

## アーキテクチャ

マーケットプレイスは、1 つ以上のプラグインを含む Git リポジトリです。各プラグインは、スキル、コマンド、エージェントをひとつのインストール可能な単位にまとめたものです。

```
Marketplace (marketplace.json)
└── Plugin (plugin.json)
    ├── Skills (SKILL.md)
    │   └── References (data files)
    ├── Commands (.md)
    └── Agents (.md)
```

- **マーケットプレイス** — レジストリ。発見とインストールのためのメタデータとともに、利用可能なプラグインを一覧表示します。
- **プラグイン** — インストール可能な単位。1 つ以上のスキル、コマンド、またはエージェントを含みます。
- **スキル** — Claude が現在のタスクに基づいて自動的に有効化するコンテキスト知識。
- **コマンド** — ユーザーが直接呼び出すスラッシュコマンド（例：`/review-mdx`）。
- **エージェント** — 複雑なマルチステップタスクを自律的に処理する専門的なサブエージェント。
