Un **marketplace di plugin di Claude Code** è una raccolta curata di
plugin che estendono Claude Code con nuove competenze, comandi e
agenti. I marketplace consentono ai team di condividere strumenti
affinché ogni sviluppatore disponga delle stesse funzionalità senza
configurazioni manuali.

Il **marketplace f5-sales-demo** fornisce plugin sviluppati per
la piattaforma di documentazione [f5-sales-demo](https://github.com/f5-sales-demo)
— un ecosistema di oltre 20 repository che
pubblicano siti Astro/Starlight su GitHub Pages utilizzando una
pipeline di build condivisa.

## Come Funziona

1. **Aggiungere il marketplace** a Claude Code — registrandolo
   come sorgente di plugin
2. **Installare i plugin** dal marketplace — ogni plugin aggiunge
   competenze e comandi alla sessione di Claude Code
3. **Utilizzare gli strumenti** — i plugin si attivano automaticamente in base al
   contesto, oppure è possibile richiamare direttamente i loro comandi

## Plugin Disponibili

| Plugin | Categoria | Descrizione |
|--------|-----------|-------------|
| [f5xc-docs-tools](/marketplace/it/plugins/f5xc-docs-tools/) | Produttività | Strumenti di validazione e revisione dei contenuti MDX per i repository di documentazione |
| [f5xc-sales-engineer](/marketplace/it/plugins/f5xc-sales-engineer/) | Produttività | Framework per il ruolo di Sales Engineer — esecuzione demo, presentazione, Q&A, gestione dell'ambiente |

## Avvio Rapido

Aggiungere il marketplace e installare il plugin degli strumenti di documentazione:

```
/plugin marketplace add f5-sales-demo/marketplace
/plugin install f5xc-docs-tools@f5-sales-demo-marketplace
```

Consultare [Per iniziare](/marketplace/it/getting-started/) per una
guida completa.

## Architettura

Un marketplace è un repository Git che contiene uno o più
plugin. Ogni plugin raccoglie competenze, comandi e agenti
in un'unica unità installabile.

```
Marketplace (marketplace.json)
└── Plugin (plugin.json)
    ├── Skills (SKILL.md)
    │   └── References (data files)
    ├── Commands (.md)
    └── Agents (.md)
```

- **Marketplace** — il registro. Elenca i plugin disponibili
  con i metadati per la ricerca e l'installazione.
- **Plugin** — l'unità installabile. Contiene una o più
  competenze, comandi o agenti.
- **Skill** — conoscenza contestuale che Claude attiva
  automaticamente in base all'attività in corso.
- **Command** — un comando slash che l'utente richiama direttamente
  (ad es., `/review-mdx`).
- **Agent** — un agente secondario specializzato che gestisce attività complesse e
  multi-step in modo autonomo.
