Questa guida illustra come aggiungere il marketplace f5-sales-demo
a Claude Code e installare il primo plugin.

## Prerequisiti

- Claude Code **v1.0.33** o versione successiva (supporto al marketplace di plugin)
- Un account GitHub con accesso all'organizzazione
  [f5-sales-demo](https://github.com/f5-sales-demo)

## Installazione

<Steps>

1. **Aggiungere il marketplace**

   In qualsiasi sessione di Claude Code, eseguire:

   ```
   /plugin marketplace add f5-sales-demo/marketplace
   ```

   Questo registra il marketplace come sorgente di plugin. È necessario
   eseguire questa operazione una sola volta.

2. **Sfogliare i plugin disponibili**

   Elencare i plugin offerti dal marketplace:

   ```
   /plugin marketplace list f5-sales-demo-marketplace
   ```

3. **Installare un plugin**

   Installare il plugin necessario:

   ```
   /plugin install f5xc-docs-tools@f5-sales-demo-marketplace
   /plugin install f5xc-sales-engineer@f5-sales-demo-marketplace
   ```

4. **Utilizzare il plugin**

   Navigare in qualsiasi repository di contenuti f5-sales-demo.

   Per gli **strumenti per la documentazione**, eseguire il comando di revisione MDX:

   ```
   /f5xc-docs-tools:review-mdx
   ```

   Per il **sales engineer**, utilizzare frasi trigger come
   "prepare the demo", "run the demo" o "walk through
   the demo" — il plugin si attiva automaticamente.

</Steps>

## Aggiornamento dei plugin

Per aggiornare un plugin installato all'ultima versione:

```
/plugin update f5xc-docs-tools@f5-sales-demo-marketplace
/plugin update f5xc-sales-engineer@f5-sales-demo-marketplace
```

## Disinstallazione

Rimuovere un plugin:

```
/plugin uninstall f5xc-docs-tools@f5-sales-demo-marketplace
/plugin uninstall f5xc-sales-engineer@f5-sales-demo-marketplace
```

Rimuovere completamente il marketplace:

```
/plugin marketplace remove f5-sales-demo-marketplace
```

## Configurazione per il team

Per pre-configurare il marketplace per l'intero team, aggiungerlo
al file `.claude/settings.json` del progetto:

```json
{
  "extraKnownMarketplaces": [
    "f5-sales-demo/marketplace"
  ],
  "enabledPlugins": [
    "f5xc-docs-tools@f5-sales-demo-marketplace",
    "f5xc-sales-engineer@f5-sales-demo-marketplace"
  ]
}
```

<Aside type="tip" title="Impostazioni condivise">
  Eseguire il commit di `.claude/settings.json` nel repository significa che
  ogni membro del team avrà a disposizione il marketplace e i plugin
  automaticamente, senza alcuna configurazione manuale.
</Aside>

Questo garantisce che ogni collaboratore disponga degli stessi strumenti
fin dalla prima sessione di Claude Code nel repository.
