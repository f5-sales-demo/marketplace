Il plugin **f5xc-meddpicc** fornisce un framework completo MEDDPICC per la qualificazione delle vendite
e l'esecuzione delle trattative per vendite B2B enterprise
complesse. Trasforma il MEDDPICC da un semplice esercizio di verifica in un sistema
di esecuzione delle trattative dinamico, con qualificazione basata su prove,
revisioni strutturate delle trattative e coaching operativo.

<Badge text="v1.0.4" variant="note" /> <Badge text="Produttività" variant="tip" />

## Installazione

```
/plugin install f5xc-meddpicc@f5-sales-demo-marketplace
```

## Che cos'è il MEDDPICC?

Il MEDDPICC è un framework di qualificazione e di esecuzione delle trattative progettato per
vendite B2B complesse, in cui più stakeholder, rischi tecnici e fasi
di approvvigionamento possono compromettere lo slancio commerciale.

| Lettera | Elemento | Scopo |
| ------ | ------- | ------- |
| **M** | Metriche | Risultati aziendali quantificati attesi dal cliente |
| **E** | Acquirente economico | Individuo con autorità finale sul budget |
| **D** | Criteri decisionali | Requisiti utilizzati per valutare le opzioni |
| **D** | Processo decisionale | Percorso passo dopo passo dalla valutazione all'accordo |
| **P** | Processo documentale | Meccanismi di approvvigionamento (legale, sicurezza, firma) |
| **I** | Identificazione del problema | Problema aziendale rilevante che genera urgenza |
| **C** | Champion | Referente interno influente |
| **C** | Concorrenza | Tutte le alternative (fornitori, sviluppo interno, non fare nulla) |

## Funzionalità

### meddpicc-coach

Coaching metodologico MEDDPICC completo che si attiva
automaticamente quando si discute di qualificazione delle trattative, stato della pipeline,
accuratezza delle previsioni o qualsiasi elemento MEDDPICC. Fornisce indicazioni basate
su prove, raccomandazioni sul ritmo operativo e collegamenti a materiale
di riferimento dettagliato.

<Aside type="tip" title="Attivazione automatica">
  Questa funzionalità si attiva automaticamente quando pertinente — non è necessario
  alcun comando slash.
</Aside>

### deal-qualification

Produce una scorecard MEDDPICC con punteggio (0&ndash;24), analisi delle lacune,
valutazione del rischio e raccomandazione di previsione. Ogni elemento viene valutato
da 0 a 3 in base a prove verificate.

### deal-review

Facilita ispezioni settimanali strutturate delle trattative con monitoraggio dei delta di prove,
identificazione dei rischi e azioni con responsabili nominati
e date di scadenza.

### mutual-action-plan

Crea un Piano d'Azione Condiviso orientato al cliente, collegato agli elementi
MEDDPICC. Include tappe di valutazione, roadmap di approvvigionamento,
preparazione all'implementazione e mappatura degli stakeholder.

### champion-test

Applica una valutazione basata su 5 criteri (Potere, Vantaggio personale, Accesso,
Intelligence, Azione) con punteggio da 0 a 15 per determinare se il
referente interno è un vero champion, un champion emergente, un coach o un
contatto favorevole.

## Comandi

| Comando | Scopo |
| ------- | ------- |
| `/f5xc-meddpicc:qualify-deal [account]` | Valutare una trattativa con la scorecard MEDDPICC |
| `/f5xc-meddpicc:deal-review [account]` | Facilitare l'ispezione settimanale della trattativa |
| `/f5xc-meddpicc:champion-test [contact]` | Valutare se un contatto è un vero champion |
| `/f5xc-meddpicc:build-map [account]` | Creare un Piano d'Azione Condiviso |

## Agenti

### deal-analyst

Agente di ricerca in sola lettura che analizza la salute delle trattative esaminando
documentazione locale, note di riunione e piani account. Produce
report di valutazione MEDDPICC strutturati con citazioni di prove,
analisi delle lacune e raccomandazioni di azioni prioritarie.

## Libreria di riferimento

Il plugin include materiale di riferimento completo che viene caricato su
richiesta:

- **Framework MEDDPICC** &mdash; definizioni complete e requisiti di prove
  per tutti gli 8 elementi
- **Domande di discovery** &mdash; domande testate sul campo per ogni
  elemento lungo il ciclo di vendita
- **Ruoli &amp; Responsabilità** &mdash; matrice RACI di ownership per
  ruolo del team (AE, SE, CSM, PS, Product, Channel, Deal Desk, Legal)
- **Gate per fase della trattativa** &mdash; condizioni che devono essere soddisfatte prima di avanzare
  (dalla Qualificazione iniziale fino a Chiuso Vinto)
- **Anti-pattern** &mdash; 10 modalità di errore comuni con
  rimedi basati su prove
- **Rubrica di punteggio** &mdash; criteri di valutazione oggettiva da 0 a 3
  per elemento
- **Template di revisione** &mdash; serie di domande per l'ispezione settimanale con
  indicatori di segnale d'allarme
- **Template MAP** &mdash; struttura del piano d'azione orientato al cliente
  con roadmap di valutazione, approvvigionamento e implementazione
