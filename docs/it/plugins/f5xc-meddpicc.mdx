Il plugin **f5xc-meddpicc** fornisce un framework completo di qualificazione
delle vendite e di esecuzione delle trattative MEDDPICC per vendite B2B
enterprise complesse. Trasforma il MEDDPICC da un semplice esercizio di
checklist in un sistema vivente di esecuzione delle trattative con
qualificazione basata sulle evidenze, revisioni strutturate delle
trattative e coaching operativo.

<Badge text="v1.0.4" variant="note" /> <Badge text="Productivity" variant="tip" />

## Installazione

```
/plugin install f5xc-meddpicc@f5xc-salesdemos-marketplace
```

## Cos'è il MEDDPICC?

Il MEDDPICC è un framework di qualificazione e di esecuzione delle
trattative progettato per vendite B2B complesse in cui molteplici
stakeholder, rischi tecnici e fasi di procurement possono compromettere
lo slancio.

| Lettera | Elemento | Scopo |
| ------- | -------- | ----- |
| **M** | Metrics | Risultati di business quantificati che il cliente si aspetta |
| **E** | Economic Buyer | Individuo con l'autorità finale sul budget |
| **D** | Decision Criteria | Requisiti utilizzati per valutare le opzioni |
| **D** | Decision Process | Percorso passo dopo passo dalla valutazione all'accordo |
| **P** | Paper Process | Meccanismi di procurement (legale, sicurezza, firma) |
| **I** | Identify Pain | Problema di business urgente che crea necessità di azione |
| **C** | Champion | Sostenitore interno influente |
| **C** | Competition | Tutte le alternative (vendor, sviluppo interno, non fare nulla) |

## Skill

### meddpicc-coach

Coaching completo sulla metodologia MEDDPICC che si attiva
automaticamente quando si discute di qualificazione delle trattative,
salute della pipeline, accuratezza delle previsioni o qualsiasi
elemento MEDDPICC. Fornisce indicazioni basate sulle evidenze,
raccomandazioni sul ritmo operativo e link a materiale di
riferimento dettagliato.

<Aside type="tip" title="Attivazione automatica">
  Questa skill si attiva automaticamente quando pertinente — non è
  necessario alcun comando slash.
</Aside>

### deal-qualification

Produce una scorecard MEDDPICC con punteggio (0&ndash;24) comprensiva
di analisi delle lacune, valutazione del rischio e raccomandazione
per la previsione. Ogni elemento è valutato da 0&ndash;3 sulla base
di evidenze verificate.

### deal-review

Facilita ispezioni settimanali strutturate delle trattative con
tracciamento dei delta delle evidenze, identificazione dei rischi e
azioni con responsabili nominati e scadenze.

### mutual-action-plan

Costruisce un Mutual Action Plan rivolto al cliente e collegato
agli elementi MEDDPICC. Include milestone di valutazione, roadmap
di procurement, preparazione all'implementazione e mappatura degli
stakeholder.

### champion-test

Applica una valutazione basata su 5 criteri (Potere, Vantaggio
Personale, Accesso, Intelligence, Azione) con punteggio da
0&ndash;15 per determinare se il vostro contatto interno è un vero
champion, un champion emergente, un coach o un contatto amichevole.

## Comandi

| Comando | Scopo |
| ------- | ----- |
| `/f5xc-meddpicc:qualify-deal [account]` | Valutare una trattativa con la scorecard MEDDPICC |
| `/f5xc-meddpicc:deal-review [account]` | Facilitare l'ispezione settimanale della trattativa |
| `/f5xc-meddpicc:champion-test [contact]` | Valutare se un contatto è un vero champion |
| `/f5xc-meddpicc:build-map [account]` | Costruire un Mutual Action Plan |

## Agenti

### deal-analyst

Agente di ricerca in sola lettura che analizza lo stato di salute
della trattativa esaminando documentazione locale, note delle
riunioni e piani account. Produce report di valutazione MEDDPICC
strutturati con citazioni delle evidenze, analisi delle lacune e
raccomandazioni operative prioritizzate.

## Libreria di Riferimento

Il plugin include materiale di riferimento completo che si carica
su richiesta:

- **Framework MEDDPICC** &mdash; definizioni complete e requisiti
  di evidenza per tutti gli 8 elementi
- **Domande di Discovery** &mdash; domande testate sul campo per
  ogni elemento lungo il ciclo della trattativa
- **Ruoli &amp; Responsabilità** &mdash; matrice RACI di ownership
  per ruolo del team (AE, SE, CSM, PS, Product, Channel, Deal Desk, Legal)
- **Gate di Fase della Trattativa** &mdash; cosa deve essere vero
  prima di avanzare (dalla Qualificazione Iniziale al Closed Won)
- **Anti-Pattern** &mdash; 10 modalità di fallimento comuni con
  rimedi basati sulle evidenze
- **Rubrica di Valutazione** &mdash; criteri di punteggio oggettivi
  da 0&ndash;3 per ogni elemento
- **Template di Revisione** &mdash; set di domande per l'ispezione
  settimanale con indicatori di segnali di allarme
- **Template MAP** &mdash; struttura del piano d'azione rivolto al
  cliente con roadmap di valutazione, procurement e implementazione
