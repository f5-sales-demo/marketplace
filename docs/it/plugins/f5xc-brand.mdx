Il plugin **f5xc-brand** applica l'identità del brand aziendale F5
su tutti i tipi di contenuto. Copre la palette F5 di 45 colori,
le regole tipografiche (titoli Neusa Next Pro Wide, corpo testo Proxima Nova),
l'utilizzo esclusivo di loghi SVG, la selezione dei pacchetti di icone, l'accessibilità
WCAG AA e la terminologia di prodotto.

<Badge text="v1.0.2" variant="note" /> <Badge text="Produttività" variant="tip" />

## Installazione

```
/plugin install f5xc-brand@f5-sales-demo-marketplace
```

## Competenze

### brand-guardian

Applicazione proattiva dell'identità del brand per la creazione di tutti i contenuti.
Si attiva automaticamente durante la creazione di documentazione, diagrammi,
presentazioni, miniature video, grafiche per i social media, grafici Mermaid
o disegni Excalidraw. Garantisce l'uso corretto di colori F5,
tipografia, loghi, icone, tono di voce e accessibilità.

### brand-reviewer

Esamina i contenuti esistenti per verificarne la conformità al brand F5. Controlla colori,
tipografia, formato del logo, selezione delle icone, testo alternativo, rapporti di contrasto
e terminologia di prodotto. Produce report strutturati con
VIOLAZIONI, AVVERTENZE e SUGGERIMENTI.

### visual-content

Regole specifiche per formato relative ai contenuti visivi brandizzati. Fornisce
indicazioni per diagrammi Mermaid, disegni Excalidraw, screenshot,
presentazioni PowerPoint, miniature YouTube, script video e
grafiche per i social media, con selezione dei pacchetti di icone, layout delle diapositive,
font di fallback e dimensioni specifiche per piattaforma.

## Comandi

### /review-brand

```
/f5xc-brand:review-brand [path-or-glob]
```

Esamina i contenuti per verificarne la conformità al brand F5. Accetta un percorso
o un pattern glob opzionale per circoscrivere la revisione. Senza argomenti, esamina
tutti i contenuti nella directory corrente.
