Il plugin **f5xc-brand** impone l'identità aziendale del marchio F5
su tutti i tipi di contenuto. Copre la palette di 45 colori F5,
le regole tipografiche (titoli in Neusa Next Pro Wide, corpo del testo
in Proxima Nova), l'utilizzo dei loghi esclusivamente in formato SVG,
la selezione dei pacchetti di icone, l'accessibilità WCAG AA
e la terminologia di prodotto.

<Badge text="v1.0.2" variant="note" /> <Badge text="Productivity" variant="tip" />

## Installazione

```
/plugin install f5xc-brand@f5xc-salesdemos-marketplace
```

## Competenze

### brand-guardian

Applicazione proattiva dell'identità del marchio per tutta la creazione di contenuti.
Si attiva automaticamente durante la creazione di documentazione, diagrammi,
presentazioni, miniature video, grafiche per i social media, grafici Mermaid
o disegni Excalidraw. Garantisce l'uso corretto di colori F5,
tipografia, loghi, icone, tono di voce e accessibilità.

### brand-reviewer

Revisiona i contenuti esistenti per la conformità al marchio F5. Verifica colori,
tipografia, formato del logo, selezione delle icone, testo alternativo, rapporti
di contrasto e terminologia di prodotto. Produce report strutturati con
VIOLAZIONI, AVVERTENZE e SUGGERIMENTI.

### visual-content

Regole specifiche per formato per i contenuti visivi brandizzati. Fornisce
indicazioni per diagrammi Mermaid, disegni Excalidraw, screenshot,
presentazioni PowerPoint, miniature YouTube, script video e
grafiche per i social media con selezione dei pacchetti di icone, layout delle slide,
font di fallback e dimensioni specifiche per piattaforma.

## Comandi

### /review-brand

```
/f5xc-brand:review-brand [path-or-glob]
```

Revisiona i contenuti per la conformità al marchio F5. Accetta un percorso opzionale
o un pattern glob per delimitare l'ambito della revisione. Senza argomenti, revisiona
tutti i contenuti nella directory corrente.
