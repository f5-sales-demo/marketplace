che assistono nella documentazione, nei flussi di lavoro demo, nella metodologia di vendita,
nel branding e nelle operazioni di sviluppo.

## Cosa contiene un Plugin?

Ogni plugin può includere:

- **Competenze (Skills)** — conoscenze contestuali che Claude attiva
  automaticamente quando pertinenti (es., revisione di file MDX)
- **Comandi (Commands)** — comandi slash che si invocano direttamente
  (es., `/f5xc-docs-tools:review-mdx`)
- **Agenti (Agents)** — sotto-agenti specializzati per compiti complessi
