Ein **Claude Code Plugin-Marktplatz** ist eine kuratierte Sammlung von
Plugins, die Claude Code um neue Fähigkeiten, Befehle und
Agenten erweitern. Marktplätze ermöglichen es Teams, Werkzeuge zu teilen,
sodass jeder Entwickler dieselben Funktionen ohne manuelle Einrichtung erhält.

Der **f5-sales-demo Marktplatz** stellt Plugins bereit, die für die
[f5-sales-demo](https://github.com/f5-sales-demo)
Dokumentationsplattform entwickelt wurden — ein Ökosystem aus mehr als 20 Repositories, die
Astro/Starlight-Seiten über eine gemeinsame Build-Pipeline auf GitHub Pages veröffentlichen.

## So funktioniert es

1. **Marktplatz hinzufügen** zu Claude Code — dies registriert ihn
   als Plugin-Quelle
2. **Plugins installieren** aus dem Marktplatz — jedes Plugin fügt
   Ihrer Claude Code-Sitzung Fähigkeiten und Befehle hinzu
3. **Werkzeuge verwenden** — Plugins werden automatisch basierend auf dem
   Kontext aktiviert, oder Sie rufen ihre Befehle direkt auf

## Schnellstart

Marktplatz hinzufügen und das Docs-Tools-Plugin installieren:

```
/plugin marketplace add f5-sales-demo/marketplace
/plugin install f5xc-docs-tools@f5-sales-demo-marketplace
```

Siehe [Erste Schritte](/marketplace/de/getting-started/) für eine
vollständige Anleitung.

## Architektur

Ein Marktplatz ist ein Git-Repository, das ein oder mehrere
Plugins enthält. Jedes Plugin bündelt Fähigkeiten, Befehle und Agenten
in eine einzelne installierbare Einheit.

```
Marketplace (marketplace.json)
└── Plugin (plugin.json)
    ├── Skills (SKILL.md)
    │   └── References (data files)
    ├── Commands (.md)
    └── Agents (.md)
```

- **Marktplatz** — die Registry. Listet verfügbare Plugins
  mit Metadaten zur Auffindbarkeit und Installation auf.
- **Plugin** — die installierbare Einheit. Enthält eine oder mehrere
  Fähigkeiten, Befehle oder Agenten.
- **Skill** — kontextuelles Wissen, das Claude automatisch basierend
  auf der jeweiligen Aufgabe aktiviert.
- **Befehl** — ein Slash-Befehl, den der Benutzer direkt aufruft
  (z. B. `/review-mdx`).
- **Agent** — ein spezialisierter Unteragent, der komplexe,
  mehrstufige Aufgaben eigenständig bearbeitet.
