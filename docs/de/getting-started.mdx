Diese Anleitung führt Sie durch das Hinzufügen des f5-sales-demo-Marktplatzes
zu Claude Code und die Installation Ihres ersten Plugins.

## Voraussetzungen

- Claude Code **v1.0.33** oder höher (Plugin-Marktplatz-Unterstützung)
- Ein GitHub-Konto mit Zugriff auf die
  [f5-sales-demo](https://github.com/f5-sales-demo)
  Organisation

## Installation

<Steps>

1. **Marktplatz hinzufügen**

   Führen Sie in einer beliebigen Claude Code-Sitzung folgenden Befehl aus:

   ```
   /plugin marketplace add f5-sales-demo/marketplace
   ```

   Dadurch wird der Marktplatz als Plugin-Quelle registriert. Dieser Schritt
   muss nur einmal durchgeführt werden.

2. **Verfügbare Plugins durchsuchen**

   Zeigen Sie an, was der Marktplatz bietet:

   ```
   /plugin marketplace list f5-sales-demo-marketplace
   ```

3. **Ein Plugin installieren**

   Installieren Sie das benötigte Plugin:

   ```
   /plugin install f5xc-docs-tools@f5-sales-demo-marketplace
   /plugin install f5xc-sales-engineer@f5-sales-demo-marketplace
   ```

4. **Das Plugin verwenden**

   Navigieren Sie zu einem beliebigen f5-sales-demo-Inhalts-Repository.

   Für **Dokumentationswerkzeuge** führen Sie den MDX-Überprüfungsbefehl aus:

   ```
   /f5xc-docs-tools:review-mdx
   ```

   Für den **Sales Engineer** verwenden Sie Trigger-Phrasen wie
   „prepare the demo", „run the demo" oder „walk through
   the demo" – das Plugin wird automatisch aktiviert.

</Steps>

## Plugins aktualisieren

So aktualisieren Sie ein installiertes Plugin auf die neueste Version:

```
/plugin update f5xc-docs-tools@f5-sales-demo-marketplace
/plugin update f5xc-sales-engineer@f5-sales-demo-marketplace
```

## Deinstallation

Ein Plugin entfernen:

```
/plugin uninstall f5xc-docs-tools@f5-sales-demo-marketplace
/plugin uninstall f5xc-sales-engineer@f5-sales-demo-marketplace
```

Den Marktplatz vollständig entfernen:

```
/plugin marketplace remove f5-sales-demo-marketplace
```

## Team-Konfiguration

Um den Marktplatz für Ihr gesamtes Team vorzukonfigurieren, fügen Sie ihn
zur `.claude/settings.json` Ihres Projekts hinzu:

```json
{
  "extraKnownMarketplaces": [
    "f5-sales-demo/marketplace"
  ],
  "enabledPlugins": [
    "f5xc-docs-tools@f5-sales-demo-marketplace",
    "f5xc-sales-engineer@f5-sales-demo-marketplace"
  ]
}
```

<Aside type="tip" title="Gemeinsame Einstellungen">
  Wenn Sie `.claude/settings.json` in Ihr Repository einchecken, erhalten
  alle Teammitglieder den Marktplatz und die Plugins
  automatisch – ohne manuelle Einrichtung.
</Aside>

Dadurch wird sichergestellt, dass jeder Mitwirkende dieselben Werkzeuge
bereits ab seiner ersten Claude Code-Sitzung im Repository zur Verfügung hat.
