Jedes Plugin fügt Claude Code Fähigkeiten, Befehle und Agenten hinzu,
die bei Dokumentation, Demo-Workflows, Vertriebsmethodik,
Branding und Entwicklungsbetrieb unterstützen.

## Was enthält ein Plugin?

Jedes Plugin kann Folgendes enthalten:

- **Fähigkeiten** — kontextbezogenes Wissen, das Claude automatisch
  aktiviert, wenn es relevant ist (z. B. beim Überprüfen von MDX-Dateien)
- **Befehle** — Slash-Befehle, die Sie direkt aufrufen
  (z. B. `/f5xc-docs-tools:review-mdx`)
- **Agenten** — spezialisierte Unteragenten für komplexe Aufgaben

## Ein Plugin installieren

```
/plugin install f5xc-docs-tools@f5-sales-demo-marketplace
/plugin install f5xc-sales-engineer@f5-sales-demo-marketplace
/plugin install f5xc-github-ops@f5-sales-demo-marketplace
/plugin install f5xc-docs-pipeline@f5-sales-demo-marketplace
/plugin install f5xc-brand@f5-sales-demo-marketplace
/plugin install f5xc-meddpicc@f5-sales-demo-marketplace
/plugin install f5xc-devcontainer@f5-sales-demo-marketplace
/plugin install f5xc-platform@f5-sales-demo-marketplace
/plugin install osint-framework@f5-sales-demo-marketplace
/plugin install f5xc-firecrawl@f5-sales-demo-marketplace
```

Weitere Informationen zur vollständigen Einrichtung finden Sie unter [Erste Schritte](/marketplace/getting-started/).
