Das **f5xc-platform**-Plugin bietet vollständige Automatisierung für die F5
Distributed Cloud Plattform. Es umfasst Web-Konsolenoperationen via
Chrome DevTools MCP, spezifikationsbewusste REST-API-Operationen via cURL,
Multi-Provider-Authentifizierung und Konfigurationsanalyse über
98 Ressourcentypen in 38 Domänen.

<Badge text="v3.1.1" variant="note" /> <Badge text="Produktivität" variant="tip" />

## Installation

```
/plugin install f5xc-platform@f5-sales-demo-marketplace
```

## Fähigkeiten

### api-operations

Spezifikationsbewusste API-CRUD-Operationen für 98 Ressourcentypen in
38 Domänen. Stellt präzise Endpunkte, Payload-Vorlagen,
Abhängigkeitsreihenfolgen und mehrstufige Workflow-Kompositionen für
HTTP-Load-Balancer, Ursprungspools, WAF-Richtlinien, DNS-Zonen,
Zertifikate, Cloud-Sites und mehr bereit.

### config-analysis

Konfigurationsanalyse und Beratungsfunktion. Analysiert
JSON-Konfigurationen von Kunden, um Fragen zur Sicherheitslage,
Feature-Aktivierung, Moduseinstellungen, Ausschlüssen und Best Practices
zu beantworten. Unterstützt mehrstufige Frage-und-Antwort-Sitzungen mit
komprimiertem Kontext zwischen den Aufrufen.

### api-auth

API-Authentifizierung mit Unterstützung für API-Token-Header-Authentifizierung
und P12-Zertifikatsauthentifizierung mit Token-Validierungsverfahren.

### console-auth

Multi-Provider-Authentifizierung für die F5 XC Web-Konsole.
Erkennt automatisch den Anmeldetyp: natives F5 XC-Login, Azure SSO mit
zwischengespeicherter Sitzung oder Azure SSO mit DUO Verified Push MFA.

### console-navigator

Deterministische Navigation zu F5 XC-Konsolenabschnitten. Übersetzt
Abschnittsnamen in URL-Pfade, überprüft den Sitzungsstatus und bestätigt
das Laden der Seite.

### platform-index / api-index / console-index

Intent-Router, die Benutzeranfragen klassifizieren und an die
entsprechende Fähigkeit und den entsprechenden Agenten delegieren.
Diese werden automatisch aktiviert und können nicht vom Benutzer
direkt aufgerufen werden.

## Befehle

### /check-api-token

```
/f5xc-platform:check-api-token
```

Validiert Ihr F5 XC-API-Token durch Delegation an den api-operator-Agenten.

### /login-console

```
/f5xc-platform:login-console
```

Authentifiziert sich bei der F5 Distributed Cloud Konsole über den
console-operator-Agenten.

## Agenten

### api-operator

Autonomer REST-API-Agent, der cURL- und jq-Sequenzen für Ressourcen-CRUD,
Token-Validierung und Konfigurationsoperationen ausführt. Läuft in einem
Unteragenten, um ausführliche API-JSON-Payloads aus dem Hauptsitzungskontext
herauszuhalten.

### console-operator

Autonomer Browser-Automatisierungsagent, der Chrome DevTools MCP-
Werkzeugsequenzen für Authentifizierung, Navigation und Formularinteraktionen
ausführt. Läuft in einem Unteragenten, um token-schwere Browser-Snapshots
aus der Hauptsitzung herauszuhalten.

### config-analyzer

Schreibgeschützter Konfigurationsanalyse-Agent. Analysiert
JSON-Konfigurationen von Kunden anhand von Ressourcenprofil-Referenzdateien,
um Fragen zur Sicherheitslage, Feature-Aktivierung, Modusübergängen und
Best Practices zu beantworten.

<Aside type="caution">
  Der config-analyzer-Agent ist **flüchtig** &mdash; er wird für jede Frage
  neu gestartet und hat kein Gedächtnis zwischen den Aufrufen. Folgefragen
  müssen komprimierte frühere Ergebnisse im Aufruf-Prompt enthalten.
</Aside>
