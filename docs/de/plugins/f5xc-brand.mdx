Das **f5xc-brand**-Plugin setzt die F5-Unternehmensmarkenidentität
für alle Inhaltstypen durch. Es umfasst die F5-Farbpalette mit 45 Farben,
Typografieregeln (Neusa Next Pro Wide für Überschriften, Proxima Nova
für Fließtext), ausschließliche SVG-Logonutzung, Symbolpaketauswahl, WCAG-AA-
Barrierefreiheit und Produktterminologie.

<Badge text="v1.0.2" variant="note" /> <Badge text="Produktivität" variant="tip" />

## Installation

```
/plugin install f5xc-brand@f5-sales-demo-marketplace
```

## Funktionen

### brand-guardian

Proaktive Durchsetzung der Markenidentität für alle Inhaltserstellungen.
Wird automatisch aktiviert beim Erstellen von Dokumentationen, Diagrammen,
Präsentationen, Video-Thumbnails, Social-Media-Grafiken, Mermaid-
Diagrammen oder Excalidraw-Zeichnungen. Stellt die korrekte Verwendung von F5-Farben,
Typografie, Logos, Icons, Tonalität und Barrierefreiheit sicher.

### brand-reviewer

Überprüft vorhandene Inhalte auf Konformität mit der F5-Markenidentität. Prüft Farben,
Typografie, Logoformat, Symbolauswahl, Alternativtexte, Kontrastverhältnisse
und Produktterminologie. Erstellt strukturierte Berichte mit
VERSTÖSSEN, WARNUNGEN und EMPFEHLUNGEN.

### visual-content

Formatspezifische Regeln für visuell gebrandete Inhalte. Bietet
Anleitungen für Mermaid-Diagramme, Excalidraw-Zeichnungen, Screenshots,
PowerPoint-Präsentationen, YouTube-Thumbnails, Videoskripte und
Social-Media-Grafiken mit Symbolpaketauswahl, Folienlayouts,
Schriftarten-Fallbacks und plattformspezifischen Abmessungen.

## Befehle

### /review-brand

```
/f5xc-brand:review-brand [path-or-glob]
```

Überprüft Inhalte auf Konformität mit der F5-Markenidentität. Akzeptiert einen optionalen Pfad
oder ein Glob-Muster zur Einschränkung der Überprüfung. Ohne Argumente werden
alle Inhalte im aktuellen Verzeichnis überprüft.
