Das Plugin **f5xc-brand** setzt die F5-Unternehmensmarkenidentität
über alle Inhaltstypen hinweg durch. Es umfasst die 45-Farben-F5-Palette,
Typografie-Regeln (Neusa Next Pro Wide für Überschriften, Proxima Nova
für Fließtext), ausschließliche SVG-Logo-Verwendung, Icon-Pack-Auswahl,
WCAG-AA-Barrierefreiheit und Produktterminologie.

<Badge text="v1.0.2" variant="note" /> <Badge text="Productivity" variant="tip" />

## Installation

```
/plugin install f5xc-brand@f5xc-salesdemos-marketplace
```

## Fähigkeiten

### brand-guardian

Proaktive Durchsetzung der Markenidentität für die gesamte Inhaltserstellung.
Wird automatisch aktiviert bei der Erstellung von Dokumentation, Diagrammen,
Präsentationen, Video-Thumbnails, Social-Media-Grafiken, Mermaid-Diagrammen
oder Excalidraw-Zeichnungen. Stellt die korrekte Verwendung von F5-Farben,
Typografie, Logos, Icons, Tonalität und Barrierefreiheit sicher.

### brand-reviewer

Überprüft bestehende Inhalte auf F5-Markenkonformität. Prüft Farben,
Typografie, Logo-Format, Icon-Auswahl, Alt-Texte, Kontrastverhältnisse
und Produktterminologie. Erstellt strukturierte Berichte mit
VERSTÖSSEN, WARNUNGEN und VORSCHLÄGEN.

### visual-content

Formatspezifische Regeln für markengerechte visuelle Inhalte. Bietet
Anleitungen für Mermaid-Diagramme, Excalidraw-Zeichnungen, Screenshots,
PowerPoint-Präsentationen, YouTube-Thumbnails, Videoskripte und
Social-Media-Grafiken mit Icon-Pack-Auswahl, Folienlayouts,
Schriftart-Fallbacks und plattformspezifischen Abmessungen.

## Befehle

### /review-brand

```
/f5xc-brand:review-brand [path-or-glob]
```

Überprüft Inhalte auf F5-Markenkonformität. Akzeptiert einen optionalen
Pfad oder ein Glob-Muster, um den Prüfungsumfang einzugrenzen. Ohne
Argumente werden alle Inhalte im aktuellen Verzeichnis überprüft.
