Das **f5xc-meddpicc**-Plugin bietet ein umfassendes MEDDPICC-Framework
für Vertriebsqualifizierung und Deal-Execution für komplexe
Enterprise-B2B-Verkäufe. Es verwandelt MEDDPICC von einer
Checkbox-Übung in ein lebendiges Deal-Execution-System mit
evidenzbasierter Qualifizierung, strukturierten Deal-Reviews und
umsetzbarem Coaching.

<Badge text="v1.0.4" variant="note" /> <Badge text="Productivity" variant="tip" />

## Installation

```
/plugin install f5xc-meddpicc@f5xc-salesdemos-marketplace
```

## Was ist MEDDPICC?

MEDDPICC ist ein Qualifizierungs- und Deal-Execution-Framework,
das für komplexe B2B-Verkäufe entwickelt wurde, bei denen mehrere
Stakeholder, technische Risiken und Beschaffungsschritte den
Fortschritt gefährden können.

| Buchstabe | Element | Zweck |
| --------- | ------- | ----- |
| **M** | Metrics | Quantifizierte Geschäftsergebnisse, die der Kunde erwartet |
| **E** | Economic Buyer | Person mit endgültiger Budgetautorität |
| **D** | Decision Criteria | Anforderungen zur Bewertung von Optionen |
| **D** | Decision Process | Schrittweiser Weg von der Evaluierung bis zur Vereinbarung |
| **P** | Paper Process | Beschaffungsmechanik (Recht, Sicherheit, Unterschrift) |
| **I** | Identify Pain | Überzeugendes geschäftliches Problem, das Dringlichkeit erzeugt |
| **C** | Champion | Einflussreicher interner Fürsprecher |
| **C** | Competition | Alle Alternativen (Anbieter, Eigenentwicklung, Nichtstun) |

## Skills

### meddpicc-coach

Umfassendes MEDDPICC-Methodik-Coaching, das automatisch aktiviert
wird, wenn Sie über Deal-Qualifizierung, Pipeline-Gesundheit,
Forecast-Genauigkeit oder ein beliebiges MEDDPICC-Element sprechen.
Bietet evidenzbasierte Anleitung, Empfehlungen für den
Betriebsrhythmus und Links zu detailliertem Referenzmaterial.

<Aside type="tip" title="Automatisch aktiviert">
  Dieser Skill wird automatisch aktiviert, wenn er relevant ist — kein
  Slash-Befehl erforderlich.
</Aside>

### deal-qualification

Erstellt eine bewertete MEDDPICC-Scorecard (0&ndash;24) mit
Gap-Analyse, Risikobewertung und Forecast-Empfehlung. Jedes Element
wird auf Basis verifizierter Evidenz mit 0&ndash;3 bewertet.

### deal-review

Ermöglicht strukturierte wöchentliche Deal-Inspektionen mit
Evidenz-Delta-Tracking, Risikoidentifikation und Aktionspunkten
mit benannten Verantwortlichen und Fälligkeitsterminen.

### mutual-action-plan

Erstellt einen kundenorientierten Mutual Action Plan, der an
MEDDPICC-Elemente gebunden ist. Enthält Evaluierungsmeilensteine,
Beschaffungsfahrplan, Implementierungsbereitschaft und
Stakeholder-Mapping.

### champion-test

Wendet eine 5-Kriterien-Bewertung (Power, Personal Win, Access,
Intel, Action) mit einer Punktzahl von 0&ndash;15 an, um
festzustellen, ob Ihr interner Kontakt ein echter Champion,
aufstrebender Champion, Coach oder freundlicher Kontakt ist.

## Befehle

| Befehl | Zweck |
| ------ | ----- |
| `/f5xc-meddpicc:qualify-deal [account]` | Deal mit MEDDPICC-Scorecard bewerten |
| `/f5xc-meddpicc:deal-review [account]` | Wöchentliche Deal-Inspektion durchführen |
| `/f5xc-meddpicc:champion-test [contact]` | Bewerten, ob ein Kontakt ein echter Champion ist |
| `/f5xc-meddpicc:build-map [account]` | Einen Mutual Action Plan erstellen |

## Agents

### deal-analyst

Schreibgeschützter Research-Agent, der die Deal-Gesundheit analysiert,
indem er lokale Dokumentation, Meeting-Notizen und Account-Pläne
überprüft. Erstellt strukturierte MEDDPICC-Bewertungsberichte mit
Evidenzzitaten, Gap-Analyse und priorisierten
Handlungsempfehlungen.

## Referenzbibliothek

Das Plugin enthält umfangreiches Referenzmaterial, das bei Bedarf
geladen wird:

- **MEDDPICC-Framework** &mdash; vollständige Definitionen und
  Evidenzanforderungen für alle 8 Elemente
- **Discovery-Fragen** &mdash; praxiserprobte Fragen für jedes
  Element über den gesamten Deal-Zyklus
- **Rollen &amp; Verantwortlichkeiten** &mdash; RACI-Zuständigkeitsmatrix
  nach Teamrolle (AE, SE, CSM, PS, Product, Channel, Deal Desk, Legal)
- **Deal Stage Gates** &mdash; was erfüllt sein muss, bevor
  fortgeschritten werden kann (von Early Qualification bis Closed Won)
- **Anti-Patterns** &mdash; 10 häufige Fehlermuster mit
  evidenzbasierter Behebung
- **Bewertungsrubrik** &mdash; objektive 0&ndash;3-Bewertungskriterien
  pro Element
- **Review-Vorlagen** &mdash; wöchentliche Inspektionsfragen mit
  Red-Flag-Indikatoren
- **MAP-Vorlagen** &mdash; kundenorientierte Aktionsplan-Struktur
  mit Evaluierungs-, Beschaffungs- und Implementierungsfahrplänen
