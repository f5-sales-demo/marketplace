Das **f5xc-meddpicc**-Plugin bietet ein umfassendes MEDDPICC-Vertriebsqualifizierungs- und Deal-Execution-Framework für komplexe B2B-Enterprise-Verkäufe. Es verwandelt MEDDPICC von einer Checkbox-Übung in ein lebendiges Deal-Execution-System mit evidenzbasierter Qualifizierung, strukturierten Deal-Reviews und umsetzbarem Coaching.

<Badge text="v1.0.4" variant="note" /> <Badge text="Productivity" variant="tip" />

## Installation

```
/plugin install f5xc-meddpicc@f5-sales-demo-marketplace
```

## Was ist MEDDPICC?

MEDDPICC ist ein Qualifizierungs- und Deal-Execution-Framework, das für komplexe B2B-Verkäufe entwickelt wurde, bei denen mehrere Stakeholder, technische Risiken und Beschaffungsschritte den Fortschritt gefährden können.

| Buchstabe | Element | Zweck |
| --------- | ------- | ----- |
| **M** | Metrics | Quantifizierte Geschäftsergebnisse, die der Kunde erwartet |
| **E** | Economic Buyer | Person mit abschließender Budgetverantwortung |
| **D** | Decision Criteria | Anforderungen zur Bewertung von Optionen |
| **D** | Decision Process | Schrittweise Vorgehensweise von der Evaluierung bis zur Einigung |
| **P** | Paper Process | Beschaffungsmechanismen (Recht, Sicherheit, Unterschrift) |
| **I** | Identify Pain | Zwingender Geschäftsschmerz, der Dringlichkeit erzeugt |
| **C** | Champion | Einflussreicher interner Befürworter |
| **C** | Competition | Alle Alternativen (Anbieter, Eigenentwicklung, Nichtstun) |

## Skills

### meddpicc-coach

Umfassendes MEDDPICC-Methoden-Coaching, das automatisch aktiviert wird, wenn Sie Deal-Qualifizierung, Pipeline-Gesundheit, Forecast-Genauigkeit oder ein MEDDPICC-Element besprechen. Bietet evidenzbasierte Anleitungen, Empfehlungen zum Betriebsrhythmus und Links zu detailliertem Referenzmaterial.

<Aside type="tip" title="Automatisch aktiviert">
  Dieser Skill wird automatisch aktiviert, wenn er relevant ist — kein Slash-Befehl erforderlich.
</Aside>

### deal-qualification

Erstellt eine bewertete MEDDPICC-Scorecard (0&ndash;24) mit Gap-Analyse, Risikobewertung und Forecast-Empfehlung. Jedes Element wird anhand verifizierter Nachweise mit 0&ndash;3 bewertet.

### deal-review

Ermöglicht strukturierte wöchentliche Deal-Inspektionen mit Evidence-Delta-Tracking, Risikoidentifizierung und Maßnahmen mit namentlich genannten Verantwortlichen und Fälligkeitsdaten.

### mutual-action-plan

Erstellt einen kundenorientierten Mutual Action Plan, der mit MEDDPICC-Elementen verknüpft ist. Enthält Evaluierungsmeilensteine, Beschaffungs-Roadmap, Implementierungsbereitschaft und Stakeholder-Mapping.

### champion-test

Wendet eine 5-Kriterien-Bewertung (Power, Personal Win, Access, Intel, Action) mit einer Punktzahl von 0&ndash;15 an, um zu bestimmen, ob Ihr interner Kontakt ein echter Champion, aufstrebender Champion, Coach oder freundlicher Kontakt ist.

## Befehle

| Befehl | Zweck |
| ------ | ----- |
| `/f5xc-meddpicc:qualify-deal [account]` | Deal mit MEDDPICC-Scorecard bewerten |
| `/f5xc-meddpicc:deal-review [account]` | Wöchentliche Deal-Inspektion durchführen |
| `/f5xc-meddpicc:champion-test [contact]` | Prüfen, ob Kontakt ein echter Champion ist |
| `/f5xc-meddpicc:build-map [account]` | Mutual Action Plan erstellen |

## Agenten

### deal-analyst

Schreibgeschützter Recherche-Agent, der den Deal-Zustand analysiert, indem er lokale Dokumentationen, Meeting-Notizen und Account-Pläne auswertet. Erstellt strukturierte MEDDPICC-Bewertungsberichte mit Nachweiszitaten, Gap-Analyse und priorisierten Handlungsempfehlungen.

## Referenzbibliothek

Das Plugin enthält umfassendes Referenzmaterial, das bei Bedarf geladen wird:

- **MEDDPICC-Framework** &mdash; vollständige Definitionen und Nachweisanforderungen für alle 8 Elemente
- **Discovery Questions** &mdash; in der Praxis erprobte Fragen für jedes Element über den gesamten Deal-Zyklus
- **Roles &amp; Responsibilities** &mdash; RACI-Verantwortungsmatrix nach Teamrolle (AE, SE, CSM, PS, Product, Channel, Deal Desk, Legal)
- **Deal Stage Gates** &mdash; was erfüllt sein muss, bevor weitergeschritten wird (Early Qualification bis Closed Won)
- **Anti-Patterns** &mdash; 10 häufige Fehlermuster mit evidenzbasierter Behebung
- **Scoring Rubric** &mdash; objektive Bewertungskriterien 0&ndash;3 pro Element
- **Review Templates** &mdash; wöchentliche Inspektionsfragensätze mit Warnindikatoren
- **MAP Templates** &mdash; kundenorientierte Aktionsplanstruktur mit Evaluierungs-, Beschaffungs- und Implementierungs-Roadmaps
