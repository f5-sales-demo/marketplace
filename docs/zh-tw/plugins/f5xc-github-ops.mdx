**f5xc-github-ops** 外掛程式為 f5-sales-demo 儲存庫提供專屬的 GitHub 操作自動化功能。它處理從問題建立到合併後監控的完整 Git 與 GitHub 生命週期，強制執行問題驅動開發及提交前程式碼檢查閘道。

<Badge text="v2.2.2" variant="note" /> <Badge text="Productivity" variant="tip" />

## 安裝

```
/plugin install f5xc-github-ops@f5-sales-demo-marketplace
```

## 技能

### workflow-lifecycle

儲存庫治理工作流程，處理問題建立、分支命名慣例、PR 建立、附帶錯誤回饋至問題的 CI 輪詢、合併後監控、驗證及任務完成標準。當您說出「commit」、「push」、「create a PR」、「merge」或執行任何 Git 操作時將自動啟動。

### upstream-contribution

適用於基於 fork 開發的上游貢獻工作流程。處理 fork 偵測、上游研究（問題、PR、CONTRIBUTING.md、CI 模式）、跨儲存庫問題與 PR 建立、本地追蹤問題管理、上游 CI 監控及引導式解決方案。

## 代理程式

### github-ops

專屬代理程式，負責所有機械性的 Git 與 GitHub 操作。處理提交前程式碼檢查閘道、問題建立、分支建立、暫存、提交、推送、PR 建立、附帶錯誤回饋的 CI 輪詢、基礎架構失敗重試、合併後監控、分支清理，以及透過 GitHub API 進行的儲存庫設定管理。

回傳結構化狀態報告：`COMPLETE`、`PRE_COMMIT_FAILED`、`CI_FAILED`、`BLOCKED` 或 `FAILED`。

<Aside type="caution">
  此代理程式是 Git 操作的**專屬**處理程式。請勿在使用此外掛程式的儲存庫中直接執行 `git commit`、`git push` 或 `gh pr create`。
</Aside>
