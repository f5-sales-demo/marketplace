**osint-framework** 外掛提供 OSINT 工具目錄與調查技能——來自 [osintframework.com](https://osintframework.com) 的 1,064 種免費情報蒐集工具，涵蓋 34 個類別。包含基於類別的技能、可執行的調查流程、CLI 工具執行、具備 OPSEC 意識的工作流程，以及用於跨調查關聯的持久化實體圖譜。

<Badge text="v1.0.1" variant="note" /> <Badge text="Security" variant="tip" />

## 安裝

```
/plugin install osint-framework@f5xc-salesdemos-marketplace
```

## 指令

### /osint-investigate

對任意目標執行完整的 OSINT 調查。自動偵測目標類型（電子郵件、網域、IP、使用者名稱、公司、人名），並路由至對應的類別技能。

```
/osint-investigate j.smith@example.com
/osint-investigate example.com
/osint-investigate 198.51.100.1
/osint-investigate example-user
```

### /osint-search

在工具目錄中搜尋符合指定能力查詢的工具。

```
/osint-search subdomain enumeration
/osint-search email breach check
/osint-search satellite imagery
```

### /osint-catalog

按類別瀏覽完整工具目錄。不帶參數時，顯示所有 34 個類別及其工具數量。

```
/osint-catalog
/osint-catalog domain
/osint-catalog threat-intel
```

## 技能

### osint-index

頂層意圖路由器。當您說出「尋找相關資訊」、「調查」、「查詢」、「背景調查」，或描述任何開源情報任務時自動啟動。根據目標類型路由至對應的類別技能。

| 目標 | 路由至 |
|--------|----------|
| 使用者名稱或帳號 | `username-recon` |
| 電子郵件地址 | `email-recon` |
| 網域名稱 | `domain-recon` |
| IP 位址 | `ip-address-recon` |
| 人名 + 公司 | `people-search` + `business-records` |
| 公司／組織 | `business-records` + `domain-recon` |
| 惡意程式雜湊值 | `malicious-file-analysis` + `threat-intelligence` |

### osint-catalog

工具目錄瀏覽器。提供完整工具登錄庫、類別索引、調查流程、速率限制參考及關聯引擎文件的存取功能。

## 代理程式

### osint-investigator

自主多工具調查協調器。執行完整調查工作流程——工具可用性檢查、目標類型偵測、順序工具執行（優先被動模式）、結果彙整，以及附帶信心等級與 OPSEC 分類的結構化報告生成。

與實體圖譜整合：每項工具結果均會在 `/tmp/osint-graph/` 中建立實體與關係，用於跨調查關聯分析。

<Aside type="caution">
  主動偵察工具（nmap、nuclei、sherlock）會直接接觸目標基礎設施。OPSEC 警告鉤子會在執行主動工具前發出提示。請務必確認您已獲得授權。
</Aside>

### osint-researcher

唯讀目錄查詢代理程式。搜尋參考檔案以推薦最適合特定任務的工具。不執行任何工具——僅提供建議與說明。由類別技能內部呼叫以進行工具探索。

### osint-executor

單一工具 CLI 執行代理程式。每次以特定參數執行一個工具，解析輸出並回傳結構化結果。包含 20 多種工具的執行方式說明，含精確旗標、輸出格式及 jq 解析模式。

## 速率限制

所有 API 呼叫均使用 `osint_curl` 輔助函式，內建對 HTTP 429/503 的指數退避機制。主要限制如下：

| API | 免費額度 | 環境變數 |
|-----|-----------|---------|
| ipinfo.io | 1,000 次／天 | — |
| crt.sh | 60 次／分鐘 | — |
| NVD | 5 次／30 秒（無金鑰） | `NVD_API_KEY` |
| OpenCorporates | 需要金鑰 | `OPENCORPORATES_API_KEY` |
| Shodan | 1 次／秒 | `SHODAN_API_KEY` |
| GitHub API | 60 次／小時（未驗證） | `GH_TOKEN` |

需要金鑰的 API 若未設定對應環境變數，將自動跳過並顯示清楚的提示訊息。

## 關聯引擎

此外掛包含位於 `/tmp/osint-graph/` 的持久化 JSON 實體圖譜，可跨調查累積發現結果。

```bash
source plugins/osint-framework/scripts/osint-graph.sh
osint_graph_init

# 新增實體
P=$(osint_entity_add "person" "J. Smith" --tool github-api)
C=$(osint_entity_add "company" "F5" --tool github-api)

# 建立關聯
osint_rel_add "$P" "$C" "works_at" --tool github-api

# 查詢：與該人物在 2 跳以內的所有實體
osint_graph_query "$P" 2

# 生成報告
osint_graph_report
```

**信心評分**採用多來源融合機制：
`confidence = 1 - product(1 - ci)`，其中 ci 為每個確認來源的信心值。
兩個各為 0.80 的來源合併後可達 0.96 的信心值。
