**f5xc-meddpicc** 外掛程式為複雜企業 B2B 銷售提供全面的 MEDDPICC 銷售資格認定與交易執行框架。它將 MEDDPICC 從勾選式作業轉化為具備實證資格認定、結構化交易審查與可執行輔導的動態交易執行系統。

<Badge text="v1.0.4" variant="note" /> <Badge text="Productivity" variant="tip" />

## 安裝

```
/plugin install f5xc-meddpicc@f5-sales-demo-marketplace
```

## 什麼是 MEDDPICC？

MEDDPICC 是一套專為複雜 B2B 銷售設計的資格認定與交易執行框架，適用於多方利害關係人、技術風險及採購流程可能阻礙推進動能的情境。

| 字母 | 要素 | 目的 |
| ------ | ------- | ------- |
| **M** | 指標（Metrics） | 客戶預期的量化業務成果 |
| **E** | 經濟採購者（Economic Buyer） | 擁有最終預算決策權的個人 |
| **D** | 決策標準（Decision Criteria） | 用於評估選項的需求條件 |
| **D** | 決策流程（Decision Process） | 從評估到達成協議的逐步路徑 |
| **P** | 文件流程（Paper Process） | 採購機制（法務、安全審查、簽署） |
| **I** | 識別痛點（Identify Pain） | 產生急迫性的關鍵業務痛點 |
| **C** | 內部支持者（Champion） | 具影響力的內部倡導者 |
| **C** | 競爭態勢（Competition） | 所有替代方案（競爭廠商、自建、維持現狀） |

## 技能

### meddpicc-coach

全面的 MEDDPICC 方法論輔導，當您討論交易資格認定、管線健康度、預測準確性或任何 MEDDPICC 要素時，將自動啟動。提供實證引導、運作節奏建議，以及詳細參考資料的連結。

<Aside type="tip" title="自動啟動">
  此技能會在相關情境下自動啟動——無需斜線指令。
</Aside>

### deal-qualification

產生帶有差距分析、風險評估及預測建議的 MEDDPICC 計分卡（0&ndash;24 分）。每個要素依據已驗證的證據，評分範圍為 0&ndash;3 分。

### deal-review

協助進行結構化的每週交易審查，包含證據差異追蹤、風險識別，以及指定負責人與截止日期的行動項目。

### mutual-action-plan

建立與 MEDDPICC 要素掛鉤的客戶端共同行動計畫（Mutual Action Plan）。包含評估里程碑、採購路線圖、實施準備度及利害關係人對應圖。

### champion-test

運用 5 項標準評估（影響力、個人利益、取得管道、情報資訊、行動力），以 0&ndash;15 分判斷您的內部聯絡人是否為真正的支持者、潛力支持者、教練，或友善聯絡人。

## 指令

| 指令 | 目的 |
| ------- | ------- |
| `/f5xc-meddpicc:qualify-deal [account]` | 使用 MEDDPICC 計分卡對交易進行評分 |
| `/f5xc-meddpicc:deal-review [account]` | 協助進行每週交易審查 |
| `/f5xc-meddpicc:champion-test [contact]` | 評估聯絡人是否為真正的支持者 |
| `/f5xc-meddpicc:build-map [account]` | 建立共同行動計畫 |

## 代理程式

### deal-analyst

唯讀研究代理程式，透過審閱本地文件、會議記錄及客戶計畫來分析交易健康度。產出含有證據引用、差距分析及優先行動建議的結構化 MEDDPICC 評估報告。

## 參考資料庫

此外掛程式包含依需求載入的完整參考資料：

- **MEDDPICC 框架** &mdash; 全部 8 個要素的完整定義及證據需求
- **探索問題** &mdash; 針對交易週期各要素經過實戰驗證的提問
- **角色與職責** &mdash; 依團隊角色（AE、SE、CSM、PS、產品、渠道、交易桌、法務）劃分的 RACI 責任矩陣
- **交易階段關卡** &mdash; 推進至下一階段前的必要條件（從早期資格認定至成交）
- **反模式** &mdash; 10 種常見失敗模式及其實證補救方案
- **評分標準** &mdash; 每個要素的客觀 0&ndash;3 分評分準則
- **審查範本** &mdash; 含紅旗指標的每週審查問題集
- **MAP 範本** &mdash; 面向客戶的行動計畫架構，涵蓋評估、採購及實施路線圖
