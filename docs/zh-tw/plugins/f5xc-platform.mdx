**f5xc-platform** 插件提供 F5 Distributed Cloud 平台的完整自動化功能。涵蓋透過 Chrome DevTools MCP 進行的網頁控制台操作、透過 cURL 進行的規格感知 REST API 操作、多供應商身份驗證，以及跨 38 個網域中 98 種資源類型的設定分析。

<Badge text="v3.1.1" variant="note" /> <Badge text="Productivity" variant="tip" />

## 安裝

```
/plugin install f5xc-platform@f5-sales-demo-marketplace
```

## 技能

### api-operations

規格感知的 API CRUD 操作，涵蓋 38 個網域中的 98 種資源類型。為 HTTP 負載平衡器、來源池、WAF 政策、DNS 區域、憑證、雲端站點等提供精確的端點、酬載範本、相依性排序以及多步驟工作流程組合。

### config-analysis

設定分析與建議功能。分析客戶的 JSON 設定，以回答有關安全態勢、功能啟用、模式設定、排除項目及最佳實踐的問題。支援在多輪問答中以壓縮上下文於每次調派之間傳遞。

### api-auth

API 身份驗證，支援 API 令牌標頭驗證及附有令牌驗證程序的 P12 憑證驗證。

### console-auth

F5 XC 網頁控制台的多供應商身份驗證。自動偵測登入類型：原生 F5 XC 登入、具快取工作階段的 Azure SSO，或具 DUO 驗證推播 MFA 的 Azure SSO。

### console-navigator

確定性導航至 F5 XC 控制台各區段。將區段名稱轉換為 URL 路徑、驗證工作階段健康狀態，並確認頁面載入。

### platform-index / api-index / console-index

意圖路由器，用於分類使用者請求並委派至適當的技能與代理程式。這些功能會自動啟用，無法由使用者直接呼叫。

## 指令

### /check-api-token

```
/f5xc-platform:check-api-token
```

透過委派給 api-operator 代理程式來驗證您的 F5 XC API 令牌。

### /login-console

```
/f5xc-platform:login-console
```

透過 console-operator 代理程式向 F5 Distributed Cloud 控制台進行身份驗證。

## 代理程式

### api-operator

自主 REST API 代理程式，執行 cURL 與 jq 指令序列，用於資源 CRUD、令牌驗證及設定操作。在子代理程式中執行，以將冗長的 API JSON 酬載排除在主要工作階段上下文之外。

### console-operator

自主瀏覽器自動化代理程式，執行 Chrome DevTools MCP 工具序列，用於身份驗證、導航及表單互動。在子代理程式中執行，以將含大量令牌的瀏覽器快照排除在主要工作階段之外。

### config-analyzer

唯讀設定分析代理程式。依據資源設定檔參考檔案分析客戶的 JSON 設定，以回答有關安全態勢、功能啟用、模式轉換及最佳實踐的問題。

<Aside type="caution">
  config-analyzer 代理程式為**臨時性**代理程式 &mdash; 每次提問時都會全新生成，各次調派之間不保留任何記憶。後續問題必須在調派提示中包含先前發現的壓縮摘要。
</Aside>
