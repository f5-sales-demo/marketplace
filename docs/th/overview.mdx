**ตลาดกลางปลั๊กอิน Claude Code** คือคอลเลกชันปลั๊กอินที่คัดสรรมาเพื่อ
ขยายความสามารถของ Claude Code ด้วยทักษะใหม่ คำสั่ง และ
เอเจนต์ ตลาดกลางช่วยให้ทีมสามารถแบ่งปันเครื่องมือได้ เพื่อให้นักพัฒนาทุกคน
ได้รับความสามารถเดียวกันโดยไม่ต้องตั้งค่าด้วยตนเอง

**f5-sales-demo ตลาดกลาง** มีปลั๊กอินที่สร้างขึ้นสำหรับ
แพลตฟอร์มเอกสาร [f5-sales-demo](https://github.com/f5-sales-demo)
— ระบบนิเวศของ 20+ รีโพสิทอรีที่
เผยแพร่ไซต์ Astro/Starlight ไปยัง GitHub Pages โดยใช้ไปป์ไลน์
การสร้างร่วมกัน

## วิธีการทำงาน

1. **เพิ่มตลาดกลาง** ลงใน Claude Code — ขั้นตอนนี้จะลงทะเบียนเป็น
   แหล่งที่มาของปลั๊กอิน
2. **ติดตั้งปลั๊กอิน** จากตลาดกลาง — แต่ละปลั๊กอินจะเพิ่ม
   ทักษะและคำสั่งให้กับเซสชัน Claude Code ของคุณ
3. **ใช้เครื่องมือ** — ปลั๊กอินจะทำงานโดยอัตโนมัติตามบริบท
   หรือคุณสามารถเรียกคำสั่งของปลั๊กอินโดยตรง

## ปลั๊กอินที่มีให้ใช้งาน

| ปลั๊กอิน | หมวดหมู่ | คำอธิบาย |
|--------|----------|-------------|
| [f5xc-docs-tools](/marketplace/th/plugins/f5xc-docs-tools/) | ประสิทธิภาพการทำงาน | เครื่องมือตรวจสอบและรีวิวเนื้อหา MDX สำหรับรีโพสิทอรีเอกสาร |
| [f5xc-sales-engineer](/marketplace/th/plugins/f5xc-sales-engineer/) | ประสิทธิภาพการทำงาน | เฟรมเวิร์กบุคลิกภาพ Sales Engineer — การสาธิต การนำเสนอ การตอบคำถาม การจัดการสภาพแวดล้อม |

## เริ่มต้นอย่างรวดเร็ว

เพิ่มตลาดกลางและติดตั้งปลั๊กอิน docs tools:

```
/plugin marketplace add f5-sales-demo/marketplace
/plugin install f5xc-docs-tools@f5-sales-demo-marketplace
```

ดู [การเริ่มต้นใช้งาน](/marketplace/th/getting-started/) สำหรับ
คำแนะนำแบบครบถ้วน

## สถาปัตยกรรม

ตลาดกลางคือ Git รีโพสิทอรีที่มีปลั๊กอินอย่างน้อยหนึ่งรายการ แต่ละปลั๊กอินรวบรวมทักษะ คำสั่ง และเอเจนต์
ไว้ในหน่วยที่ติดตั้งได้หน่วยเดียว

```
Marketplace (marketplace.json)
└── Plugin (plugin.json)
    ├── Skills (SKILL.md)
    │   └── References (data files)
    ├── Commands (.md)
    └── Agents (.md)
```

- **ตลาดกลาง** — รีจิสทรี แสดงรายการปลั๊กอินที่มีให้ใช้งาน
  พร้อมข้อมูลเมตาสำหรับการค้นพบและการติดตั้ง
- **ปลั๊กอิน** — หน่วยที่ติดตั้งได้ ประกอบด้วยทักษะ คำสั่ง
  หรือเอเจนต์อย่างน้อยหนึ่งรายการ
- **ทักษะ** — ความรู้เชิงบริบทที่ Claude เปิดใช้งาน
  โดยอัตโนมัติตามงานที่กำลังดำเนินการ
- **คำสั่ง** — slash command ที่ผู้ใช้เรียกใช้โดยตรง
  (เช่น `/review-mdx`)
- **เอเจนต์** — ซับเอเจนต์เฉพาะทางที่จัดการงานที่ซับซ้อน
  หลายขั้นตอนได้อย่างอิสระ
