**Marketplace สำหรับปลั๊กอิน Claude Code** คือคอลเลกชันที่คัดสรรของ
ปลั๊กอินที่ขยายความสามารถของ Claude Code ด้วยทักษะ คำสั่ง และ
ตัวแทน (Agent) ใหม่ๆ Marketplace ช่วยให้ทีมสามารถแชร์เครื่องมือ
เพื่อให้นักพัฒนาทุกคนได้รับความสามารถเดียวกันโดยไม่ต้องตั้งค่าเอง

**f5xc-salesdemos Marketplace** มอบปลั๊กอินที่สร้างขึ้นสำหรับ
แพลตฟอร์มเอกสาร [f5xc-salesdemos](https://github.com/f5xc-salesdemos)
— ระบบนิเวศของ repository กว่า 20 แห่งที่เผยแพร่เว็บไซต์ Astro/Starlight
ไปยัง GitHub Pages โดยใช้ pipeline การ build ร่วมกัน

## วิธีการทำงาน

1. **เพิ่ม marketplace** ไปยัง Claude Code — การดำเนินการนี้จะลงทะเบียน
   เป็นแหล่งที่มาของปลั๊กอิน
2. **ติดตั้งปลั๊กอิน** จาก marketplace — แต่ละปลั๊กอินจะเพิ่มทักษะ
   และคำสั่งไปยังเซสชัน Claude Code ของคุณ
3. **ใช้เครื่องมือ** — ปลั๊กอินจะเปิดใช้งานโดยอัตโนมัติตามบริบท
   หรือคุณสามารถเรียกใช้คำสั่งโดยตรง

## ปลั๊กอินที่พร้อมใช้งาน

| ปลั๊กอิน | หมวดหมู่ | คำอธิบาย |
|--------|----------|-------------|
| [f5xc-docs-tools](/marketplace/plugins/f5xc-docs-tools/) | เพิ่มประสิทธิภาพ | เครื่องมือตรวจสอบและรีวิวเนื้อหา MDX สำหรับ repository เอกสาร |
| [f5xc-sales-engineer](/marketplace/plugins/f5xc-sales-engineer/) | เพิ่มประสิทธิภาพ | เฟรมเวิร์กบุคลิก Sales Engineer — การดำเนินการสาธิต การนำเสนอ ถาม-ตอบ การจัดการสภาพแวดล้อม |

## เริ่มต้นอย่างรวดเร็ว

เพิ่ม marketplace และติดตั้งปลั๊กอินเครื่องมือเอกสาร:

```
/plugin marketplace add f5xc-salesdemos/marketplace
/plugin install f5xc-docs-tools@f5xc-salesdemos-marketplace
```

ดู [เริ่มต้นใช้งาน](/marketplace/getting-started/) สำหรับ
คำแนะนำแบบครบถ้วน

## สถาปัตยกรรม

Marketplace คือ Git repository ที่ประกอบด้วยปลั๊กอินหนึ่งตัวหรือมากกว่า
แต่ละปลั๊กอินจะรวมทักษะ คำสั่ง และตัวแทน (Agent)
เป็นหน่วยที่ติดตั้งได้หน่วยเดียว

```
Marketplace (marketplace.json)
└── Plugin (plugin.json)
    ├── Skills (SKILL.md)
    │   └── References (data files)
    ├── Commands (.md)
    └── Agents (.md)
```

- **Marketplace** — รีจิสทรี แสดงรายการปลั๊กอินที่พร้อมใช้งาน
  พร้อมข้อมูลเมตาสำหรับการค้นหาและติดตั้ง
- **Plugin (ปลั๊กอิน)** — หน่วยที่ติดตั้งได้ ประกอบด้วยทักษะ
  คำสั่ง หรือตัวแทน (Agent) หนึ่งรายการหรือมากกว่า
- **Skill (ทักษะ)** — ความรู้ตามบริบทที่ Claude เปิดใช้งาน
  โดยอัตโนมัติตามงานที่กำลังดำเนินการ
- **Command (คำสั่ง)** — คำสั่งแบบ slash ที่ผู้ใช้เรียกใช้โดยตรง
  (เช่น `/review-mdx`)
- **Agent (ตัวแทน)** — ตัวแทนย่อยเฉพาะทางที่จัดการงานซับซ้อน
  หลายขั้นตอนโดยอัตโนมัติ
