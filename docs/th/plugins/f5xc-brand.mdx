ปลั๊กอิน **f5xc-brand** บังคับใช้อัตลักษณ์แบรนด์องค์กรของ F5
ในเนื้อหาทุกประเภท ครอบคลุมจานสี F5 จำนวน 45 สี,
กฎไทโปกราฟี (หัวเรื่อง Neusa Next Pro Wide, เนื้อหา Proxima Nova),
การใช้โลโก้แบบ SVG เท่านั้น, การเลือกชุดไอคอน, การเข้าถึงตามมาตรฐาน
WCAG AA และศัพท์เฉพาะผลิตภัณฑ์

<Badge text="v1.0.2" variant="note" /> <Badge text="Productivity" variant="tip" />

## การติดตั้ง

```
/plugin install f5xc-brand@f5xc-salesdemos-marketplace
```

## ทักษะ

### brand-guardian

การบังคับใช้อัตลักษณ์แบรนด์เชิงรุกสำหรับการสร้างเนื้อหาทั้งหมด
เปิดใช้งานโดยอัตโนมัติเมื่อสร้างเอกสาร แผนภาพ
งานนำเสนอ ภาพขนาดย่อวิดีโอ กราฟิกสื่อสังคมออนไลน์ แผนภูมิ Mermaid
หรือภาพวาด Excalidraw ตรวจสอบให้แน่ใจว่ามีการใช้สี
ไทโปกราฟี โลโก้ ไอคอน น้ำเสียงภาษา และการเข้าถึงของ F5 อย่างถูกต้อง

### brand-reviewer

ตรวจสอบเนื้อหาที่มีอยู่เพื่อให้สอดคล้องกับแบรนด์ F5 ตรวจสอบสี
ไทโปกราฟี รูปแบบโลโก้ การเลือกไอคอน ข้อความ alt อัตราส่วนคอนทราสต์
และศัพท์เฉพาะผลิตภัณฑ์ สร้างรายงานที่มีโครงสร้างพร้อมระบุ
VIOLATIONS, WARNINGS และ SUGGESTIONS

### visual-content

กฎเฉพาะรูปแบบสำหรับเนื้อหาภาพที่มีแบรนด์ ให้คำแนะนำ
สำหรับแผนภาพ Mermaid, ภาพวาด Excalidraw, ภาพหน้าจอ,
งานนำเสนอ PowerPoint, ภาพขนาดย่อ YouTube, สคริปต์วิดีโอ และ
กราฟิกสื่อสังคมออนไลน์ พร้อมการเลือกชุดไอคอน เค้าโครงสไลด์
ฟอนต์สำรอง และขนาดเฉพาะแพลตฟอร์ม

## คำสั่ง

### /review-brand

```
/f5xc-brand:review-brand [path-or-glob]
```

ตรวจสอบเนื้อหาเพื่อให้สอดคล้องกับแบรนด์ F5 รับพาธหรือ
รูปแบบ glob ที่เป็นตัวเลือกเพื่อกำหนดขอบเขตการตรวจสอบ หากไม่มีอาร์กิวเมนต์
จะตรวจสอบเนื้อหาทั้งหมดในไดเรกทอรีปัจจุบัน
