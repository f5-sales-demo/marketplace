ปลั๊กอิน **f5xc-brand** บังคับใช้อัตลักษณ์แบรนด์องค์กร F5
ในเนื้อหาทุกประเภท ครอบคลุมชุดสี F5 จำนวน 45 สี,
กฎการใช้ตัวอักษร (พาดหัวด้วย Neusa Next Pro Wide, เนื้อหาด้วย Proxima Nova),
การใช้โลโก้เฉพาะรูปแบบ SVG, การเลือกชุดไอคอน, การเข้าถึงตามมาตรฐาน WCAG AA
และคำศัพท์เฉพาะผลิตภัณฑ์

<Badge text="v1.0.2" variant="note" /> <Badge text="Productivity" variant="tip" />

## การติดตั้ง

```
/plugin install f5xc-brand@f5-sales-demo-marketplace
```

## ทักษะ

### brand-guardian

การบังคับใช้อัตลักษณ์แบรนด์เชิงรุกสำหรับการสร้างเนื้อหาทุกประเภท
เปิดใช้งานโดยอัตโนมัติเมื่อสร้างเอกสาร, ไดอะแกรม,
การนำเสนอ, ภาพขนาดย่อสำหรับวิดีโอ, กราฟิกโซเชียลมีเดีย, แผนภูมิ Mermaid,
หรือภาพวาด Excalidraw รับรองการใช้สี F5, ตัวอักษร, โลโก้, ไอคอน,
น้ำเสียง และการเข้าถึงที่ถูกต้อง

### brand-reviewer

ตรวจสอบเนื้อหาที่มีอยู่เพื่อประเมินความสอดคล้องกับแบรนด์ F5 ตรวจสอบสี,
ตัวอักษร, รูปแบบโลโก้, การเลือกไอคอน, ข้อความ alt, อัตราส่วนคอนทราสต์,
และคำศัพท์เฉพาะผลิตภัณฑ์ สร้างรายงานที่มีโครงสร้างพร้อม VIOLATIONS, WARNINGS และ SUGGESTIONS

### visual-content

กฎเฉพาะรูปแบบสำหรับเนื้อหาภาพที่มีแบรนด์ ให้คำแนะนำสำหรับ
ไดอะแกรม Mermaid, ภาพวาด Excalidraw, ภาพหน้าจอ,
การนำเสนอ PowerPoint, ภาพขนาดย่อ YouTube, สคริปต์วิดีโอ และ
กราฟิกโซเชียลมีเดีย พร้อมการเลือกชุดไอคอน, เลย์เอาต์สไลด์,
แบบอักษรสำรอง และขนาดเฉพาะแพลตฟอร์ม

## คำสั่ง

### /review-brand

```
/f5xc-brand:review-brand [path-or-glob]
```

ตรวจสอบเนื้อหาเพื่อประเมินความสอดคล้องกับแบรนด์ F5 รับพาธหรือรูปแบบ glob
เป็นตัวเลือกเสริมเพื่อกำหนดขอบเขตการตรวจสอบ หากไม่ระบุอาร์กิวเมนต์
จะตรวจสอบเนื้อหาทั้งหมดในไดเรกทอรีปัจจุบัน
