ปลั๊กอิน **f5xc-platform** ให้การทำงานอัตโนมัติอย่างครบถ้วนสำหรับแพลตฟอร์ม F5
Distributed Cloud ครอบคลุมการดำเนินงานเว็บคอนโซลผ่าน
Chrome DevTools MCP, การดำเนินงาน REST API ที่รับรู้ข้อกำหนดผ่าน cURL,
การยืนยันตัวตนหลายผู้ให้บริการ และการวิเคราะห์การกำหนดค่าสำหรับ
ประเภททรัพยากร 98 รายการใน 38 โดเมน

<Badge text="v3.1.1" variant="note" /> <Badge text="Productivity" variant="tip" />

## การติดตั้ง

```
/plugin install f5xc-platform@f5-sales-demo-marketplace
```

## ทักษะ

### api-operations

การดำเนินงาน CRUD ของ API ที่รับรู้ข้อกำหนด ครอบคลุมประเภททรัพยากร 98 รายการใน
38 โดเมน ให้เอนด์พอยต์ที่แม่นยำ, เทมเพลตเพย์โหลด,
ลำดับการพึ่งพา และการประกอบเวิร์กโฟลว์หลายขั้นตอนสำหรับ
HTTP load balancer, origin pool, นโยบาย WAF, DNS zone,
ใบรับรอง, cloud site และอื่น ๆ

### config-analysis

ความสามารถในการวิเคราะห์และให้คำแนะนำด้านการกำหนดค่า วิเคราะห์
การกำหนดค่า JSON ของลูกค้าเพื่อตอบคำถามเกี่ยวกับสถานะความปลอดภัย,
การเปิดใช้งานฟีเจอร์, การตั้งค่าโหมด, การยกเว้น และแนวปฏิบัติที่ดีที่สุด
รองรับการถามตอบหลายรอบพร้อมบริบทที่บีบอัดระหว่างการส่ง

### api-auth

การยืนยันตัวตน API ที่รองรับการยืนยันตัวตนด้วยส่วนหัว API token และ
การยืนยันตัวตนด้วยใบรับรอง P12 พร้อมขั้นตอนการตรวจสอบ token

### console-auth

การยืนยันตัวตนหลายผู้ให้บริการสำหรับเว็บคอนโซล F5 XC
ตรวจจับประเภทการเข้าสู่ระบบโดยอัตโนมัติ: การเข้าสู่ระบบ F5 XC แบบดั้งเดิม,
Azure SSO พร้อมเซสชันที่แคชไว้ หรือ Azure SSO พร้อม DUO verified push MFA

### console-navigator

การนำทางแบบกำหนดตายตัวไปยังส่วนต่าง ๆ ของ F5 XC คอนโซล แปลงชื่อส่วน
เป็นเส้นทาง URL, ตรวจสอบความสมบูรณ์ของเซสชัน และยืนยันการโหลดหน้า

### platform-index / api-index / console-index

เราเตอร์ความตั้งใจที่จำแนกคำขอของผู้ใช้และมอบหมายไปยัง
ทักษะและเอเจนต์ที่เหมาะสม เปิดใช้งานโดยอัตโนมัติและ
ไม่สามารถเรียกใช้โดยผู้ใช้ได้โดยตรง

## คำสั่ง

### /check-api-token

```
/f5xc-platform:check-api-token
```

ตรวจสอบความถูกต้องของ F5 XC API token โดยมอบหมายให้เอเจนต์ api-operator

### /login-console

```
/f5xc-platform:login-console
```

ยืนยันตัวตนกับ F5 Distributed Cloud คอนโซล ผ่านเอเจนต์ console-operator

## เอเจนต์

### api-operator

เอเจนต์ REST API อัตโนมัติที่ดำเนินการลำดับ cURL และ jq สำหรับ
CRUD ทรัพยากร, การตรวจสอบ token และการดำเนินงานการกำหนดค่า
ทำงานในซับเอเจนต์เพื่อให้เพย์โหลด JSON ของ API ที่มีรายละเอียดมากไม่ปรากฏใน
บริบทเซสชันหลัก

### console-operator

เอเจนต์การทำงานอัตโนมัติของเบราว์เซอร์ที่ดำเนินการลำดับเครื่องมือ Chrome DevTools MCP
สำหรับการยืนยันตัวตน, การนำทาง และการโต้ตอบกับฟอร์ม
ทำงานในซับเอเจนต์เพื่อให้สแนปชอตเบราว์เซอร์ที่มี token จำนวนมากไม่ปรากฏใน
เซสชันหลัก

### config-analyzer

เอเจนต์วิเคราะห์การกำหนดค่าแบบอ่านอย่างเดียว วิเคราะห์การกำหนดค่า JSON
ของลูกค้าเทียบกับไฟล์อ้างอิงโปรไฟล์ทรัพยากรเพื่อตอบคำถามเกี่ยวกับสถานะความปลอดภัย,
การเปิดใช้งานฟีเจอร์, การเปลี่ยนโหมด และแนวปฏิบัติที่ดีที่สุด

<Aside type="caution">
  เอเจนต์ config-analyzer มีลักษณะ **ชั่วคราว** &mdash; ถูกสร้างใหม่สำหรับ
  แต่ละคำถามโดยไม่มีหน่วยความจำระหว่างการส่ง คำถามติดตามจะต้องรวม
  ผลการค้นพบก่อนหน้าที่บีบอัดแล้วไว้ในพรอมต์การส่ง
</Aside>
