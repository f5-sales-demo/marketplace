ปลั๊กอิน **f5xc-platform** ให้การทำงานอัตโนมัติอย่างเต็มรูปแบบสำหรับแพลตฟอร์ม F5
Distributed Cloud ครอบคลุมการดำเนินการผ่านเว็บคอนโซลด้วย
Chrome DevTools MCP, การดำเนินการ REST API ที่รองรับ spec ผ่าน cURL,
การยืนยันตัวตนแบบหลายผู้ให้บริการ และการวิเคราะห์การตั้งค่าครอบคลุม
98 ประเภททรัพยากรใน 38 โดเมน

<Badge text="v3.1.1" variant="note" /> <Badge text="Productivity" variant="tip" />

## การติดตั้ง

```
/plugin install f5xc-platform@f5xc-salesdemos-marketplace
```

## ทักษะ

### api-operations

การดำเนินการ CRUD ของ API ที่รองรับ spec ครอบคลุม 98 ประเภททรัพยากรใน
38 โดเมน ให้ endpoint ที่แม่นยำ, เทมเพลต payload,
การจัดลำดับ dependency และการประกอบเวิร์กโฟลว์แบบหลายขั้นตอนสำหรับ
HTTP load balancer, origin pool, นโยบาย WAF, โซน DNS,
ใบรับรอง, cloud site และอื่นๆ อีกมากมาย

### config-analysis

ความสามารถในการวิเคราะห์และให้คำแนะนำเกี่ยวกับการตั้งค่า วิเคราะห์
การตั้งค่า JSON ของลูกค้าเพื่อตอบคำถามเกี่ยวกับสถานะความปลอดภัย,
การเปิดใช้งานฟีเจอร์, การตั้งค่าโหมด, การยกเว้น และแนวทางปฏิบัติที่ดีที่สุด
รองรับการถามตอบแบบหลายรอบพร้อมบริบทที่บีบอัดระหว่างการส่งงาน

### api-auth

การยืนยันตัวตนของ API รองรับการยืนยันตัวตนด้วย API token header และ
การยืนยันตัวตนด้วยใบรับรอง P12 พร้อมขั้นตอนการตรวจสอบ token

### console-auth

การยืนยันตัวตนแบบหลายผู้ให้บริการสำหรับเว็บคอนโซล F5 XC
ตรวจจับประเภทการเข้าสู่ระบบโดยอัตโนมัติ: การเข้าสู่ระบบ F5 XC แบบดั้งเดิม, Azure SSO พร้อม
เซสชันที่แคชไว้ หรือ Azure SSO พร้อม DUO verified push MFA

### console-navigator

การนำทางแบบกำหนดแน่นอนไปยังส่วนต่างๆ ของคอนโซล F5 XC แปลง
ชื่อส่วนเป็นเส้นทาง URL, ตรวจสอบสุขภาพของเซสชัน และยืนยัน
การโหลดหน้า

### platform-index / api-index / console-index

ตัวจัดเส้นทาง intent ที่จำแนกคำร้องขอของผู้ใช้และส่งต่อไปยัง
ทักษะและเอเจนต์ที่เหมาะสม สิ่งเหล่านี้ทำงานโดยอัตโนมัติและ
ผู้ใช้ไม่สามารถเรียกใช้ได้โดยตรง

## คำสั่ง

### /check-api-token

```
/f5xc-platform:check-api-token
```

ตรวจสอบ API token ของ F5 XC ของคุณโดยส่งต่อไปยังเอเจนต์
api-operator

### /login-console

```
/f5xc-platform:login-console
```

ยืนยันตัวตนเข้าสู่คอนโซล F5 Distributed Cloud ผ่าน
เอเจนต์ console-operator

## เอเจนต์

### api-operator

เอเจนต์ REST API อัตโนมัติที่ดำเนินการลำดับ cURL และ jq สำหรับ
CRUD ทรัพยากร, การตรวจสอบ token และการดำเนินการตั้งค่า
ทำงานในเอเจนต์ย่อยเพื่อเก็บ JSON payload ของ API ที่มีรายละเอียดมาก
ออกจากบริบทของเซสชันหลัก

### console-operator

เอเจนต์อัตโนมัติสำหรับการทำงานผ่านเบราว์เซอร์ที่ดำเนินการลำดับเครื่องมือ Chrome DevTools MCP
สำหรับการยืนยันตัวตน, การนำทาง และการโต้ตอบกับฟอร์ม
ทำงานในเอเจนต์ย่อยเพื่อเก็บ snapshot ของเบราว์เซอร์ที่ใช้ token จำนวนมาก
ออกจากเซสชันหลัก

### config-analyzer

เอเจนต์วิเคราะห์การตั้งค่าแบบอ่านอย่างเดียว วิเคราะห์
การตั้งค่า JSON ของลูกค้าเทียบกับไฟล์อ้างอิงโปรไฟล์ทรัพยากรเพื่อตอบ
คำถามเกี่ยวกับสถานะความปลอดภัย, การเปิดใช้งานฟีเจอร์, การเปลี่ยนแปลง
โหมด และแนวทางปฏิบัติที่ดีที่สุด

<Aside type="caution">
  เอเจนต์ config-analyzer เป็นแบบ **ชั่วคราว** &mdash; ถูกสร้างขึ้นใหม่
  สำหรับแต่ละคำถามโดยไม่มีหน่วยความจำระหว่างการส่งงาน คำถามติดตาม
  จะต้องรวมผลการค้นพบก่อนหน้าที่บีบอัดไว้ใน
  prompt การส่งงาน
</Aside>
