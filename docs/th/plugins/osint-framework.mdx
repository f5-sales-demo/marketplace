ปลั๊กอิน **osint-framework** มอบแค็ตตาล็อกเครื่องมือ OSINT และ
ทักษะการสืบสวน — เครื่องมือรวบรวมข่าวกรองฟรี 1,064 รายการ
ใน 34 หมวดหมู่ที่แมปจาก [osintframework.com](https://osintframework.com)
ประกอบด้วยทักษะตามหมวดหมู่ ไปป์ไลน์การสืบสวนที่ปฏิบัติได้จริง
การรันเครื่องมือผ่าน CLI เวิร์กโฟลว์ที่คำนึงถึง OPSEC และกราฟ
เอนทิตีถาวรสำหรับการเชื่อมโยงข้ามการสืบสวน

<Badge text="v1.0.1" variant="note" /> <Badge text="Security" variant="tip" />

## การติดตั้ง

```
/plugin install osint-framework@f5xc-salesdemos-marketplace
```

## คำสั่ง

### /osint-investigate

รันการสืบสวน OSINT แบบครบวงจรกับเป้าหมายใดก็ได้ ตรวจจับประเภท
เป้าหมายโดยอัตโนมัติ (อีเมล, โดเมน, IP, ชื่อผู้ใช้, บริษัท,
ชื่อบุคคล) และนำทางไปยังทักษะหมวดหมู่ที่ถูกต้อง

```
/osint-investigate j.smith@example.com
/osint-investigate example.com
/osint-investigate 198.51.100.1
/osint-investigate example-user
```

### /osint-search

ค้นหาแค็ตตาล็อกเครื่องมือสำหรับเครื่องมือที่ตรงกับการสืบค้นความสามารถ

```
/osint-search subdomain enumeration
/osint-search email breach check
/osint-search satellite imagery
```

### /osint-catalog

เรียกดูแค็ตตาล็อกเครื่องมือทั้งหมดตามหมวดหมู่ หากไม่ระบุอาร์กิวเมนต์
จะแสดงทั้ง 34 หมวดหมู่พร้อมจำนวนเครื่องมือ

```
/osint-catalog
/osint-catalog domain
/osint-catalog threat-intel
```

## ทักษะ

### osint-index

เราเตอร์ความตั้งใจระดับบนสุด เปิดใช้งานอัตโนมัติเมื่อคุณพูดว่า "find
information about", "investigate", "look up", "background check"
หรืออธิบายงาน open-source intelligence ใดๆ นำทางไปยังทักษะ
หมวดหมู่ที่ถูกต้องตามประเภทเป้าหมาย

| เป้าหมาย | นำทางไปยัง |
|--------|----------|
| ชื่อผู้ใช้หรือ handle | `username-recon` |
| ที่อยู่อีเมล | `email-recon` |
| ชื่อโดเมน | `domain-recon` |
| ที่อยู่ IP | `ip-address-recon` |
| ชื่อบุคคล + บริษัท | `people-search` + `business-records` |
| บริษัท / องค์กร | `business-records` + `domain-recon` |
| แฮชมัลแวร์ | `malicious-file-analysis` + `threat-intelligence` |

### osint-catalog

เบราว์เซอร์แค็ตตาล็อกเครื่องมือ ให้การเข้าถึงรีจิสทรีเครื่องมือ
ทั้งหมด, ดัชนีหมวดหมู่, ไปป์ไลน์การสืบสวน, การอ้างอิงขีดจำกัด
อัตราการใช้งาน และเอกสารประกอบของเอนจินความสัมพันธ์

## เอเจนต์

### osint-investigator

ออร์เคสเตรเตอร์การสืบสวนแบบหลายเครื่องมืออัตโนมัติ รันเวิร์กโฟลว์
การสืบสวนแบบครบวงจร — การตรวจสอบความพร้อมใช้งานของเครื่องมือ,
การตรวจจับประเภทเป้าหมาย, การรันเครื่องมือตามลำดับ (passive ก่อน),
การรวบรวมผลลัพธ์ และการสร้างรายงานที่มีโครงสร้างพร้อมระดับความเชื่อมั่น
และการจัดประเภท OPSEC

ผสานรวมกับกราฟเอนทิตี: ผลลัพธ์ของแต่ละเครื่องมือสร้างเอนทิตีและ
ความสัมพันธ์ใน `/tmp/osint-graph/` สำหรับการเชื่อมโยงข้ามการสืบสวน

<Aside type="caution">
  เครื่องมือ active recon (nmap, nuclei, sherlock) ทำการติดต่อโดยตรง
  กับโครงสร้างพื้นฐานของเป้าหมาย hook คำเตือน OPSEC จะแจ้งเตือน
  ก่อนรันเครื่องมือ active เสมอ ตรวจสอบให้แน่ใจว่าคุณได้รับอนุญาต
</Aside>

### osint-researcher

เอเจนต์ค้นหาแค็ตตาล็อกแบบอ่านอย่างเดียว ค้นหาไฟล์อ้างอิงเพื่อ
แนะนำเครื่องมือที่ดีที่สุดสำหรับงานที่กำหนด ไม่รันเครื่องมือเลย —
แนะนำและอธิบายเท่านั้น ใช้ภายในโดยทักษะหมวดหมู่สำหรับการค้นหาเครื่องมือ

### osint-executor

เอเจนต์รัน CLI เครื่องมือเดียว รันเครื่องมือหนึ่งรายการในแต่ละครั้ง
ด้วยพารามิเตอร์เฉพาะ, แยกวิเคราะห์ผลลัพธ์ และส่งคืนผลลัพธ์ที่มี
โครงสร้าง รวมถึงสูตรการรันต่อเครื่องมือสำหรับเครื่องมือกว่า 20 รายการ
พร้อม flags ที่แน่นอน, รูปแบบผลลัพธ์ และรูปแบบการแยกวิเคราะห์ jq

## การจำกัดอัตราการใช้งาน

การเรียก API ทั้งหมดใช้ตัวช่วย `osint_curl` พร้อม exponential
backoff ในตัวสำหรับ HTTP 429/503 ขีดจำกัดหลัก:

| API | ขีดจำกัดฟรี | ตัวแปรสภาพแวดล้อม |
|-----|-----------|---------|
| ipinfo.io | 1,000/วัน | — |
| crt.sh | 60/นาที | — |
| NVD | 5/30 วินาที (ไม่มี key) | `NVD_API_KEY` |
| OpenCorporates | ต้องใช้ key | `OPENCORPORATES_API_KEY` |
| Shodan | 1/วินาที | `SHODAN_API_KEY` |
| GitHub API | 60/ชั่วโมง (ไม่ยืนยันตัวตน) | `GH_TOKEN` |

API ที่ต้องใช้ key จะถูกข้ามโดยอัตโนมัติพร้อมข้อความชัดเจน
หากไม่ได้ตั้งค่าตัวแปรสภาพแวดล้อม

## เอนจินความสัมพันธ์

ปลั๊กอินนี้มีกราฟเอนทิตี JSON ถาวรที่ `/tmp/osint-graph/`
ที่สะสมผลการค้นพบข้ามการสืบสวน

```bash
source plugins/osint-framework/scripts/osint-graph.sh
osint_graph_init

# Add entities
P=$(osint_entity_add "person" "J. Smith" --tool github-api)
C=$(osint_entity_add "company" "F5" --tool github-api)

# Link them
osint_rel_add "$P" "$C" "works_at" --tool github-api

# Query: all entities within 2 hops of the person
osint_graph_query "$P" 2

# Generate report
osint_graph_report
```

**การให้คะแนนความเชื่อมั่น** ใช้การผสานข้อมูลหลายแหล่ง:
`confidence = 1 - product(1 - ci)` สำหรับแต่ละแหล่งที่ยืนยัน
สองแหล่งที่ 0.80 → ความเชื่อมั่นรวม 0.96

## ข้อความทางกฎหมาย

เครื่องมือทั้งหมดใช้เฉพาะข้อมูลที่เปิดเผยต่อสาธารณะ ผู้ใช้มีหน้าที่
รับผิดชอบในการปฏิบัติตามกฎหมาย ระเบียบข้อบังคับ และข้อกำหนดในการ
ให้บริการของแพลตฟอร์มที่เกี่ยวข้อง ห้ามใช้เครื่องมือ OSINT เพื่อการ
เข้าถึงโดยไม่ได้รับอนุญาต, การคุกคาม หรือกิจกรรมที่ผิดกฎหมายใดๆ
