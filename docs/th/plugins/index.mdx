ตลาดกลาง f5-sales-demo มีปลั๊กอินดังต่อไปนี้
ปลั๊กอินแต่ละตัวเพิ่มทักษะ คำสั่ง และตัวแทน (agents) ให้กับ Claude Code

## ปลั๊กอินประกอบด้วยอะไรบ้าง?

ปลั๊กอินแต่ละตัวสามารถประกอบด้วย:

- **ทักษะ (Skills)** — ความรู้เชิงบริบทที่ Claude เปิดใช้งาน
  โดยอัตโนมัติเมื่อเกี่ยวข้อง (เช่น การรีวิวไฟล์ MDX)
- **คำสั่ง (Commands)** — slash commands ที่คุณเรียกใช้โดยตรง
  (เช่น `/f5xc-docs-tools:review-mdx`)
- **ตัวแทน (Agents)** — sub-agents เฉพาะทางสำหรับงานที่ซับซ้อน
