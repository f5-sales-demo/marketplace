marketplace ของ f5xc-salesdemos มีปลั๊กอินดังต่อไปนี้
ปลั๊กอินแต่ละตัวจะเพิ่มทักษะ คำสั่ง และเอเจนต์ให้กับ Claude Code
ที่ช่วยในด้านเอกสาร เวิร์กโฟลว์สาธิต วิธีการขาย
การสร้างแบรนด์ และการดำเนินงานด้านการพัฒนา

## ปลั๊กอินประกอบด้วยอะไรบ้าง?

ปลั๊กอินแต่ละตัวสามารถประกอบด้วย:

- **ทักษะ** — ความรู้เชิงบริบทที่ Claude เปิดใช้งาน
  โดยอัตโนมัติเมื่อเกี่ยวข้อง (เช่น การทบทวนไฟล์ MDX)
- **คำสั่ง** — คำสั่งแบบ slash ที่คุณเรียกใช้โดยตรง
  (เช่น `/f5xc-docs-tools:review-mdx`)
- **เอเจนต์** — เอเจนต์ย่อยเฉพาะทางสำหรับงานที่ซับซ้อน
