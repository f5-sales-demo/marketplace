ปลั๊กอิน **f5xc-meddpicc** มอบเฟรมเวิร์กการคัดกรองและดำเนินดีลตามแนวทาง MEDDPICC อย่างครอบคลุม สำหรับการขาย B2B ระดับองค์กรที่มีความซับซ้อน โดยเปลี่ยน MEDDPICC จากการทำเช็คลิสต์ธรรมดาให้กลายเป็นระบบดำเนินดีลที่มีชีวิต พร้อมด้วยการคัดกรองบนพื้นฐานหลักฐาน การตรวจสอบดีลอย่างเป็นระบบ และคำแนะนำที่นำไปปฏิบัติได้จริง

## MEDDPICC คืออะไร?

MEDDPICC คือเฟรมเวิร์กสำหรับการคัดกรองและดำเนินดีลที่ออกแบบมาสำหรับการขาย B2B ที่มีความซับซ้อน ซึ่งมีผู้มีส่วนได้ส่วนเสียหลายฝ่าย ความเสี่ยงทางเทคนิค และขั้นตอนการจัดซื้อที่อาจทำให้ดีลหยุดชะงักได้

| ตัวอักษร | องค์ประกอบ | วัตถุประสงค์ |
| ------ | ------- | ------- |
| **M** | Metrics | ผลลัพธ์ทางธุรกิจเชิงปริมาณที่ลูกค้าคาดหวัง |
| **E** | Economic Buyer | บุคคลที่มีอำนาจตัดสินใจด้านงบประมาณขั้นสุดท้าย |
| **D** | Decision Criteria | ข้อกำหนดที่ใช้ในการประเมินทางเลือก |
| **D** | Decision Process | เส้นทางทีละขั้นตอนจากการประเมินไปสู่ข้อตกลง |
| **P** | Paper Process | กลไกการจัดซื้อ (กฎหมาย ความปลอดภัย การลงนาม) |
| **I** | Identify Pain | ปัญหาทางธุรกิจที่สำคัญซึ่งสร้างความเร่งด่วน |
| **C** | Champion | ผู้สนับสนุนภายในที่มีอำนาจ |
| **C** | Competition | ทางเลือกทั้งหมด (ผู้ขายรายอื่น สร้างเอง ไม่ทำอะไร) |

## ทักษะ

### meddpicc-coach

การโค้ชตามระเบียบวิธี MEDDPICC อย่างครอบคลุมที่เปิดใช้งานอัตโนมัติเมื่อคุณพูดคุยเกี่ยวกับการคัดกรองดีล สุขภาพไปป์ไลน์ ความแม่นยำของการพยากรณ์ หรือองค์ประกอบ MEDDPICC ใดก็ตาม โดยให้คำแนะนำบนพื้นฐานหลักฐาน ข้อแนะนำจังหวะการทำงาน และลิงก์ไปยังเอกสารอ้างอิงโดยละเอียด

  ทักษะนี้เปิดใช้งานอัตโนมัติเมื่อเกี่ยวข้อง — ไม่จำเป็นต้องใช้คำสั่ง slash

### deal-qualification

สร้างสกอร์การ์ด MEDDPICC พร้อมคะแนน (0&ndash;24) รวมถึงการวิเคราะห์ช่องว่าง การประเมินความเสี่ยง และคำแนะนำการพยากรณ์ แต่ละองค์ประกอบจะได้คะแนน 0&ndash;3 ตามหลักฐานที่ตรวจสอบแล้ว

### deal-review

อำนวยความสะดวกในการตรวจสอบดีลรายสัปดาห์อย่างเป็นระบบ พร้อมการติดตามความเปลี่ยนแปลงของหลักฐาน การระบุความเสี่ยง และรายการดำเนินการที่ระบุเจ้าของและกำหนดส่งที่ชัดเจน

### mutual-action-plan

สร้างแผนปฏิบัติการร่วม (Mutual Action Plan) สำหรับลูกค้าที่เชื่อมโยงกับองค์ประกอบ MEDDPICC ประกอบด้วยเหตุการณ์สำคัญในการประเมิน แผนงานจัดซื้อ ความพร้อมในการนำไปใช้งาน และการจัดทำแผนที่ผู้มีส่วนได้ส่วนเสีย

### champion-test

ใช้การประเมิน 5 เกณฑ์ (อำนาจ ผลประโยชน์ส่วนตัว การเข้าถึง ข้อมูลภายใน การลงมือทำ) โดยให้คะแนน 0&ndash;15 เพื่อพิจารณาว่าผู้ติดต่อภายในของคุณเป็นแชมเปี้ยนตัวจริง แชมเปี้ยนที่กำลังเติบโต โค้ช หรือเพียงผู้ติดต่อที่เป็นมิตร

## เอเจนต์

### deal-analyst

เอเจนต์วิจัยแบบอ่านอย่างเดียวที่วิเคราะห์สุขภาพของดีลโดยการตรวจสอบเอกสารในเครื่อง บันทึกการประชุม และแผนบัญชี สร้างรายงานการประเมิน MEDDPICC อย่างเป็นระบบ พร้อมการอ้างอิงหลักฐาน การวิเคราะห์ช่องว่าง และคำแนะนำการดำเนินการที่จัดลำดับความสำคัญ

## คลังเอกสารอ้างอิง

ปลั๊กอินนี้รวมเอกสารอ้างอิงอย่างครอบคลุมที่โหลดตามความต้องการ:

- **เฟรมเวิร์ก MEDDPICC** &mdash; คำจำกัดความฉบับสมบูรณ์และข้อกำหนดหลักฐานสำหรับทั้ง 8 องค์ประกอบ
- **คำถามเพื่อค้นหาข้อมูล** &mdash; คำถามที่ผ่านการทดสอบภาคสนามสำหรับทุกองค์ประกอบตลอดวงจรดีล
- **บทบาทและความรับผิดชอบ** &mdash; เมทริกซ์ความเป็นเจ้าของแบบ RACI ตามบทบาทในทีม (AE, SE, CSM, PS, Product, Channel, Deal Desk, Legal)
- **เกณฑ์ผ่านขั้นตอนดีล** &mdash; สิ่งที่ต้องเป็นจริงก่อนเลื่อนขั้น (ตั้งแต่การคัดกรองเบื้องต้นจนถึงปิดดีลสำเร็จ)
- **รูปแบบที่ควรหลีกเลี่ยง** &mdash; 10 รูปแบบความล้มเหลวที่พบบ่อย พร้อมการแก้ไขบนพื้นฐานหลักฐาน
- **เกณฑ์การให้คะแนน** &mdash; เกณฑ์การให้คะแนน 0&ndash;3 อย่างเป็นกลางสำหรับแต่ละองค์ประกอบ
- **เทมเพลตการตรวจสอบ** &mdash; ชุดคำถามตรวจสอบรายสัปดาห์พร้อมตัวบ่งชี้สัญญาณเตือน
- **เทมเพลต MAP** &mdash; โครงสร้างแผนปฏิบัติการสำหรับลูกค้า พร้อมแผนงานการประเมิน จัดซื้อ และนำไปใช้งาน
