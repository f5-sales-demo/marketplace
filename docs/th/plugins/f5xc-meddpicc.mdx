ปลั๊กอิน **f5xc-meddpicc** มอบกรอบงาน MEDDPICC ที่ครอบคลุมสำหรับการคัดกรองการขายและการดำเนินการปิดดีลในการขาย B2B ระดับองค์กรที่ซับซ้อน โดยแปลง MEDDPICC จากการตรวจสอบรายการแบบ checkbox ให้กลายเป็นระบบดำเนินการปิดดีลที่มีชีวิต พร้อมการคัดกรองที่อ้างอิงหลักฐาน การทบทวนดีลแบบมีโครงสร้าง และการโค้ชที่นำไปปฏิบัติได้จริง

<Badge text="v1.0.4" variant="note" /> <Badge text="Productivity" variant="tip" />

## การติดตั้ง

```
/plugin install f5xc-meddpicc@f5-sales-demo-marketplace
```

## MEDDPICC คืออะไร?

MEDDPICC คือกรอบงานการคัดกรองและการดำเนินการปิดดีล ที่ออกแบบมาสำหรับการขาย B2B ที่ซับซ้อน ซึ่งผู้มีส่วนเกี่ยวข้องหลายฝ่าย ความเสี่ยงทางเทคนิค และขั้นตอนการจัดซื้ออาจทำให้โมเมนตัมสะดุดได้

| ตัวอักษร | องค์ประกอบ | วัตถุประสงค์ |
| ------ | ------- | ------- |
| **M** | Metrics | ผลลัพธ์ทางธุรกิจที่วัดได้ซึ่งลูกค้าคาดหวัง |
| **E** | Economic Buyer | บุคคลที่มีอำนาจอนุมัติงบประมาณขั้นสุดท้าย |
| **D** | Decision Criteria | ข้อกำหนดที่ใช้ประเมินตัวเลือกต่าง ๆ |
| **D** | Decision Process | เส้นทางทีละขั้นตอนจากการประเมินสู่ข้อตกลง |
| **P** | Paper Process | กลไกการจัดซื้อ (กฎหมาย ความปลอดภัย การลงนาม) |
| **I** | Identify Pain | ความเจ็บปวดทางธุรกิจที่น่าสนใจซึ่งสร้างความเร่งด่วน |
| **C** | Champion | ผู้สนับสนุนภายในที่มีอิทธิพล |
| **C** | Competition | ทางเลือกทั้งหมด (ผู้ขาย การสร้างเอง การไม่ทำอะไร) |

## ทักษะ

### meddpicc-coach

การโค้ชวิธีการ MEDDPICC อย่างครอบคลุม ซึ่งเปิดใช้งานโดยอัตโนมัติเมื่อคุณพูดถึงการคัดกรองดีล ความสมบูรณ์ของไปป์ไลน์ ความแม่นยำในการพยากรณ์ หรือองค์ประกอบ MEDDPICC ใด ๆ พร้อมคำแนะนำที่อ้างอิงหลักฐาน คำแนะนำเกี่ยวกับจังหวะการดำเนินงาน และลิงก์ไปยังเอกสารอ้างอิงโดยละเอียด

<Aside type="tip" title="เปิดใช้งานอัตโนมัติ">
  ทักษะนี้เปิดใช้งานโดยอัตโนมัติเมื่อมีความเกี่ยวข้อง — ไม่จำเป็นต้องใช้คำสั่ง slash
</Aside>

### deal-qualification

สร้าง scorecard MEDDPICC ที่มีคะแนน (0&ndash;24) พร้อมการวิเคราะห์ช่องว่าง การประเมินความเสี่ยง และคำแนะนำในการพยากรณ์ แต่ละองค์ประกอบจะได้รับคะแนน 0&ndash;3 โดยอ้างอิงจากหลักฐานที่ตรวจสอบแล้ว

### deal-review

อำนวยความสะดวกในการตรวจสอบดีลรายสัปดาห์แบบมีโครงสร้าง พร้อมการติดตามความเปลี่ยนแปลงของหลักฐาน การระบุความเสี่ยง และรายการดำเนินการพร้อมผู้รับผิดชอบและวันครบกำหนด

### mutual-action-plan

สร้างแผนปฏิบัติการร่วม (Mutual Action Plan) ที่มุ่งเผชิญกับลูกค้าและผูกกับองค์ประกอบ MEDDPICC ประกอบด้วยเหตุการณ์สำคัญในการประเมิน แผนการจัดซื้อ ความพร้อมในการนำไปใช้งาน และการทำแผนผังผู้มีส่วนเกี่ยวข้อง

### champion-test

ใช้การประเมิน 5 เกณฑ์ (Power, Personal Win, Access, Intel, Action) ให้คะแนน 0&ndash;15 เพื่อพิจารณาว่าผู้ติดต่อภายในของคุณเป็น champion ที่แท้จริง champion ที่กำลังพัฒนา coach หรือผู้ติดต่อที่เป็นมิตร

## Agent

### deal-analyst

Agent วิจัยแบบอ่านอย่างเดียวที่วิเคราะห์ความสมบูรณ์ของดีลโดยการตรวจสอบเอกสารในเครื่อง บันทึกการประชุม และแผนบัญชี สร้างรายงานการประเมิน MEDDPICC แบบมีโครงสร้างพร้อมการอ้างอิงหลักฐาน การวิเคราะห์ช่องว่าง และคำแนะนำดำเนินการที่จัดลำดับความสำคัญ

## คลังเอกสารอ้างอิง

ปลั๊กอินนี้มีเอกสารอ้างอิงที่ครอบคลุมซึ่งโหลดตามต้องการ:

- **กรอบงาน MEDDPICC** &mdash; คำจำกัดความที่สมบูรณ์และข้อกำหนดหลักฐานสำหรับองค์ประกอบทั้ง 8 รายการ
- **คำถามในการค้นพบ** &mdash; คำถามที่ผ่านการทดสอบภาคสนามสำหรับทุกองค์ประกอบตลอดวงจรดีล
- **บทบาทและความรับผิดชอบ** &mdash; เมทริกซ์ความเป็นเจ้าของ RACI ตามบทบาทในทีม (AE, SE, CSM, PS, Product, Channel, Deal Desk, Legal)
- **เกทวัดขั้นตอนดีล** &mdash; สิ่งที่ต้องเป็นจริงก่อนที่จะก้าวหน้าต่อ (ตั้งแต่ Early Qualification จนถึง Closed Won)
- **รูปแบบต่อต้าน** &mdash; รูปแบบความล้มเหลวทั่วไป 10 รูปแบบพร้อมการแก้ไขที่อ้างอิงหลักฐาน
- **เกณฑ์การให้คะแนน** &mdash; เกณฑ์การให้คะแนน 0&ndash;3 อย่างเป็นกลางต่อองค์ประกอบ
- **เทมเพลตการทบทวน** &mdash; ชุดคำถามการตรวจสอบรายสัปดาห์พร้อมตัวบ่งชี้สัญญาณเตือน
- **เทมเพลต MAP** &mdash; โครงสร้างแผนปฏิบัติการที่มุ่งเผชิญกับลูกค้าพร้อมแผนการประเมิน การจัดซื้อ และการนำไปใช้งาน
