ปลั๊กอิน **f5xc-devcontainer** ให้ความสามารถในการรับรู้คอนเทนเนอร์สำหรับ
สภาพแวดล้อมการพัฒนา f5-sales-demo โดยรักษาแคตตาล็อกเครื่องมือ
ที่ครอบคลุม เปิดใช้งานการตรวจสอบคอนเทนเนอร์แบบเรียลไทม์
และจัดการการติดตั้งเครื่องมือพร้อมการติดตามความต่อเนื่องผ่าน GitHub issues

<Badge text="v1.1.6" variant="note" /> <Badge text="Productivity" variant="tip" />

## การติดตั้ง

```
/plugin install f5xc-devcontainer@f5-sales-demo-marketplace
```

## ทักษะ

### tool-catalog

ฐานความรู้ของเครื่องมือ CLI ทุกตัวที่ติดตั้งในคอนเทนเนอร์สำหรับพัฒนา
จัดเรียงตามหมวดหมู่ (คลาวด์, เครือข่าย, ความปลอดภัย, การพัฒนา
และอื่นๆ) จะเปิดใช้งานเมื่อคุณถามว่าควรใช้เครื่องมือใด วิธีใช้งานเครื่องมือ
ว่ามีสิ่งใดติดตั้งอยู่หรือไม่ หรือต้องการความช่วยเหลือเกี่ยวกับการดำเนินการเฉพาะ

### self-awareness

ทักษะด้านการระบุตัวตนและการวินิจฉัยตัวเองของคอนเทนเนอร์ ให้การตรวจสอบ
คอนเทนเนอร์ที่กำลังทำงานแบบเรียลไทม์ผ่าน GitHub API และข้อมูลเมตาดาต้า
การบิลด์ภายในเครื่อง จะเปิดใช้งานสำหรับคำถามเกี่ยวกับตัวตน เวอร์ชัน
ข้อมูลการบิลด์ การตรวจสอบสุขภาพ และประวัติผู้มีส่วนร่วม

<Aside type="tip">
  ถามว่า "who are you?" หรือ "what version am I running?" เพื่อเรียกใช้งาน
  self-awareness
</Aside>

## เอเจนต์

### tool-advisor

อ่านแคตตาล็อกเครื่องมือและส่งคืนคำแนะนำพร้อมวัตถุประสงค์
คำสั่งเริ่มต้นใช้งานอย่างรวดเร็ว และข้อกำหนดการยืนยันตัวตนสำหรับ
คำถามเกี่ยวกับเครื่องมือ CLI ทุกคำถาม

### tool-auditor

ตรวจสอบ Dockerfile ของคอนเทนเนอร์สำหรับพัฒนาเทียบกับไฟล์อ้างอิง
แคตตาล็อกเครื่องมือเพื่อตรวจจับความเบี่ยงเบน &mdash; เครื่องมือที่เพิ่มเข้าไปในอิมเมจ
แต่ยังไม่ได้ลงบัญชีแคตตาล็อก หรือรายการแคตตาล็อกสำหรับเครื่องมือที่ถูกลบออกไปแล้ว

### container-introspector

ดำเนินการตรวจสอบคอนเทนเนอร์แบบเรียลไทม์ผ่าน GitHub API และข้อมูลเมตาดาต้า
ภายในเครื่องสำหรับการระบุตัวตน ลำดับวงศ์ตระกูล การวินิจฉัยตัวเอง ประวัติผู้มีส่วนร่วม
และสายการบิลด์ เฉพาะการดำเนินการแบบอ่านอย่างเดียวเท่านั้น

### container-maintainer

ติดตั้ง ลบ ค้นหา และอัปเดตเครื่องมือ CLI ในคอนเทนเนอร์ที่กำลังทำงาน
โดยใช้รูปแบบตัวจัดการแพ็กเกจที่ถูกต้อง สร้าง GitHub issues กับ
ที่เก็บคอนเทนเนอร์สำหรับพัฒนาเพื่อการติดตามความต่อเนื่อง
