ปลั๊กอิน **f5xc-docs-pipeline** ให้ความรู้เกี่ยวกับสถาปัตยกรรม pipeline เอกสาร
สำหรับ repository ของ f5-sales-demo โดยแนะนำการเขียนเนื้อหา
ชี้แจงความเป็นเจ้าของการตั้งค่าในการสร้างแบบหลาย repo
และให้การแสดงตัวอย่างในเครื่องผ่าน Docker

<Badge text="v1.0.2" variant="note" /> <Badge text="Productivity" variant="tip" />

## การติดตั้ง

```
/plugin install f5xc-docs-pipeline@f5-sales-demo-marketplace
```

## ทักษะ

### content-author

คู่มือการเขียนเนื้อหาที่ครอบคลุมโครงสร้างไฟล์ กฎไวยากรณ์ MDX
ข้อกำหนด frontmatter การอ้างอิงรูปภาพ และการแสดงตัวอย่างในเครื่องผ่าน Docker
เปิดใช้งานเมื่อสร้างหรือแก้ไขเนื้อหาใน `docs/` ทำงานกับไฟล์ MDX
หรือตั้งค่าการแสดงตัวอย่างในเครื่อง

### pipeline-navigator

อธิบายสถาปัตยกรรม pipeline เอกสาร โดยระบุตำแหน่งที่ต้องทำการเปลี่ยนแปลง
ใน repository ต้นทางทั้งสี่ ได้แก่ docs-theme, docs-builder, docs-control
และ docs-icons ชี้แจงความเป็นเจ้าของการตั้งค่า (`astro.config.mjs`, `content.config.ts`)
ห่วงโซ่การส่งต่อ release และการติดตามไฟล์ที่จัดการ

<Aside type="tip">
  ใช้ pipeline-navigator เมื่อต้องการทราบว่า repository ใด
  เป็นเจ้าของไฟล์การตั้งค่าหรือส่วนประกอบที่ระบุ
</Aside>

## คำสั่ง

### /preview-docs

```
/f5xc-docs-pipeline:preview-docs
```

เริ่มต้น Docker dev server ในเครื่องเพื่อแสดงตัวอย่างเอกสาร เรียกใช้
image Docker build ที่ใช้ร่วมกันโดยเมาท์ไดเรกทอรี `docs/` ในเครื่องของคุณ
สำหรับการแสดงตัวอย่างแบบ live
