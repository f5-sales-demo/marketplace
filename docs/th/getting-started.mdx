คู่มือนี้จะแนะนำขั้นตอนการเพิ่มตลาดกลาง f5-sales-demo
ลงใน Claude Code และการติดตั้งปลั๊กอินแรกของคุณ

## ข้อกำหนดเบื้องต้น

- Claude Code **v1.0.33** หรือใหม่กว่า (รองรับตลาดกลางปลั๊กอิน)
- บัญชี GitHub ที่มีสิทธิ์เข้าถึงองค์กร
  [f5-sales-demo](https://github.com/f5-sales-demo)

## การติดตั้ง

<Steps>

1. **เพิ่มตลาดกลาง**

   ในเซสชัน Claude Code ใดก็ได้ ให้รันคำสั่ง:

   ```
   /plugin marketplace add f5-sales-demo/marketplace
   ```

   การดำเนินการนี้จะลงทะเบียนตลาดกลางเป็นแหล่งปลั๊กอิน
   คุณต้องทำเพียงครั้งเดียวเท่านั้น

2. **ดูปลั๊กอินที่มีให้บริการ**

   แสดงรายการปลั๊กอินที่ตลาดกลางนำเสนอ:

   ```
   /plugin marketplace list f5-sales-demo-marketplace
   ```

3. **ติดตั้งปลั๊กอิน**

   ติดตั้งปลั๊กอินที่คุณต้องการ:

   ```
   /plugin install f5xc-docs-tools@f5-sales-demo-marketplace
   /plugin install f5xc-sales-engineer@f5-sales-demo-marketplace
   ```

4. **ใช้งานปลั๊กอิน**

   ไปที่ที่เก็บเนื้อหา f5-sales-demo ใดก็ได้

   สำหรับ **docs tools** ให้รันคำสั่งตรวจสอบ MDX:

   ```
   /f5xc-docs-tools:review-mdx
   ```

   สำหรับ **sales engineer** ให้ใช้วลีทริกเกอร์ เช่น
   "prepare the demo", "run the demo" หรือ "walk through
   the demo" — ปลั๊กอินจะเปิดใช้งานโดยอัตโนมัติ

</Steps>

## การอัปเดตปลั๊กอิน

หากต้องการอัปเดตปลั๊กอินที่ติดตั้งไว้เป็นเวอร์ชันล่าสุด:

```
/plugin update f5xc-docs-tools@f5-sales-demo-marketplace
/plugin update f5xc-sales-engineer@f5-sales-demo-marketplace
```

## การถอนการติดตั้ง

ลบปลั๊กอิน:

```
/plugin uninstall f5xc-docs-tools@f5-sales-demo-marketplace
/plugin uninstall f5xc-sales-engineer@f5-sales-demo-marketplace
```

ลบตลาดกลางทั้งหมด:

```
/plugin marketplace remove f5-sales-demo-marketplace
```

## การกำหนดค่าสำหรับทีม

หากต้องการกำหนดค่าตลาดกลางล่วงหน้าสำหรับทีมทั้งหมด ให้เพิ่ม
ลงใน `.claude/settings.json` ของโปรเจกต์:

```json
{
  "extraKnownMarketplaces": [
    "f5-sales-demo/marketplace"
  ],
  "enabledPlugins": [
    "f5xc-docs-tools@f5-sales-demo-marketplace",
    "f5xc-sales-engineer@f5-sales-demo-marketplace"
  ]
}
```

<Aside type="tip" title="การตั้งค่าที่ใช้ร่วมกัน">
  การคอมมิต `.claude/settings.json` ลงในที่เก็บของคุณ
  หมายความว่าสมาชิกทุกคนในทีมจะได้รับตลาดกลางและปลั๊กอิน
  โดยอัตโนมัติ — ไม่ต้องตั้งค่าด้วยตนเอง
</Aside>

การดำเนินการนี้ทำให้ผู้มีส่วนร่วมทุกคนมีเครื่องมือเดียวกัน
ตั้งแต่เซสชัน Claude Code แรกในที่เก็บ
