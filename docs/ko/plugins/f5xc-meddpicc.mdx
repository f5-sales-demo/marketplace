**f5xc-meddpicc** 플러그인은 복잡한 엔터프라이즈 B2B 영업을 위한 포괄적인 MEDDPICC 영업 자격 검증 및 딜 실행 프레임워크를 제공합니다. 이 플러그인은 MEDDPICC를 단순한 체크박스 작업에서 증거 기반 자격 검증, 구조화된 딜 리뷰, 실행 가능한 코칭을 갖춘 살아있는 딜 실행 시스템으로 전환합니다.

<Badge text="v1.0.4" variant="note" /> <Badge text="생산성" variant="tip" />

## 설치

```
/plugin install f5xc-meddpicc@f5-sales-demo-marketplace
```

## MEDDPICC란 무엇인가?

MEDDPICC는 복수의 이해관계자, 기술적 리스크, 조달 단계가 모멘텀을 저해할 수 있는 복잡한 B2B 영업을 위해 설계된 자격 검증 및 딜 실행 프레임워크입니다.

| 문자 | 요소 | 목적 |
| ------ | ------- | ------- |
| **M** | 메트릭스 (Metrics) | 고객이 기대하는 정량화된 비즈니스 성과 |
| **E** | 경제적 구매자 (Economic Buyer) | 최종 예산 권한을 가진 개인 |
| **D** | 결정 기준 (Decision Criteria) | 옵션 평가에 사용되는 요구사항 |
| **D** | 결정 프로세스 (Decision Process) | 평가부터 합의까지의 단계별 경로 |
| **P** | 문서 프로세스 (Paper Process) | 조달 메커니즘 (법무, 보안, 서명) |
| **I** | 고통 식별 (Identify Pain) | 긴박감을 조성하는 핵심 비즈니스 고충 |
| **C** | 챔피언 (Champion) | 강력한 내부 지지자 |
| **C** | 경쟁 (Competition) | 모든 대안 (벤더, 자체 구축, 현상 유지) |

## 스킬

### meddpicc-coach

딜 자격 검증, 파이프라인 상태, 예측 정확도, 또는 MEDDPICC 요소를 논의할 때 자동으로 활성화되는 포괄적인 MEDDPICC 방법론 코칭입니다. 증거 기반 가이던스, 운영 리듬 권장 사항, 상세 참조 자료 링크를 제공합니다.

<Aside type="tip" title="자동 활성화">
  이 스킬은 관련 상황에서 자동으로 활성화됩니다 — 슬래시 명령어가 필요하지 않습니다.
</Aside>

### deal-qualification

갭 분석, 리스크 평가, 예측 권장 사항이 포함된 점수화된 MEDDPICC 스코어카드 (0&ndash;24)를 생성합니다. 각 요소는 검증된 증거를 기반으로 0&ndash;3점으로 채점됩니다.

### deal-review

증거 델타 추적, 리스크 식별, 담당자와 기한이 명시된 실행 항목을 포함한 구조화된 주간 딜 점검을 진행합니다.

### mutual-action-plan

MEDDPICC 요소와 연계된 고객 대상 상호 실행 계획(Mutual Action Plan)을 구축합니다. 평가 마일스톤, 조달 로드맵, 구현 준비 상태, 이해관계자 매핑이 포함됩니다.

### champion-test

내부 담당자가 진정한 챔피언인지, 성장 중인 챔피언인지, 코치인지, 또는 우호적인 담당자인지를 판단하기 위해 5가지 기준(파워, 개인적 이익, 접근성, 인텔, 행동)을 적용하여 0&ndash;15점으로 평가합니다.

## 명령어

| 명령어 | 목적 |
| ------- | ------- |
| `/f5xc-meddpicc:qualify-deal [account]` | MEDDPICC 스코어카드로 딜 점수화 |
| `/f5xc-meddpicc:deal-review [account]` | 주간 딜 점검 진행 |
| `/f5xc-meddpicc:champion-test [contact]` | 담당자가 진정한 챔피언인지 평가 |
| `/f5xc-meddpicc:build-map [account]` | 상호 실행 계획 구축 |

## 에이전트

### deal-analyst

로컬 문서, 회의록, 계정 계획을 검토하여 딜 상태를 분석하는 읽기 전용 리서치 에이전트입니다. 증거 인용, 갭 분석, 우선순위가 지정된 실행 권장 사항이 포함된 구조화된 MEDDPICC 평가 보고서를 생성합니다.

## 참조 라이브러리

플러그인에는 필요에 따라 로드되는 포괄적인 참조 자료가 포함되어 있습니다:

- **MEDDPICC 프레임워크** &mdash; 8가지 요소 전체에 대한 완전한 정의 및 증거 요구사항
- **디스커버리 질문** &mdash; 딜 사이클 전반에 걸쳐 모든 요소에 대한 현장 검증 질문
- **역할 및 책임** &mdash; 팀 역할별 RACI 소유권 매트릭스 (AE, SE, CSM, PS, 제품, 채널, 딜 데스크, 법무)
- **딜 스테이지 게이트** &mdash; 다음 단계로 진행하기 전에 충족해야 할 조건 (초기 자격 검증부터 Closed Won까지)
- **안티 패턴** &mdash; 증거 기반 해결책이 포함된 10가지 일반적인 실패 유형
- **채점 루브릭** &mdash; 요소별 객관적인 0&ndash;3점 채점 기준
- **리뷰 템플릿** &mdash; 레드 플래그 지표가 포함된 주간 점검 질문 세트
- **MAP 템플릿** &mdash; 평가, 조달, 구현 로드맵이 포함된 고객 대상 실행 계획 구조
