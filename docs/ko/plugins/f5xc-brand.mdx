**f5xc-brand** 플러그인은 모든 콘텐츠 유형에 걸쳐 F5 기업 브랜드 아이덴티티를
적용합니다. 45가지 색상의 F5 팔레트, 타이포그래피 규칙 (Neusa Next Pro Wide 헤드라인, Proxima Nova
본문), SVG 전용 로고 사용, 아이콘 팩 선택, WCAG AA
접근성, 그리고 제품 용어를 포함합니다.

<Badge text="v1.0.2" variant="note" /> <Badge text="Productivity" variant="tip" />

## 설치

```
/plugin install f5xc-brand@f5-sales-demo-marketplace
```

## 기능

### brand-guardian

모든 콘텐츠 제작에 대한 능동적 브랜드 아이덴티티 적용.
문서, 다이어그램, 프레젠테이션, 비디오 썸네일, 소셜 미디어 그래픽, Mermaid
차트, 또는 Excalidraw 드로잉을 생성할 때 자동으로 활성화됩니다. F5 색상,
타이포그래피, 로고, 아이콘, 어조, 그리고 접근성의 올바른 사용을 보장합니다.

### brand-reviewer

기존 콘텐츠의 F5 브랜드 준수 여부를 검토합니다. 색상,
타이포그래피, 로고 형식, 아이콘 선택, 대체 텍스트, 대비 비율,
그리고 제품 용어를 감사합니다. VIOLATIONS(위반), WARNINGS(경고), SUGGESTIONS(제안)이 포함된 구조화된 보고서를 생성합니다.

### visual-content

브랜드화된 시각적 콘텐츠를 위한 형식별 규칙. Mermaid
다이어그램, Excalidraw 드로잉, 스크린샷, PowerPoint 프레젠테이션, YouTube 썸네일,
비디오 스크립트, 그리고 아이콘 팩 선택, 슬라이드 레이아웃,
폰트 대체, 플랫폼별 치수가 포함된 소셜 미디어 그래픽에 대한 가이드를 제공합니다.

## 명령어

### /review-brand

```
/f5xc-brand:review-brand [path-or-glob]
```

F5 브랜드 준수 여부에 대한 콘텐츠를 검토합니다. 검토 범위를 지정하기 위해 선택적 경로
또는 글로브 패턴을 허용합니다. 인수 없이 사용 시, 현재 디렉터리의
모든 콘텐츠를 검토합니다.
