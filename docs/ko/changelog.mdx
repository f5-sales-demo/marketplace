f5-sales-demo 마켓플레이스의 주요 변경 사항입니다.

## 1.0.0

두 개의 플러그인이 포함된 마켓플레이스 최초 릴리스.

### f5xc-docs-tools v1.0.0

- 7가지 검사 항목을 포함한 MDX 콘텐츠 유효성 검사 기능
  (프론트매터, 구문, 임포트, 구성 요소, 이미지,
  구조, 익스포트)
- 온디맨드 유효성 검사를 위한 `/review-mdx` 명령어

### f5xc-sales-engineer v1.0.0

- 4가지 페르소나 기능: sales-engineer (디스패처),
  demo-executor (4단계 라이프사이클), presenter
  (워크스루), subject-matter-expert (인용 기반 Q&A)
- 2개의 에이전트: demo-housekeeping (준비 상태 확인
  및 정리), demo-researcher (참조 기반 리서치)
- 데모 저장소 전반에 걸쳐 제품에 구애받지 않는
  재사용을 위한 컨벤션 파일 아키텍처
