**Claude Code 플러그인 마켓플레이스**는 새로운 기술, 명령어, 에이전트로 Claude Code를 확장하는 플러그인의 큐레이션된 컬렉션입니다. 마켓플레이스를 통해 팀은 도구를 공유할 수 있으므로 모든 개발자가 수동 설정 없이 동일한 기능을 사용할 수 있습니다.

**f5-sales-demo 마켓플레이스**는 [f5-sales-demo](https://github.com/f5-sales-demo) 문서화 플랫폼을 위해 구축된 플러그인을 제공합니다 — 공유 빌드 파이프라인을 사용하여 Astro/Starlight 사이트를 GitHub Pages에 게시하는 20개 이상의 저장소로 구성된 에코시스템입니다.

## 작동 방식

1. **마켓플레이스 추가** — Claude Code에 플러그인 소스로 등록합니다
2. **플러그인 설치** — 마켓플레이스에서 플러그인을 설치하면 각 플러그인이 Claude Code 세션에 기술과 명령어를 추가합니다
3. **도구 사용** — 플러그인은 컨텍스트에 따라 자동으로 활성화되거나, 명령어를 직접 호출하여 사용합니다

## 사용 가능한 플러그인

| 플러그인 | 카테고리 | 설명 |
|--------|----------|-------------|
| [f5xc-docs-tools](/marketplace/plugins/f5xc-docs-tools/) | 생산성 | 문서 저장소를 위한 MDX 콘텐츠 검증 및 검토 도구 |
| [f5xc-sales-engineer](/marketplace/plugins/f5xc-sales-engineer/) | 생산성 | 세일즈 엔지니어 페르소나 프레임워크 — 데모 실행, 프레젠테이션, Q&A, 환경 관리 |

## 빠른 시작

마켓플레이스를 추가하고 문서 도구 플러그인을 설치합니다:

```
/plugin marketplace add f5-sales-demo/marketplace
/plugin install f5xc-docs-tools@f5-sales-demo-marketplace
```

전체 안내는 [시작하기](/marketplace/getting-started/)를 참조하십시오.

## 아키텍처

마켓플레이스는 하나 이상의 플러그인을 포함하는 Git 저장소입니다. 각 플러그인은 기술, 명령어, 에이전트를 단일 설치 가능한 단위로 패키징합니다.

```
Marketplace (marketplace.json)
└── Plugin (plugin.json)
    ├── Skills (SKILL.md)
    │   └── References (data files)
    ├── Commands (.md)
    └── Agents (.md)
```

- **마켓플레이스** — 레지스트리. 검색 및 설치를 위한 메타데이터와 함께 사용 가능한 플러그인을 나열합니다.
- **플러그인** — 설치 가능한 단위. 하나 이상의 기술, 명령어, 에이전트를 포함합니다.
- **기술(Skill)** — Claude가 현재 작업에 따라 자동으로 활성화하는 맥락적 지식입니다.
- **명령어(Command)** — 사용자가 직접 호출하는 슬래시 명령어입니다 (예: `/review-mdx`).
- **에이전트(Agent)** — 복잡한 다단계 작업을 자율적으로 처리하는 특화된 서브 에이전트입니다.
