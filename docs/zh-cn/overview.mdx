**Claude Code 插件市场**是一个经过精心整理的插件集合，
用于为 Claude Code 扩展新技能、命令和代理。
插件市场允许团队共享工具，使每位开发者无需手动配置即可获得相同的能力。

**f5-sales-demo 市场**提供专为
[f5-sales-demo](https://github.com/f5-sales-demo)
文档平台构建的插件——该生态系统包含 20 余个代码仓库，
使用共享构建流水线将 Astro/Starlight 站点发布至 GitHub Pages。

## 工作原理

1. **添加市场** 至 Claude Code——将其注册为插件来源
2. **从市场安装插件**——每个插件为您的 Claude Code 会话添加技能和命令
3. **使用工具**——插件根据上下文自动激活，或由您直接调用其命令

## 可用插件

| 插件 | 类别 | 描述 |
|--------|----------|-------------|
| [f5xc-docs-tools](/marketplace/zh-cn/plugins/f5xc-docs-tools/) | 生产力 | 适用于文档仓库的 MDX 内容验证与审查工具 |
| [f5xc-sales-engineer](/marketplace/zh-cn/plugins/f5xc-sales-engineer/) | 生产力 | 销售工程师角色框架——演示执行、演示文稿、问答及环境管理 |

## 快速开始

添加市场并安装文档工具插件：

```
/plugin marketplace add f5-sales-demo/marketplace
/plugin install f5xc-docs-tools@f5-sales-demo-marketplace
```

完整操作流程请参阅[入门指南](/marketplace/zh-cn/getting-started/)。

## 架构

插件市场是一个包含一个或多个插件的 Git 仓库。每个插件将技能、命令和代理打包为一个可安装单元。

```
Marketplace (marketplace.json)
└── Plugin (plugin.json)
    ├── Skills (SKILL.md)
    │   └── References (data files)
    ├── Commands (.md)
    └── Agents (.md)
```

- **市场（Marketplace）**——注册表。列出可供发现和安装的插件及其元数据。
- **插件（Plugin）**——可安装单元。包含一个或多个技能、命令或代理。
- **技能（Skill）**——Claude 根据当前任务自动激活的上下文知识。
- **命令（Command）**——由用户直接调用的斜杠命令（例如 `/review-mdx`）。
- **代理（Agent）**——自主处理复杂多步骤任务的专用子代理。
