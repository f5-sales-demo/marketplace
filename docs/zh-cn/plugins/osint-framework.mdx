**osint-framework** 插件提供 OSINT 工具目录与调查技能——来自 [osintframework.com](https://osintframework.com) 的 1,064 种免费情报收集工具，涵盖 34 个类别。它包括基于类别的技能、可执行的调查流水线、CLI 工具执行、具备操作安全意识的工作流，以及用于跨调查关联的持久化实体图谱。

<Badge text="v1.0.1" variant="note" /> <Badge text="Security" variant="tip" />

## 安装

```
/plugin install osint-framework@f5xc-salesdemos-marketplace
```

## 命令

### /osint-investigate

对任意目标执行完整的 OSINT 调查。自动检测目标类型（电子邮件、域名、IP 地址、用户名、公司、人名），并路由至相应的类别技能。

```
/osint-investigate j.smith@example.com
/osint-investigate example.com
/osint-investigate 198.51.100.1
/osint-investigate example-user
```

### /osint-search

在工具目录中搜索与能力查询匹配的工具。

```
/osint-search subdomain enumeration
/osint-search email breach check
/osint-search satellite imagery
```

### /osint-catalog

按类别浏览完整的工具目录。不带参数时显示所有 34 个类别及其工具数量。

```
/osint-catalog
/osint-catalog domain
/osint-catalog threat-intel
```

## 技能

### osint-index

顶层意图路由器。当您说"查找相关信息"、"调查"、"查询"、"背景核查"，或描述任何开源情报任务时自动激活。根据目标类型路由至相应的类别技能。

| 目标 | 路由至 |
|--------|----------|
| 用户名或账号 | `username-recon` |
| 电子邮件地址 | `email-recon` |
| 域名 | `domain-recon` |
| IP 地址 | `ip-address-recon` |
| 姓名 + 公司 | `people-search` + `business-records` |
| 公司 / 组织 | `business-records` + `domain-recon` |
| 恶意软件哈希 | `malicious-file-analysis` + `threat-intelligence` |

### osint-catalog

工具目录浏览器。提供对完整工具注册表、类别索引、调查流水线、速率限制参考及关联引擎文档的访问。

## 智能体

### osint-investigator

自主多工具调查编排器。执行完整的调查工作流——工具可用性检查、目标类型检测、顺序工具执行（优先被动模式）、结果聚合，以及带有置信度级别和操作安全分类的结构化报告生成。

与实体图谱集成：每次工具执行结果均在 `/tmp/osint-graph/` 中创建实体和关系，用于跨调查关联。

<Aside type="caution">
  主动侦察工具（nmap、nuclei、sherlock）会与目标基础设施直接通信。操作安全警告钩子会在运行主动工具前发出提示。请务必确保您已获得授权。
</Aside>

### osint-researcher

只读目录查询智能体。搜索参考文件以推荐适合特定任务的最佳工具。从不执行工具——仅提供推荐和说明。由类别技能内部调用以进行工具发现。

### osint-executor

单工具 CLI 执行智能体。每次执行一个工具，使用指定参数，解析输出并返回结构化结果。包含 20 余种工具的逐一执行方案，涵盖精确标志、输出格式及 jq 解析模式。

## 速率限制

所有 API 调用均使用内置指数退避的 `osint_curl` 辅助函数，在收到 HTTP 429/503 时自动重试。主要限制如下：

| API | 免费限额 | 环境变量 |
|-----|-----------|---------|
| ipinfo.io | 1,000 次/天 | — |
| crt.sh | 60 次/分钟 | — |
| NVD | 5 次/30 秒（无密钥） | `NVD_API_KEY` |
| OpenCorporates | 需要密钥 | `OPENCORPORATES_API_KEY` |
| Shodan | 1 次/秒 | `SHODAN_API_KEY` |
| GitHub API | 60 次/小时（未认证） | `GH_TOKEN` |

若未设置相应的环境变量，需要密钥的 API 将自动跳过并给出明确提示。

## 关联引擎

该插件在 `/tmp/osint-graph/` 包含一个持久化 JSON 实体图谱，可在多次调查中累积发现结果。

```bash
source plugins/osint-framework/scripts/osint-graph.sh
osint_graph_init

# 添加实体
P=$(osint_entity_add "person" "J. Smith" --tool github-api)
C=$(osint_entity_add "company" "F5" --tool github-api)

# 建立关联
osint_rel_add "$P" "$C" "works_at" --tool github-api

# 查询：距该人物 2 跳以内的所有实体
osint_graph_query "$P" 2

# 生成报告
osint_graph_report
```

**置信度评分**采用多源融合方式：
`confidence = 1 - product(1 - ci)`，对每个确认来源进行计算。
两个置信度均为 0.80 的来源 → 综合置信度 0.96。
