**f5xc-meddpicc** 插件为复杂企业级 B2B 销售提供了一套全面的 MEDDPICC
销售资格评估与交易执行框架。它将 MEDDPICC 从一个简单的清单勾选练习
转变为一个活的交易执行系统，具备基于证据的资格评估、结构化交易审查
以及可操作的辅导指引。

<Badge text="v1.0.4" variant="note" /> <Badge text="Productivity" variant="tip" />

## 安装

```
/plugin install f5xc-meddpicc@f5xc-salesdemos-marketplace
```

## 什么是 MEDDPICC？

MEDDPICC 是一套资格评估与交易执行框架，专为复杂 B2B 销售场景设计，
在这些场景中，多个利益相关方、技术风险和采购流程都可能导致交易势头
中断。

| 字母 | 要素 | 用途 |
| ------ | ------- | ------- |
| **M** | Metrics（量化指标） | 客户期望的可量化业务成果 |
| **E** | Economic Buyer（经济决策者） | 拥有最终预算审批权的个人 |
| **D** | Decision Criteria（决策标准） | 用于评估各选项的需求条件 |
| **D** | Decision Process（决策流程） | 从评估到签约的逐步路径 |
| **P** | Paper Process（文书流程） | 采购机制（法务、安全审查、签署） |
| **I** | Identify Pain（识别痛点） | 创造紧迫性的关键业务痛点 |
| **C** | Champion（内部拥护者） | 有影响力的内部倡导者 |
| **C** | Competition（竞争） | 所有替代方案（竞争厂商、自建、维持现状） |

## 技能

### meddpicc-coach

全面的 MEDDPICC 方法论辅导，当您讨论交易资格评估、管道健康度、
预测准确性或任何 MEDDPICC 要素时会自动激活。提供基于证据的指导、
运营节奏建议以及详细参考资料的链接。

<Aside type="tip" title="自动激活">
  该技能在相关场景下会自动激活——无需使用斜杠命令。
</Aside>

### deal-qualification

生成带评分的 MEDDPICC 记分卡（0&ndash;24），包含差距分析、
风险评估和预测建议。每个要素根据经过验证的证据评分 0&ndash;3。

### deal-review

支持结构化的每周交易检查，包含证据变化追踪、风险识别，
以及指定负责人和截止日期的行动项。

### mutual-action-plan

构建与 MEDDPICC 要素关联的面向客户的共同行动计划（Mutual Action Plan）。
包括评估里程碑、采购路线图、实施准备度和利益相关方映射。

### champion-test

应用 5 项标准评估（权力、个人收益、访问权限、情报、行动力），
评分 0&ndash;15，以判断您的内部联系人是真正的拥护者、潜在拥护者、
教练还是友好联系人。

## 命令

| 命令 | 用途 |
| ------- | ------- |
| `/f5xc-meddpicc:qualify-deal [account]` | 使用 MEDDPICC 记分卡为交易评分 |
| `/f5xc-meddpicc:deal-review [account]` | 进行每周交易检查 |
| `/f5xc-meddpicc:champion-test [contact]` | 评估联系人是否为真正的拥护者 |
| `/f5xc-meddpicc:build-map [account]` | 构建共同行动计划 |

## 代理

### deal-analyst

只读研究代理，通过审查本地文档、会议记录和客户计划来分析交易健康度。
生成结构化的 MEDDPICC 评估报告，包含证据引用、差距分析和优先级排序的
行动建议。
