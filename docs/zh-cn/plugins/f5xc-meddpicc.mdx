**f5xc-meddpicc** 插件为复杂企业B2B销售提供全面的MEDDPICC销售资质认定与交易执行框架。它将MEDDPICC从一项核查练习转变为一个基于证据的资质认定、结构化交易评审和可操作辅导指导的实时交易执行系统。

<Badge text="v1.0.4" variant="note" /> <Badge text="生产力" variant="tip" />

## 安装

```
/plugin install f5xc-meddpicc@f5-sales-demo-marketplace
```

## 什么是MEDDPICC？

MEDDPICC是一套资质认定与交易执行框架，专为复杂B2B销售场景设计——在这类场景中，多方利益相关者、技术风险和采购环节均可能导致销售推进受阻。

| 字母 | 要素 | 目的 |
| ------ | ------- | ------- |
| **M** | 量化指标（Metrics） | 客户期望实现的量化业务成果 |
| **E** | 经济购买方（Economic Buyer） | 拥有最终预算审批权的个人 |
| **D** | 决策标准（Decision Criteria） | 用于评估各选项的要求 |
| **D** | 决策流程（Decision Process） | 从评估到达成协议的逐步路径 |
| **P** | 采购流程（Paper Process） | 采购机制（法务、安全、签署） |
| **I** | 识别痛点（Identify Pain） | 制造紧迫感的迫切业务痛点 |
| **C** | 内部支持者（Champion） | 强有力的内部倡导者 |
| **C** | 竞争态势（Competition） | 所有替代方案（竞争厂商、自建、维持现状） |

## 技能

### meddpicc-coach

全面的MEDDPICC方法论辅导功能，在您讨论交易资质认定、管道健康度、预测准确性或任何MEDDPICC要素时自动激活。提供基于证据的指导、运营节奏建议以及详细参考资料的链接。

<Aside type="tip" title="自动激活">
  此技能在相关情境下自动激活——无需斜杠命令。
</Aside>

### deal-qualification

生成带有差距分析、风险评估和预测建议的MEDDPICC评分卡（0&ndash;24分）。每个要素根据已验证的证据评定0&ndash;3分。

### deal-review

协助开展结构化的每周交易检视，包含证据变化追踪、风险识别，以及带有责任人和截止日期的行动事项。

### mutual-action-plan

构建与MEDDPICC要素关联的面向客户的共同行动计划（Mutual Action Plan）。涵盖评估里程碑、采购路线图、实施准备情况及利益相关者映射。

### champion-test

应用五项标准评估（影响力、个人收益、可达性、信息获取、行动力），以0&ndash;15分判断您的内部联系人是真正的支持者、成长中的支持者、教练还是友好联系人。

## 命令

| 命令 | 目的 |
| ------- | ------- |
| `/f5xc-meddpicc:qualify-deal [account]` | 使用MEDDPICC评分卡对交易进行评分 |
| `/f5xc-meddpicc:deal-review [account]` | 协助开展每周交易检视 |
| `/f5xc-meddpicc:champion-test [contact]` | 评估联系人是否为真正的内部支持者 |
| `/f5xc-meddpicc:build-map [account]` | 构建共同行动计划 |

## 智能体

### deal-analyst

只读研究型智能体，通过审阅本地文档、会议记录和客户计划来分析交易健康状况。生成带有证据引用、差距分析和优先行动建议的结构化MEDDPICC评估报告。
