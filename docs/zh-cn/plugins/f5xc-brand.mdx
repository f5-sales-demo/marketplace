**f5xc-brand** 插件在所有内容类型中强制执行 F5 企业品牌标识规范。涵盖 45 色 F5 调色板、字体排版规则（Neusa Next Pro Wide 标题字体、Proxima Nova 正文字体）、仅限 SVG 的徽标使用、图标包选择、WCAG AA 无障碍访问标准以及产品术语规范。

<Badge text="v1.0.2" variant="note" /> <Badge text="Productivity" variant="tip" />

## 安装

```
/plugin install f5xc-brand@f5-sales-demo-marketplace
```

## 技能

### brand-guardian

针对所有内容创作的主动品牌标识执行规范。在创建文档、图表、演示文稿、视频缩略图、社交媒体图形、Mermaid 图表或 Excalidraw 绘图时自动激活。确保正确使用 F5 颜色、字体排版、徽标、图标、语调风格及无障碍访问规范。

### brand-reviewer

审查现有内容是否符合 F5 品牌规范。对颜色、字体排版、徽标格式、图标选择、替代文本、对比度及产品术语进行审计。生成包含违规项（VIOLATIONS）、警告项（WARNINGS）和建议项（SUGGESTIONS）的结构化报告。

### visual-content

品牌视觉内容的格式专项规则。为 Mermaid 图表、Excalidraw 绘图、截图、PowerPoint 演示文稿、YouTube 缩略图、视频脚本及社交媒体图形提供指导，涵盖图标包选择、幻灯片布局、字体备选方案及平台专属尺寸规范。

## 命令

### /review-brand

```
/f5xc-brand:review-brand [path-or-glob]
```

审查内容是否符合 F5 品牌规范。接受可选的路径或 glob 模式以缩小审查范围。若不提供参数，则审查当前目录中的所有内容。
