**f5xc-docs-pipeline** 插件为 f5-sales-demo 仓库提供文档管道架构知识。它指导内容创作，厘清多仓库构建中的配置归属关系，并提供基于本地 Docker 的预览功能。

<Badge text="v1.0.2" variant="note" /> <Badge text="Productivity" variant="tip" />

## 安装

```
/plugin install f5xc-docs-pipeline@f5-sales-demo-marketplace
```

## 技能

### content-author

内容创作指南，涵盖文件结构、MDX 语法规则、frontmatter 要求、图片引用及本地 Docker 预览。在创建或编辑 `docs/` 内容、操作 MDX 文件或配置本地预览时自动激活。

### pipeline-navigator

文档管道架构说明，解释如何在四个上游仓库（docs-theme、docs-builder、docs-control 和 docs-icons）中进行变更。厘清配置归属关系（`astro.config.mjs`、`content.config.ts`）、发布分发链及受管文件追踪机制。

<Aside type="tip">
  当您需要了解哪个仓库负责特定配置文件或组件时，请使用
  pipeline-navigator。
</Aside>

## 命令

### /preview-docs

```
/f5xc-docs-pipeline:preview-docs
```

启动本地 Docker 开发服务器以预览文档。运行共享 Docker 构建镜像，并将您本地的 `docs/` 目录挂载以实现实时预览。
