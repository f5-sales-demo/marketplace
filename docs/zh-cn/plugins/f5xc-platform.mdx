**f5xc-platform** 插件为 F5 分布式云平台提供完整的自动化支持。它涵盖通过 Chrome DevTools MCP 进行的 Web 控制台操作、基于规范的 cURL REST API 操作、多提供商身份验证，以及跨 38 个域中 98 种资源类型的配置分析。

<Badge text="v3.1.1" variant="note" /> <Badge text="Productivity" variant="tip" />

## 安装

```
/plugin install f5xc-platform@f5-sales-demo-marketplace
```

## 技能

### api-operations

基于规范的 API CRUD 操作，涵盖 38 个域中的 98 种资源类型。为 HTTP 负载均衡器、源池、WAF 策略、DNS 区域、证书、云站点等提供精确的端点、负载模板、依赖排序及多步骤工作流组合。

### config-analysis

配置分析与咨询能力。分析客户 JSON 配置，以回答有关安全态势、功能启用、模式设置、排除规则及最佳实践的问题。支持在调度之间使用压缩上下文进行多轮问答。

### api-auth

API 身份验证，支持 API 令牌头部认证和带令牌验证流程的 P12 证书认证。

### console-auth

F5 XC Web 控制台的多提供商身份验证。自动检测登录类型：原生 F5 XC 登录、使用缓存会话的 Azure SSO，或使用 DUO 已验证推送 MFA 的 Azure SSO。

### console-navigator

确定性导航至 F5 XC 控制台各部分。将区域名称转换为 URL 路径，验证会话健康状态，并确认页面加载完成。

### platform-index / api-index / console-index

意图路由器，负责对用户请求进行分类并委派给相应的技能和代理。这些路由器自动激活，不可由用户直接调用。

## 命令

### /check-api-token

```
/f5xc-platform:check-api-token
```

通过委派给 api-operator 代理来验证您的 F5 XC API 令牌。

### /login-console

```
/f5xc-platform:login-console
```

通过 console-operator 代理对 F5 分布式云控制台进行身份验证。

## 代理

### api-operator

自主 REST API 代理，执行 cURL 和 jq 序列，用于资源 CRUD、令牌验证及配置操作。在子代理中运行，以避免冗长的 API JSON 负载占用主会话上下文。

### console-operator

自主浏览器自动化代理，执行 Chrome DevTools MCP 工具序列，用于身份验证、导航及表单交互。在子代理中运行，以避免包含大量令牌的浏览器快照占用主会话上下文。

### config-analyzer

只读配置分析代理。根据资源配置文件参考文件分析客户 JSON 配置，以回答有关安全态势、功能启用、模式转换及最佳实践的问题。

<Aside type="caution">
  config-analyzer 代理是**临时性的** &mdash; 每次提问时都会全新生成，且调度之间没有记忆。后续问题必须在调度提示中包含经过压缩的先前分析结果。
</Aside>
