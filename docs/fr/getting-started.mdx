Ce guide vous explique comment ajouter la place de marché f5-sales-demo
à Claude Code et installer votre premier plugin.

## Prérequis

- Claude Code **v1.0.33** ou version ultérieure (prise en charge de la place de marché de plugins)
- Un compte GitHub avec accès à l'organisation
  [f5-sales-demo](https://github.com/f5-sales-demo)

## Installation

<Steps>

1. **Ajouter la place de marché**

   Dans n'importe quelle session Claude Code, exécutez :

   ```
   /plugin marketplace add f5-sales-demo/marketplace
   ```

   Cette commande enregistre la place de marché en tant que source de plugins. Vous
   n'avez besoin de le faire qu'une seule fois.

2. **Parcourir les plugins disponibles**

   Listez ce que propose la place de marché :

   ```
   /plugin marketplace list f5-sales-demo-marketplace
   ```

3. **Installer un plugin**

   Installez le plugin dont vous avez besoin :

   ```
   /plugin install f5xc-docs-tools@f5-sales-demo-marketplace
   /plugin install f5xc-sales-engineer@f5-sales-demo-marketplace
   ```

4. **Utiliser le plugin**

   Accédez à n'importe quel dépôt de contenu f5-sales-demo.

   Pour les **outils de documentation**, exécutez la commande de révision MDX :

   ```
   /f5xc-docs-tools:review-mdx
   ```

   Pour **sales engineer**, utilisez des expressions déclencheurs telles que
   « prepare the demo », « run the demo » ou « walk through
   the demo » — le plugin s'active automatiquement.

</Steps>

## Mise à jour des plugins

Pour mettre à jour un plugin installé vers la dernière version :

```
/plugin update f5xc-docs-tools@f5-sales-demo-marketplace
/plugin update f5xc-sales-engineer@f5-sales-demo-marketplace
```

## Désinstallation

Supprimer un plugin :

```
/plugin uninstall f5xc-docs-tools@f5-sales-demo-marketplace
/plugin uninstall f5xc-sales-engineer@f5-sales-demo-marketplace
```

Supprimer entièrement la place de marché :

```
/plugin marketplace remove f5-sales-demo-marketplace
```

## Configuration pour l'équipe

Pour préconfigurer la place de marché pour toute votre équipe, ajoutez-la
au fichier `.claude/settings.json` de votre projet :

```json
{
  "extraKnownMarketplaces": [
    "f5-sales-demo/marketplace"
  ],
  "enabledPlugins": [
    "f5xc-docs-tools@f5-sales-demo-marketplace",
    "f5xc-sales-engineer@f5-sales-demo-marketplace"
  ]
}
```

<Aside type="tip" title="Paramètres partagés">
  Valider `.claude/settings.json` dans votre dépôt permet à
  chaque membre de l'équipe d'obtenir automatiquement la place de marché et les plugins
  — sans configuration manuelle requise.
</Aside>

Cela garantit que chaque contributeur dispose des mêmes outils disponibles
dès sa première session Claude Code dans le dépôt.
