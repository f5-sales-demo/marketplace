La place de marché f5xc-salesdemos fournit les plugins suivants.
Chaque plugin ajoute des compétences, des commandes et des agents à Claude Code
qui facilitent la documentation, les flux de démonstration, la méthodologie de vente,
l'image de marque et les opérations de développement.

## Que contient un plugin ?

Chaque plugin peut inclure :

- **Des compétences** — connaissances contextuelles que Claude active
  automatiquement lorsque cela est pertinent (ex. : révision de fichiers MDX)
- **Des commandes** — commandes slash que vous invoquez directement
  (ex. : `/f5xc-docs-tools:review-mdx`)
- **Des agents** — sous-agents spécialisés pour les tâches complexes

## Installer un plugin

```
/plugin install f5xc-docs-tools@f5xc-salesdemos-marketplace
/plugin install f5xc-sales-engineer@f5xc-salesdemos-marketplace
/plugin install f5xc-github-ops@f5xc-salesdemos-marketplace
/plugin install f5xc-docs-pipeline@f5xc-salesdemos-marketplace
/plugin install f5xc-brand@f5xc-salesdemos-marketplace
/plugin install f5xc-meddpicc@f5xc-salesdemos-marketplace
/plugin install f5xc-devcontainer@f5xc-salesdemos-marketplace
/plugin install f5xc-platform@f5xc-salesdemos-marketplace
/plugin install osint-framework@f5xc-salesdemos-marketplace
/plugin install f5xc-firecrawl@f5xc-salesdemos-marketplace
```
