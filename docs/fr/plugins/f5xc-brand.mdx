Le plugin **f5xc-brand** applique l'identité de marque F5 sur tous les types de contenu. Il couvre la palette de 45 couleurs F5, les règles typographiques (titres Neusa Next Pro Wide, corps Proxima Nova), l'utilisation exclusive de logos SVG, la sélection de paquets d'icônes, l'accessibilité WCAG AA et la terminologie produit.

<Badge text="v1.0.2" variant="note" /> <Badge text="Productivity" variant="tip" />

## Installation

```
/plugin install f5xc-brand@f5-sales-demo-marketplace
```

## Compétences

### brand-guardian

Application proactive de l'identité de marque pour toute création de contenu. S'active automatiquement lors de la création de documentation, de diagrammes, de présentations, de miniatures vidéo, de graphiques pour les réseaux sociaux, de graphiques Mermaid ou de dessins Excalidraw. Garantit l'utilisation correcte des couleurs F5, de la typographie, des logos, des icônes, du ton éditorial et de l'accessibilité.

### brand-reviewer

Examine le contenu existant pour vérifier sa conformité à la marque F5. Contrôle les couleurs, la typographie, le format des logos, la sélection des icônes, le texte alternatif, les ratios de contraste et la terminologie produit. Produit des rapports structurés avec les catégories VIOLATIONS, AVERTISSEMENTS et SUGGESTIONS.

### visual-content

Règles spécifiques par format pour le contenu visuel de marque. Fournit des recommandations pour les diagrammes Mermaid, les dessins Excalidraw, les captures d'écran, les présentations PowerPoint, les miniatures YouTube, les scripts vidéo et les graphiques pour les réseaux sociaux, avec la sélection des Paquets d'icônes, les mises en page de diapositives, les polices de substitution et les dimensions spécifiques à chaque plateforme.

## Commandes

### /review-brand

```
/f5xc-brand:review-brand [path-or-glob]
```

Examine le contenu pour vérifier sa conformité à la marque F5. Accepte un chemin ou un motif glob optionnel pour délimiter la portée de la révision. Sans arguments, examine l'ensemble du contenu du répertoire courant.
